### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/numeric/special_functions/beta&action=edit&section=3 "Edit section: Error handling")] Error handling

Errors may be reported as specified in [math_errhandling](../math/math_errhandling.html "cpp/numeric/math/math errhandling"). 

  * If any argument is NaN, NaN is returned and domain error is not reported. 
  * The function is only required to be defined where both x and y are greater than zero, and is allowed to report a domain error otherwise. 



### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/numeric/special_functions/beta&action=edit&section=4 "Edit section: Notes")] Notes

Implementations that do not support C++17, but support [ISO 29124:2010](../../experimental/special_math.html "cpp/experimental/special math"), provide this function if `__STDCPP_MATH_SPEC_FUNCS__` is defined by the implementation to a value at least 201003L and if the user defines `__STDCPP_WANT_MATH_SPEC_FUNCS__` before including any standard library headers. 

Implementations that do not support ISO 29124:2010 but support TR 19768:2007 (TR1), provide this function in the header `tr1/cmath` and namespace `std::tr1`. 

An implementation of this function is also [available in boost.math](https://www.boost.org/doc/libs/release/libs/math/doc/html/math_toolkit/sf_beta/beta_function.html). 

std::beta(x, y) equals std::beta(y, x).

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/numeric/special_functions/beta&action=edit&section=5 "Edit section: Example")] Example

Run this code
    
    
    #include <cassert>
    #include <cmath>
    #include <iomanip>
    #include <iostream>
    #include <numbers>
    #include <string>
     
    long binom_via_beta(int n, int k)
    {
        return [std::lround](../math/round.html)(1 / ((n + 1) * std::beta(n - k + 1, k + 1)));
    }
     
    long binom_via_gamma(int n, int k)
    {
        return [std::lround](../math/round.html)([std::tgamma](../math/tgamma.html)(n + 1) /
                          ([std::tgamma](../math/tgamma.html)(n - k + 1) * 
                           [std::tgamma](../math/tgamma.html)(k + 1)));
    }
     
    int main()
    {
        [std::cout](../../io/cout.html) << "Pascal's triangle:\n";
        for (int n = 1; n < 10; ++n)
        {
            [std::cout](../../io/cout.html) << [std::string](../../string/basic_string.html)(20 - n * 2, ' ');
            for (int k = 1; k < n; ++k)
            {
                [std::cout](../../io/cout.html) << [std::setw](../../io/manip/setw.html)(3) << binom_via_beta(n, k) << ' ';
                [assert](../../error/assert.html)(binom_via_beta(n, k) == binom_via_gamma(n, k));
            }
            [std::cout](../../io/cout.html) << '\n';
        }
     
        // A spot-check
        const long double p = 0.123; // a random value in [0, 1]
        const long double q = 1 - p;
        const long double π = [std::numbers::pi_v](../constants.html)<long double>;
        [std::cout](../../io/cout.html) << "\n\n" << [std::setprecision](../../io/manip/setprecision.html)(19)
                  << "β(p,1-p)   = " << std::beta(p, q) << '\n'
                  << "π/sin(π*p) = " << π / [std::sin](../math/sin.html)(π * p) << '\n';
    }
