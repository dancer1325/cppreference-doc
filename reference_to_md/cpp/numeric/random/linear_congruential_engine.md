`linear_congruential_engine` is a random number engine based on [Linear congruential generator](https://en.wikipedia.org/wiki/Linear_congruential_generator "enwiki:Linear congruential generator") (LCG).

### Template parameters

UIntType  |  \-  |  The result type generated by the generator. The effect is undefined if this is not one of unsigned short, unsigned int, unsigned long, or unsigned long long.   
---|---|---  
a  |  \-  |  the multiplier term   
c  |  \-  |  the increment term   
m  |  \-  |  the modulus term   
  
When m is not zero, if a >= m or c >= m is true, the program is ill-formed. 

### Generator properties

The [size](../random.html#Random_number_engines "cpp/numeric/random") of the states of `linear_congruential_engine` is 1, each of them consists of a single integer. 

The actual modulus \\(\scriptsize m_0\\)m0 is defined as follows: 

  * If m is not zero, \\(\scriptsize m_0\\)m0 is m. 
  * If m is zero, \\(\scriptsize m_0\\)m0 is the value of [std::numeric_limits](../../types/numeric_limits.html)<result_type>::max() plus 1 (which means \\(\scriptsize m_0\\)m0 need not be representable as `result_type`). 



The [transition algorithm](../random.html#Random_number_engines "cpp/numeric/random") of `linear_congruential_engine` is \\(\scriptsize TA(x_i) = (a \cdot x_i + c)\mod m_0\\)TA(xi) = (a·xi+c) mod m0. 

The [generation algorithm](../random.html#Random_number_engines "cpp/numeric/random") of `linear_congruential_engine` is \\(\scriptsize GA(x_i) = (a \cdot x_i + c)\mod m_0\\)GA(xi) = (a·xi+c) mod m0. 

The pseudo-random number generated with the current state is also the successor state.
