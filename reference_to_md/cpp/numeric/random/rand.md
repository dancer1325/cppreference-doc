int rand(); |  |   
| |   
  
Returns a pseudo-random integral value from the range `[`​0​`, `[RAND_MAX](RAND_MAX.html)`]`. 

[std::srand()](srand.html "cpp/numeric/random/srand") seeds the pseudo-random number generator used by `rand()`. If `rand()` is used before any calls to [std::srand()](srand.html "cpp/numeric/random/srand"), `rand()` behaves as if it was seeded with [std::srand](srand.html)(1). 

Each time `rand()` is seeded with [std::srand()](srand.html "cpp/numeric/random/srand"), it must produce the same sequence of values on successive calls. 

Other functions in the standard library may call `rand`. It is implementation-defined which functions do so. 

It is implementation-defined whether `rand()` is thread-safe.

### Return value

Pseudo-random integral value between ​0​ and [RAND_MAX](RAND_MAX.html "cpp/numeric/random/RAND MAX"). 

### Notes

There are no guarantees as to the quality of the random sequence produced. In the past, some implementations of `rand()` have had serious shortcomings in the randomness, distribution and period of the sequence produced (in one well-known example, the low-order bit simply alternated between 1 and ​0​ between calls). 

`rand()` is not recommended for serious random-number generation needs. It is recommended to use C++11's [random number generation](../random.html "cpp/numeric/random") facilities to replace `rand()`.(since C++11)

### Example

The function `bounded_rand()` below is an adapted version of [Debiased Modulo (Once)](https://pcg-random.org/posts/bounded-rands.html#debiased-modulo-once-javas-method).

Run this code
    
    
    #include <cstdlib>
    #include <ctime>
    #include <initializer_list>
    #include <iostream>
     
    unsigned bounded_rand(unsigned range)
    {
        for (unsigned x, r;;)
            if (x = rand(), r = x % range, x - r <= -range)
                return r;
    }
     
    int main() 
    {
        [std::srand](srand.html)([std::time](../../chrono/c/time.html)({})); // use current time as seed for random generator
        const int random_value = std::rand();
        [std::cout](../../io/cout.html) << "Random value on [0, " << [RAND_MAX](RAND_MAX.html) << "]: " << random_value << '\n';
     
        for (const unsigned sides : {2, 4, 6, 8})
        {
            [std::cout](../../io/cout.html) << "Roll " << sides << "-sided die 8 times: ";
            for (int n = 8; n; --n)
                [std::cout](../../io/cout.html) << 1 + bounded_rand(sides) << ' ';
     
            [std::cout](../../io/cout.html) << '\n';
        }
    }

Possible output: 
    
    
    Random value on [0, 2147483647]: 948298199
    Roll 2-sided die 8 times: 2 2 1 2 1 1 2 2 
    Roll 4-sided die 8 times: 1 3 4 2 1 3 3 1 
    Roll 6-sided die 8 times: 3 2 1 6 6 4 4 2 
    Roll 8-sided die 8 times: 4 5 6 6 3 6 1 2
