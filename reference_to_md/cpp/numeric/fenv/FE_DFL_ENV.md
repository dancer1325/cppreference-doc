**FE_DFL_ENV**(C++11)  
  
[[edit]](https://en.cppreference.com/mwiki/index.php?title=Template:cpp/numeric/fenv/navbar_content&action=edit)

Defined in header `[<cfenv>](../../header/cfenv.html "cpp/header/cfenv")` |  |   
---|---|---  
#define FE_DFL_ENV /*implementation defined*/ |  |  (since C++11)  
| |   
  
The macro constant `FE_DFL_ENV` expands to an expression of type const [std::fenv_t](../fenv.html)*, which points to a full copy of the default floating-point environment, that is, the environment as loaded at program startup. 

Additional macros that begin with `FE_` followed by uppercase letters, and have the type const [std::fenv_t](../fenv.html)*, may be supported by an implementation. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/numeric/fenv/FE_DFL_ENV&action=edit&section=1 "Edit section: Example")] Example

Run this code
    
    
    #include <cfenv>
    #include <iostream>
    // #pragma STDC FENV_ACCESS ON
     
    void show_env()
    {
        const int e = [std::fetestexcept](fetestexcept.html)([FE_ALL_EXCEPT](FE_exceptions.html));
        if (e & [FE_DIVBYZERO](FE_exceptions.html))
            [std::cout](../../io/cout.html) << "division by zero is raised\n";
        if (e & [FE_INEXACT](FE_exceptions.html))
            [std::cout](../../io/cout.html) << "inexact is raised\n";
        if (e & [FE_INVALID](FE_exceptions.html))
            [std::cout](../../io/cout.html) << "invalid is raised\n";
        if (e & [FE_UNDERFLOW](FE_exceptions.html))
            [std::cout](../../io/cout.html) << "underflow is raised\n";
        if (e & [FE_OVERFLOW](FE_exceptions.html))
            [std::cout](../../io/cout.html) << "overflow is raised\n";
     
        switch ([std::fegetround](feround.html)())
        {  
            case [FE_DOWNWARD](FE_round.html):
                [std::cout](../../io/cout.html) << "rounding down\n";
                break;
            case [FE_TONEAREST](FE_round.html):
                [std::cout](../../io/cout.html) << "rounding to nearest\n";
                break;
            case [FE_TOWARDZERO](FE_round.html):
                [std::cout](../../io/cout.html) << "rounding to zero\n";
                break;
            case [FE_UPWARD](FE_round.html):
                [std::cout](../../io/cout.html) << "rounding up\n";
                break;
        }
    }
     
    int main()
    {
        [std::cout](../../io/cout.html) << "On startup:\n";
        show_env();
     
        [std::feraiseexcept](feraiseexcept.html)([FE_UNDERFLOW](FE_exceptions.html) | [FE_OVERFLOW](FE_exceptions.html));
        [std::fesetround](feround.html)([FE_UPWARD](FE_round.html));
     
        [std::cout](../../io/cout.html) << "\nBefore restoration:\n";
        show_env();
     
        [std::fesetenv](feenv.html)(FE_DFL_ENV);
     
        [std::cout](../../io/cout.html) << "\nAfter reset to default:\n";
        show_env();
    }

Output: 
    
    
    On startup: 
    rounding to nearest
     
    Before restoration: 
    underflow is raised
    overflow is raised
    rounding up
     
    After reset to default: 
    rounding to nearest
