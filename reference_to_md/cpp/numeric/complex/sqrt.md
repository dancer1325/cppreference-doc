template< class T >  
[std::complex](../complex.html)<T> sqrt( const [std::complex](../complex.html)<T>& z ); |  |   
| |   
  
Computes the square root of the complex number z with a branch cut along the negative real axis. 

## Contents

  * [1 Parameters](sqrt.html#Parameters)
  * [2 Return value](sqrt.html#Return_value)
  * [3 Error handling and special values](sqrt.html#Error_handling_and_special_values)
  * [4 Notes](sqrt.html#Notes)
  * [5 Example](sqrt.html#Example)
  * [6 Defect reports](sqrt.html#Defect_reports)
  * [7 See also](sqrt.html#See_also)

  
---  
  
### Parameters

z  |  \-  |  complex number to take the square root of   
---|---|---  
  
### Return value

If no errors occur, returns the square root of z, in the range of the right half-plane, including the imaginary axis ([0; +∞) along the real axis and (−∞; +∞) along the imaginary axis). 

### Error handling and special values

Errors are reported consistent with [math_errhandling](../math/math_errhandling.html "cpp/numeric/math/math errhandling"). 

If the implementation supports IEEE floating-point arithmetic, 

  * The function is continuous onto the branch cut taking into account the sign of imaginary part 
  * [std::sqrt](../math/sqrt.html)([std::conj](conj.html)(z)) == [std::conj](conj.html)([std::sqrt](../math/sqrt.html)(z))
  * If z is `(±0,+0)`, the result is `(+0,+0)`
  * If z is `(x,+∞)`, the result is `(+∞,+∞)` even if x is NaN 
  * If z is `(x,NaN)`, the result is `(NaN,NaN)` (unless x is ±∞) and [FE_INVALID](../fenv/FE_exceptions.html "cpp/numeric/fenv/FE exceptions") may be raised 
  * If z is `(-∞,y)`, the result is `(+0,+∞)` for finite positive y 
  * If z is `(+∞,y)`, the result is `(+∞,+0)` for finite positive y 
  * If z is `(-∞,NaN)`, the result is `(NaN,∞)` (sign of imaginary part unspecified) 
  * If z is `(+∞,NaN)`, the result is `(+∞,NaN)`
  * If z is `(NaN,y)`, the result is `(NaN,NaN)` and [FE_INVALID](../fenv/FE_exceptions.html "cpp/numeric/fenv/FE exceptions") may be raised 
  * If z is `(NaN,NaN)`, the result is `(NaN,NaN)`



### Notes

The semantics of this function are intended to be consistent with the C function [`csqrt`](../../../c/numeric/complex/csqrt.html "c/numeric/complex/csqrt"). 

### Example

Run this code
    
    
    #include <complex>
    #include <iostream>
     
    int main()
    {
        [std::cout](../../io/cout.html) << "Square root of -4 is "
                  << [std::sqrt](../math/sqrt.html)([std::complex](../complex.html)<double>(-4.0, 0.0)) << '\n'
                  << "Square root of (-4,-0) is "
                  << [std::sqrt](../math/sqrt.html)([std::complex](../complex.html)<double>(-4.0, -0.0))
                  << " (the other side of the cut)\n";
    }

Output: 
    
    
    Square root of -4 is (0,2)
    Square root of (-4,-0) is (0,-2) (the other side of the cut)
