template< class T >  
[std::complex](../complex.html)<T> log10( const [std::complex](../complex.html)<T>& z ); |  |   
| |   
  
Computes complex common (base 10) logarithm of a complex value z with a branch cut along the negative real axis. 

The behavior of this function is equivalent to `[std::log](log.html "cpp/numeric/complex/log")(z) / [std::log](../math/log.html "cpp/numeric/math/log")(T(10))`.

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/numeric/complex/log10&action=edit&section=3 "Edit section: Example")] Example

Run this code
    
    
    #include <cmath>
    #include <complex>
    #include <iostream>
     
    int main()
    {
        [std::complex](../complex.html)<double> z(0.0, 1.0); // r = 0, θ = pi / 2
        [std::cout](../../io/cout.html) << "2 * log10" << z << " = " << 2.0 * [std::log10](../math/log10.html)(z) << '\n';
     
        [std::complex](../complex.html)<double> z2(sqrt(2.0) / 2, sqrt(2.0) / 2); // r = 1, θ = pi / 4
        [std::cout](../../io/cout.html) << "4 * log10" << z2 << " = " << 4.0 * [std::log10](../math/log10.html)(z2) << '\n';
     
        [std::complex](../complex.html)<double> z3(-100.0, 0.0); // r = 100, θ = pi
        [std::cout](../../io/cout.html) << "log10" << z3 << " = " << [std::log10](../math/log10.html)(z3) << '\n';
        [std::complex](../complex.html)<double> z4(-100.0, -0.0); // the other side of the cut
        [std::cout](../../io/cout.html) << "log10" << z4 << " = " << [std::log10](../math/log10.html)(z4) << " "
                     "(the other side of the cut)\n"
                     "(note: pi / log(10) = " << [std::acos](../math/acos.html)(-1.0) / [std::log](../math/log.html)(10.0) << ")\n";
    }

Possible output: 
    
    
    2 * log10(0,1) = (0,1.36438)
    4 * log10(0.707107,0.707107) = (0,1.36438)
    log10(-100,0) = (2,1.36438)
    log10(-100,-0) = (2,-1.36438) (the other side of the cut)
    (note: pi / log(10) = 1.36438)
