Apply binary operators to each element of two valarrays, or a valarray and a value. 

1) The operators works on valarrays of the same size and returns a valarray with the same size as the parameters with the operation applied to every elements of the two arguments.

2,3) Applies the operator between each element of the valarray and the scalar.

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/numeric/valarray/operator_arith3&action=edit&section=2 "Edit section: Return value")] Return value

A valarray with the same size as the parameter. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/numeric/valarray/operator_arith3&action=edit&section=3 "Edit section: Note")] Note

The behaviour is undefined when the two arguments are valarrays with different sizes. 

The function can be implemented with the return type different from [std::valarray](../valarray.html "cpp/numeric/valarray"). In this case, the replacement type has the following properties: 

    

  * All const member functions of [std::valarray](../valarray.html "cpp/numeric/valarray") are provided. 
  * [std::valarray](../valarray.html "cpp/numeric/valarray"), [std::slice_array](slice_array.html "cpp/numeric/valarray/slice array"), [std::gslice_array](gslice_array.html "cpp/numeric/valarray/gslice array"), [std::mask_array](mask_array.html "cpp/numeric/valarray/mask array") and [std::indirect_array](indirect_array.html "cpp/numeric/valarray/indirect array") can be constructed from the replacement type. 
  * For every function taking a const [std::valarray](../valarray.html)<T>& except [`begin()`](begin2.html "cpp/numeric/valarray/begin2") and [`end()`](end2.html "cpp/numeric/valarray/end2")(since C++11), identical functions taking the replacement types shall be added; 
  * For every function taking two const [std::valarray](../valarray.html)<T>& arguments, identical functions taking every combination of const [std::valarray](../valarray.html)<T>& and replacement types shall be added. 
  * The return type does not add more than two levels of template nesting over the most deeply-nested argument type. 



### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/numeric/valarray/operator_arith3&action=edit&section=4 "Edit section: Example")] Example

Finds real roots of multiple [Quadratic equations](https://en.wikipedia.org/wiki/Quadratic_equation "enwiki:Quadratic equation").

Run this code
    
    
    #include <cstddef>
    #include <iostream>
    #include <valarray>
     
    int main()
    {
        [std::valarray](../valarray.html)<double> a(1, 8);
        [std::valarray](../valarray.html)<double> b{1, 2, 3, 4, 5, 6, 7, 8};
        [std::valarray](../valarray.html)<double> c = -b;
        // literals must also be of type T until LWG3074 (double in this case)
        [std::valarray](../valarray.html)<double> d = [std::sqrt](../math/sqrt.html)(b * b - 4.0 * a * c);
        [std::valarray](../valarray.html)<double> x1 = 2.0 * c / (-b + d);
        [std::valarray](../valarray.html)<double> x2 = 2.0 * c / (-b - d);
        [std::cout](../../io/cout.html) << "quadratic equation:  root 1:    root 2:   b: c:\n";
        for ([std::size_t](../../types/size_t.html) i = 0; i < a.size(); ++i)
            [std::cout](../../io/cout.html) << a[i] << "\u00B7x\u00B2 + " << b[i] << "\u00B7x + "
                      << c[i] << " = 0  " << [std::fixed](../../io/manip/fixed.html) << x1[i]
                      << "  " << x2[i] << [std::defaultfloat](../../io/manip/fixed.html)
                      << "  " << -x1[i] - x2[i]
                      << "  " << x1[i] * x2[i] << '\n';
    }

Output: 
    
    
    quadratic equation:  root 1:    root 2:   b: c:
    1·x² + 1·x + -1 = 0  -1.618034  0.618034  1  -1
    1·x² + 2·x + -2 = 0  -2.732051  0.732051  2  -2
    1·x² + 3·x + -3 = 0  -3.791288  0.791288  3  -3
    1·x² + 4·x + -4 = 0  -4.828427  0.828427  4  -4
    1·x² + 5·x + -5 = 0  -5.854102  0.854102  5  -5
    1·x² + 6·x + -6 = 0  -6.872983  0.872983  6  -6
    1·x² + 7·x + -7 = 0  -7.887482  0.887482  7  -7
    1·x² + 8·x + -8 = 0  -8.898979  0.898979  8  -8
