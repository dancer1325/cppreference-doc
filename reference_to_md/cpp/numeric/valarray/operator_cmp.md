Compares each value within the numeric array with another value. 

1) Returns a numeric array of bool containing elements each of which is obtained by applying the indicated comparison operator to the corresponding values of lhs and rhs.

The behavior is undefined if size() != v.size(). 

2) Returns a numeric array of bool containing elements each of which is obtained by applying the indicated comparison operator to lhsv and the corresponding value of rhs.

3) Returns a numeric array of bool containing elements each of which is obtained by applying the indicated comparison operator to the corresponding value of lhs and rhsv.

May throw implementation-defined exceptions. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/numeric/valarray/operator_cmp&action=edit&section=3 "Edit section: Notes")] Notes

Each of the operators can only be instantiated if the following requirements are met: 

    

  * The indicated operator can be applied to type `T`. 
  * The result value can be unambiguously converted to bool. 



The function can be implemented with the return type different from [std::valarray](../valarray.html "cpp/numeric/valarray"). In this case, the replacement type has the following properties: 

    

  * All const member functions of [std::valarray](../valarray.html "cpp/numeric/valarray") are provided. 
  * [std::valarray](../valarray.html "cpp/numeric/valarray"), [std::slice_array](slice_array.html "cpp/numeric/valarray/slice array"), [std::gslice_array](gslice_array.html "cpp/numeric/valarray/gslice array"), [std::mask_array](mask_array.html "cpp/numeric/valarray/mask array") and [std::indirect_array](indirect_array.html "cpp/numeric/valarray/indirect array") can be constructed from the replacement type. 
  * For every function taking a const [std::valarray](../valarray.html)<T>& except [`begin()`](begin2.html "cpp/numeric/valarray/begin2") and [`end()`](end2.html "cpp/numeric/valarray/end2")(since C++11), identical functions taking the replacement types shall be added; 
  * For every function taking two const [std::valarray](../valarray.html)<T>& arguments, identical functions taking every combination of const [std::valarray](../valarray.html)<T>& and replacement types shall be added. 
  * The return type does not add more than two levels of template nesting over the most deeply-nested argument type. 



### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/numeric/valarray/operator_cmp&action=edit&section=4 "Edit section: Example")] Example

Run this code
    
    
    #include <iostream>
    #include <valarray>
     
    int main()
    {
        // zero all negatives in a valarray
        [std::valarray](../valarray.html)<int> v = {1, -1, 0, -3, 10, -1, -2};
        [std::cout](../../io/cout.html) << "Before: ";
        for (auto n : v)
            [std::cout](../../io/cout.html) << n << ' ';
        [std::cout](../../io/cout.html) << '\n';
        v[v < 0] = 0;
        [std::cout](../../io/cout.html) << "After: ";
        for (auto n : v)
            [std::cout](../../io/cout.html) << n << ' ';
        [std::cout](../../io/cout.html) << '\n';
     
        // convert the valarray<bool> result of == to a single bool
        [std::valarray](../valarray.html)<int> a = {1, 2, 3};
        [std::valarray](../valarray.html)<int> b = {2, 4, 6};
     
        [std::cout](../../io/cout.html) << "2*a == b is " << [std::boolalpha](../../io/manip/boolalpha.html)
                  << (2 * a == b).min() << '\n';
    }

Output: 
    
    
    Before: 1 -1 0 -3 10 -1 -2
    After: 1 0 0 0 10 0 0
    2*a == b is true
