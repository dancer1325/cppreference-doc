Returns an iterator to the reverse-beginning of the given range. 

1,2) Returns c.rbegin(), which is typically an iterator to the reverse-beginning of the sequence represented by c.

1) If `C` is a standard [Container](../named_req/Container.html "cpp/named req/Container"), returns a `C::reverse_iterator` object.

2) If `C` is a standard [Container](../named_req/Container.html "cpp/named req/Container"), returns a `C::const_reverse_iterator` object.

3) Returns a [std::reverse_iterator](reverse_iterator.html)<T*> object to the reverse-beginning of array.

4) Returns a [std::reverse_iterator](reverse_iterator.html)<const T*> object to the reverse-beginning of il.

5) Returns std::rbegin(c), with c always treated as const-qualified.

If `C` is a standard [Container](../named_req/Container.html "cpp/named req/Container"), returns a `C::const_reverse_iterator` object.

### Parameters

c  |  \-  |  a container or view with a `rbegin` member function   
---|---|---  
array  |  \-  |  an array of arbitrary type   
il  |  \-  |  an [std::initializer_list](../utility/initializer_list.html)  
  
### Return value

1,2) c.rbegin()

3) [std::reverse_iterator](reverse_iterator.html)<T*>(array + N)

4) [std::reverse_iterator](reverse_iterator.html)<const T*>(il.end())

5) c.rbegin()

### Exceptions

May throw implementation-defined exceptions. 

### Overloads

Custom overloads of `rbegin` may be provided for classes and enumerations that do not expose a suitable `rbegin()` member function, yet can be iterated. 

Overloads of `rbegin` found by [argument-dependent lookup](../language/adl.html "cpp/language/adl") can be used to customize the behavior of std::[ranges::rbegin](../ranges/rbegin.html) and [std::ranges::crbegin](../ranges/crbegin.html "cpp/ranges/crbegin").  | (since C++20)  
---|---  
  
### Notes

The overload for [std::initializer_list](../utility/initializer_list.html "cpp/utility/initializer list") is necessary because it does not have a member function `rbegin`. 

### Example

Run this code
    
    
    #include <iostream>
    #include <iterator>
    #include <vector>
     
    int main()
    {
        [std::vector](../container/vector.html)<int> v = {3, 1, 4};
        auto vi = std::rbegin(v); // the type of “vi” is std::vector<int>::reverse_iterator
        [std::cout](../io/cout.html) << "*vi = " << *vi << '\n';
     
        *std::rbegin(v) = 42; // OK: after assignment v[2] == 42
    //  *std::crbegin(v) = 13; // error: the location is read-only
     
        int a[] = {-5, 10, 15};
        auto ai = std::rbegin(a); // the type of “ai” is std::reverse_iterator<int*>
        [std::cout](../io/cout.html) << "*ai = " << *ai << '\n';
     
        auto il = {3, 1, 4};
        // the type of “it” below is std::reverse_iterator<int const*>:
        for (auto it = std::rbegin(il); it != [std::rend](rend.html)(il); ++it)
            [std::cout](../io/cout.html) << *it << ' ';
        [std::cout](../io/cout.html) << '\n';
    }

Output: 
    
    
    *vi = 4
    *ai = 15
    4 1 3
