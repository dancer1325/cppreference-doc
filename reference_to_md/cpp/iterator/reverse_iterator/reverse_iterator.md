### Example

Run this code
    
    
    #include <cassert>
    #include <concepts>
    #include <iterator>
    #include <vector>
     
    int main()
    {
        [std::vector](../../container/vector.html) v{0, 1, 2, 3, 4};
     
        using ReverseIt = [std::reverse_iterator](../reverse_iterator.html)<[std::vector](../../container/vector.html)<int>::const_iterator>;
     
        ReverseIt i1; // overload (1)
        i1 = v.crbegin();
        [assert](../../error/assert.html)(*i1 == 4);
     
        ReverseIt i2(i1); // overload (2)
        [assert](../../error/assert.html)(i2[0] == 4);
     
        int x[]{1, 2, 3};
        auto i3 = [std::reverse_iterator](../reverse_iterator.html)<int*>(x + [std::size](../size.html)(x)); // overload (1)
        i3[0] = -3;
        [assert](../../error/assert.html)(x[2] == -3);
        [std::reverse_iterator](../reverse_iterator.html)<int const*> i4(i3); // overload (3): int => const int
        static_assert([std::convertible_to](../../concepts/convertible_to.html)<decltype(i3)::value_type,
                                          decltype(i4)::value_type>);
        // i4[0] = 5; // Error: assignment of read-only location
    }

### Defect reports

The following behavior-changing defect reports were applied retroactively to previously published C++ standards. 

DR  | Applied to  | Behavior as published  | Correct behavior   
---|---|---|---  
[LWG 235](https://cplusplus.github.io/LWG/issue235) | C++98  | the effect of overload ([1](reverse_iterator.html#Version_1)) was not specified  | specified   
[LWG 1012](https://cplusplus.github.io/LWG/issue1012) | C++98  | overload ([1](reverse_iterator.html#Version_1)) default-initialized `[current](../reverse_iterator.html#current "cpp/iterator/reverse iterator")` | it is value-initialized   
[LWG 3435](https://cplusplus.github.io/LWG/issue3435) | C++20  | overload ([3](reverse_iterator.html#Version_3)) was not constrained  | constrained
