`std::ostreambuf_iterator` is a single-pass [LegacyOutputIterator](../named_req/OutputIterator.html "cpp/named req/OutputIterator") that writes successive characters into the [std::basic_streambuf](../io/basic_streambuf.html "cpp/io/basic streambuf") object for which it was constructed. The actual write operation is performed when the iterator (whether dereferenced or not) is assigned to. Incrementing the `std::ostreambuf_iterator` is a no-op. 

In a typical implementation, the only data members of `std::ostreambuf_iterator` are a pointer to the associated `std::basic_streambuf` and a boolean flag indicating if the end of file condition has been reached.
