Returns a pointer to the block of memory containing the elements of the range. 

1,2) Returns c.data().

3) Returns array.

4) Returns il.begin().

### Parameters

c  |  \-  |  a container or view with a data() member function   
---|---|---  
array  |  \-  |  an array of arbitrary type   
il  |  \-  |  an [std::initializer_list](../utility/initializer_list.html "cpp/utility/initializer list")  
  
### Return value

1,2) c.data()

3) array

4) il.begin()

### Exceptions

1) May throw implementation-defined exceptions.

### Notes

The overload for [std::initializer_list](../utility/initializer_list.html "cpp/utility/initializer list") is necessary because it does not have a member function `data`. 

[Feature-test](../utility/feature_test.html "cpp/utility/feature test") macro | Value | Std | Feature   
---|---|---|---  
[`__cpp_lib_nonmember_container_access`](../experimental/feature_test.html#cpp_lib_nonmember_container_access "cpp/feature test") | [`201411L`](../compiler_support/17.html#cpp_lib_nonmember_container_access_201411L "cpp/compiler support/17") | (C++17) | [std::size()](size.html "cpp/iterator/size"), `std::data()`, and [std::empty()](empty.html "cpp/iterator/empty")  
  
### Possible implementation

[First version](data.html#Version_1)  
---  
      
    
    template<class C>
    constexpr auto data(C& c) -> decltype(c.data())
    {
        return c.data();
    }  
  
[Second version](data.html#Version_2)  
      
    
    template<class C>
    constexpr auto data(const C& c) -> decltype(c.data())
    {
        return c.data();
    }  
  
[Third version](data.html#Version_3)  
      
    
    template<class T, [std::size_t](../types/size_t.html) N>
    constexpr T* data(T (&array)[N]) noexcept
    {
        return array;
    }  
  
[Fourth version](data.html#Version_4)  
      
    
    template<class E>
    constexpr const E* data([std::initializer_list](../utility/initializer_list.html)<E> il) noexcept
    {
        return il.begin();
    }  
  
### Example

Run this code
    
    
    #include <cstring>
    #include <iostream>
    #include <string>
     
    int main()
    {
        [std::string](../string/basic_string.html) s{"Hello world!\n"};
     
        char a[20]; // storage for a C-style string
        [std::strcpy](../string/byte/strcpy.html)(a, std::data(s));
    //  [s.data(), s.data() + s.size()] is guaranteed to be an NTBS since C++11
     
        [std::cout](../io/cout.html) << a;
    }

Output: 
    
    
    Hello world!
