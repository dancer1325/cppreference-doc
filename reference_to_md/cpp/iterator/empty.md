Returns whether the given range is empty. 

1) Returns c.empty().

2) Returns false.

3) Returns il.size() == 0.

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/iterator/empty&action=edit&section=4 "Edit section: Notes")] Notes

The overload for [std::initializer_list](../utility/initializer_list.html "cpp/utility/initializer list") is necessary because it does not have a member function `empty`.

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/iterator/empty&action=edit&section=5 "Edit section: Possible implementation")] Possible implementation

[First version](empty.html#Version_1)  
---  
      
    
    template<class C>
    [[nodiscard]] constexpr auto empty(const C& c) -> decltype(c.empty())
    {
        return c.empty();
    }  
  
[Second version](empty.html#Version_2)  
      
    
    template<class T, [std::size_t](../types/size_t.html) N>
    [[nodiscard]] constexpr bool empty(const T (&array)[N]) noexcept
    {
        return false;
    }  
  
[Third version](empty.html#Version_3)  
      
    
    template<class E>
    [[nodiscard]] constexpr bool empty([std::initializer_list](../utility/initializer_list.html)<E> il) noexcept
    {
        return il.size() == 0;
    }  
  
### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/iterator/empty&action=edit&section=6 "Edit section: Example")] Example

Run this code
    
    
    #include <iostream>
    #include <vector>
     
    template<class T>
    void print(const T& container)
    {
        if (std::empty(container))
            [std::cout](../io/cout.html) << "Empty\n";
        else
        {
            [std::cout](../io/cout.html) << "Elements:";
            for (const auto& element : container)
                [std::cout](../io/cout.html) << ' ' << element;
            [std::cout](../io/cout.html) << '\n';
        }
    }
     
    int main()
    {
        [std::vector](../container/vector.html)<int> c = {1, 2, 3};
        print(c);
        c.clear();
        print(c);
     
        int array[] = {4, 5, 6};
        print(array);
     
        auto il = {7, 8, 9};
        print(il);
    }

Output: 
    
    
    Elements: 1 2 3
    Empty
    Elements: 4 5 6
    Elements: 7 8 9
