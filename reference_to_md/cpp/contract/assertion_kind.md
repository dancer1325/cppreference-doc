enum class assertion_kind : /* unspecified */ {  
pre = 1,  
post = 2,  
[assert](../error/assert.html) = 3  
}; |  |  (since C++26)  
| |   
  
The enumerators of `std::contracts::assertion_kind` correspond to the syntactic forms of a [contract assertion](../language/contracts.html "cpp/language/contracts"), with meanings listed below: 

Enumerator  |  Meaning   
---|---  
`pre` |  the contract assertion is a [precondition assertion](../language/function.html#Precondition_assertions "cpp/language/function")  
`post` |  the contract assertion is a [postcondition assertion](../language/function.html#Postcondition_assertions "cpp/language/function")  
`assert` |  the contract assertion is a [`contract_assert`](../language/contract_assert.html "cpp/language/contract assert") statement
