Replaces the contents of the string. 

1) Replaces the contents with a copy of str. If *this and str are the same object, this function has no effect.

2) Replaces the contents with those of str using [SequenceContainer](../../named_req/SequenceContainer.html "cpp/named req/SequenceContainer")'s move assignment semantics.

Unlike other sequence container move assignments, references, pointers, and iterators to elements of str may be invalidated.

3) Replaces the contents with those of null-terminated character string pointed to by s as if by assign(s, Traits::length(s)).

4) Replaces the contents with character ch as if by assign([std::addressof](../../memory/addressof.html)(ch), 1).

5) Replaces the contents with those of the initializer list ilist as if by assign(ilist.begin(), ilist.size()).

6) Implicitly converts t to a string view sv as if by [std::basic_string_view](../basic_string_view.html)<CharT, Traits> sv = t;, then replaces the contents with those of the sv as if by assign(sv).

This overload participates in overload resolution only if [std::is_convertible_v](../../types/is_convertible.html)<const StringViewLike&,  
[std::basic_string_view](../basic_string_view.html)<CharT, Traits>> is true and [std::is_convertible_v](../../types/is_convertible.html)<const StringViewLike&, const CharT*> is false.

7) `std::basic_string` cannot be assigned from nullptr.

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/string/basic_string/operator%3D&action=edit&section=2 "Edit section: Return value")] Return value

*this

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/string/basic_string/operator%3D&action=edit&section=3 "Edit section: Complexity")] Complexity

1) Linear in size of str.

2) Linear in the size of *this (formally, each `CharT` has to be destroyed). If allocators do not compare equal and do not propagate, then also linear in the size of str (copy must be made).

3) Linear in size of s.

4) Constant.

5) Linear in size of ilist.

6) Linear in size of t.

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/string/basic_string/operator%3D&action=edit&section=4 "Edit section: Exceptions")] Exceptions

propagate_on_container_move_assignment::value ||

If an exception is thrown for any reason, this function has no effect ([strong exception safety guarantee](../../language/exceptions.html#Exception_safety "cpp/language/exceptions")). 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/string/basic_string/operator%3D&action=edit&section=5 "Edit section: Example")] Example

Run this code
    
    
    #include <iomanip>
    #include <iostream>
    #include <string>
     
    int main()
    {
        [std::string](../basic_string.html) str1;
        [std::string](../basic_string.html) str2{"alpha"};
     
        // (1) operator=(const basic_string&);
        str1 = str2;
        [std::cout](../../io/cout.html) << [std::quoted](../../io/manip/quoted.html)(str1) << ' '   // "alpha"
                  << [std::quoted](../../io/manip/quoted.html)(str2) << '\n'; // "alpha"
     
        // (2) operator=(basic_string&&);
        str1 = std::move(str2);
        [std::cout](../../io/cout.html) << [std::quoted](../../io/manip/quoted.html)(str1) << ' '   // "alpha"
                  << [std::quoted](../../io/manip/quoted.html)(str2) << '\n'; // "" or "alpha" (unspecified)
     
        // (3) operator=(const CharT*);
        str1 = "beta";
        [std::cout](../../io/cout.html) << [std::quoted](../../io/manip/quoted.html)(str1) << '\n'; // "beta"
     
        // (4) operator=(CharT);
        str1 = '!'; 
        [std::cout](../../io/cout.html) << [std::quoted](../../io/manip/quoted.html)(str1) << '\n'; // "!"
     
        // (5) operator=(std::initializer_list<CharT>);
        str1 = {'g', 'a', 'm', 'm', 'a'};
        [std::cout](../../io/cout.html) << [std::quoted](../../io/manip/quoted.html)(str1) << '\n'; // "gamma"
     
        // (6) operator=(const T&);
        str1 = 35U; // equivalent to str1 = static_cast<char>(35U);
        [std::cout](../../io/cout.html) << [std::quoted](../../io/manip/quoted.html)(str1) << '\n'; // "#" (ASCII = 35)
    }

Possible output: 
    
    
    "alpha" "alpha"
    "alpha" ""
    "beta"
    "!"
    "gamma"
    "#"

DR  | Applied to  | Behavior as published  | Correct behavior

[LWG 2063](https://cplusplus.github.io/LWG/issue2063) | C++11  | the move assignment operator did not follow  
[SequenceContainer](../../named_req/SequenceContainer.html "cpp/named req/SequenceContainer")'s semantic requirement  | follows   
[LWG 2946](https://cplusplus.github.io/LWG/issue2946) | C++17  | overload ([6](operator=.html#Version_6)) caused ambiguity in some cases  | avoided by making it a template
