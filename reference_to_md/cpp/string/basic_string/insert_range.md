template< [container-compatible-range](../../ranges/to.html)<CharT> R >  
constexpr iterator insert_range( const_iterator pos, R&& rg ); |  |  (since C++23)  
---|---|---  
| |   
  
Inserts characters from the range rg before the element (if any) pointed by pos. 

Equivalent to 
    
    
    return insert(pos - begin(),
        [std::basic_string](../basic_string.html)(
            [std::from_range](../../ranges/from_range.html),
            std​::​forward<R>(rg),
            get_allocator())
    );

If pos is not a valid iterator on *this, the behavior is undefined.

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/string/basic_string/insert_range&action=edit&section=1 "Edit section: Parameters")] Parameters

pos  |  \-  |  iterator before which the characters will be inserted   
---|---|---  
rg  |  \-  |  a [container compatible range](../../ranges/to.html#container_compatible_range "cpp/ranges/to")  
  
### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/string/basic_string/insert_range&action=edit&section=2 "Edit section: Return value")] Return value

An iterator which refers to the first inserted character, or pos if no characters were inserted because rg was empty. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/string/basic_string/insert_range&action=edit&section=3 "Edit section: Complexity")] Complexity

Linear in size of rg. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/string/basic_string/insert_range&action=edit&section=4 "Edit section: Exceptions")] Exceptions

If [std::allocator_traits](../../memory/allocator_traits.html)<Allocator>::allocate throws an exception, it is rethrown. 

If the operation would cause [`size()`](size.html "cpp/string/basic string/size") to exceed [`max_size()`](max_size.html "cpp/string/basic string/max size"), throws [std::length_error](../../error/length_error.html "cpp/error/length error"). 

If an exception is thrown for any reason, this function has no effect ([strong exception safety guarantee](../../language/exceptions.html#Exception_safety "cpp/language/exceptions")).

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/string/basic_string/insert_range&action=edit&section=6 "Edit section: Example")] Example

Run this code
    
    
    #include <cassert>
    #include <iterator>
    #include <string>
     
    int main()
    {
        const auto source = {'l', 'i', 'b', '_'};
        [std::string](../basic_string.html) target{"__cpp_containers_ranges"};
        //                        ^insertion will occur
        //                         before this position
     
        const auto pos = target.find("container");
        [assert](../../error/assert.html)(pos != target.npos);
        auto iter = [std::next](../../iterator/next.html)(target.begin(), pos);
     
    #ifdef __cpp_lib_containers_ranges
        target.insert_range(iter, source);
    #else
        target.insert(iter, source.begin(), source.end());
    #endif
     
        [assert](../../error/assert.html)(target == "__cpp_lib_containers_ranges");
        //                      ^^^^
    }
