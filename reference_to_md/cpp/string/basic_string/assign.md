Replaces the contents of the string. 

1) Equivalent to return *this = str;.

2) Equivalent to return *this = std::move(str);.

3) Replaces the contents with count copies of character ch.

Equivalent to clear(); resize(n, c); return *this;.

4) Replaces the contents with copies of the characters in the range `[`s`, `s + count`)`.

If `[`s`, `s + count`)` is not a [valid range](../../iterator.html#Ranges "cpp/iterator"), the behavior is undefined.

5) Equivalent to return assign(s, Traits::length(s));.

6,7) Replaces the contents with characters in a string view sv constructed from t. 

  * If only t is provided, replaces the contents with all characters in sv. 
  * If pos is also provided: 
    * If count is [`npos`](../basic_string.html#Data_members "cpp/string/basic string"), replaces the contents with all characters in sv starting from pos. 
    * Otherwise, replaces the contents with the [std::min](../../algorithm/min.html)(count, sv.size() - pos) characters in sv starting from pos.



These overloads participate in overload resolution only if all following conditions are satisfied: 

  * [std::is_convertible_v](../../types/is_convertible.html)<const SV&, [std::basic_string_view](../basic_string_view.html)<CharT, Traits>> is true. 
  * [std::is_convertible_v](../../types/is_convertible.html)<const SV&, const CharT*> is false.



6) Equivalent to [std::basic_string_view](../basic_string_view.html)<CharT, Traits> sv = t;  
return assign(sv.data(), sv.size());.

7) Equivalent to [std::basic_string_view](../basic_string_view.html)<CharT, Traits> sv = t;  
return assign(sv.substr(pos, count));.

8) Replaces the contents with characters in str. 

  * If count is [`npos`](../basic_string.html#Data_members "cpp/string/basic string"), replaces the contents with all characters in str starting from pos. 
  * Otherwise, replaces the contents with the [std::min](../../algorithm/min.html)(count, str.size() - pos) characters in str starting from pos.



Equivalent to return assign([std::basic_string_view](../basic_string_view.html)<CharT, Traits>  
(str).substr(pos, count));. | (since C++20)

### Parameters

str  |  \-  |  string to be used as source to initialize the characters with   
---|---|---  
count  |  \-  |  size of the resulting string   
ch  |  \-  |  value to initialize characters of the string with   
s  |  \-  |  pointer to a character string to use as source to initialize the string with   
t  |  \-  |  object (convertible to [std::basic_string_view](../basic_string_view.html "cpp/string/basic string view")) to initialize the characters of the string with   
pos  |  \-  |  index of the first character to take   
first, last  |  \-  |  range to copy the characters from   
ilist  |  \-  |  [std::initializer_list](../../utility/initializer_list.html "cpp/utility/initializer list") to initialize the characters of the string with

*this

### Example

Run this code
    
    
    #include <iostream>
    #include <iterator>
    #include <string>
     
    int main()
    {
        [std::string](../basic_string.html) s;
        // assign(size_type count, CharT ch)
        s.assign(4, '=');
        [std::cout](../../io/cout.html) << s << '\n'; // "===="
     
        [std::string](../basic_string.html) const c("Exemplary");
        // assign(const basic_string& str)
        s.assign(c);
        [std::cout](../../io/cout.html) << c << " == " << s << '\n'; // "Exemplary == Exemplary"
     
        // assign(const basic_string& str, size_type pos, size_type count)
        s.assign(c, 0, c.length() - 1);
        [std::cout](../../io/cout.html) << s << '\n'; // "Exemplar";
     
        // assign(basic_string&& str)
        s.assign([std::string](../basic_string.html)("C++ by ") + "example");
        [std::cout](../../io/cout.html) << s << '\n'; // "C++ by example"
     
        // assign(const CharT* s, size_type count)
        s.assign("C-style string", 7);
        [std::cout](../../io/cout.html) << s << '\n'; // "C-style"
     
        // assign(const CharT* s)
        s.assign("C-style\0string");
        [std::cout](../../io/cout.html) << s << '\n'; // "C-style"
     
        char mutable_c_str[] = "C-style string";
        // assign(InputIt first, InputIt last)
        s.assign([std::begin](../../iterator/begin.html)(mutable_c_str), [std::end](../../iterator/end.html)(mutable_c_str) - 1);
        [std::cout](../../io/cout.html) << s << '\n'; // "C-style string"
     
        // assign(std::initializer_list<CharT> ilist)
        s.assign({'C', '-', 's', 't', 'y', 'l', 'e'});
        [std::cout](../../io/cout.html) << s << '\n'; // "C-style"
    }

Output: 
    
    
    ====
    Exemplary == Exemplary
    Exemplar
    C++ by example
    C-style
    C-style
    C-style string
    C-style

### Defect reports

The following behavior-changing defect reports were applied retroactively to previously published C++ standards. 

DR  | Applied to  | Behavior as published  | Correct behavior   
---|---|---|---  
[LWG 847](https://cplusplus.github.io/LWG/issue847) | C++98  | there was no exception safety guarantee  | added strong exception safety guarantee   
[LWG 2063](https://cplusplus.github.io/LWG/issue2063) | C++11  | non-normative note stated that overload ([2](assign.html#Version_2))  
can be implemented by swapping  | corrected to require move assignment   
[LWG 2250](https://cplusplus.github.io/LWG/issue2250) | C++98  | the behavior of overload ([8](assign.html#Version_8)) was  
undefined if pos > str.size() is true | always throws an exception in this case   
[LWG 2579](https://cplusplus.github.io/LWG/issue2579) | C++98  | overload ([1](assign.html#Version_1)) and the copy assignment  
operator had different effects  | they have the same effect   
[LWG 2946](https://cplusplus.github.io/LWG/issue2946) | C++17  | overload ([6](assign.html#Version_6)) caused ambiguity in some cases  | avoided by making it a template
