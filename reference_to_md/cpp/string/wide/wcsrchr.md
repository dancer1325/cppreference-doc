const wchar_t* wcsrchr( const wchar_t* str, wchar_t ch ); |  |   
wchar_t* wcsrchr( wchar_t* str, wchar_t ch ); |  |   
| |   
  
Finds the last occurrence of the wide character ch in the wide string pointed to by str. 

## Contents

  * [1 Parameters](wcsrchr.html#Parameters)
  * [2 Return value](wcsrchr.html#Return_value)
  * [3 Example](wcsrchr.html#Example)
  * [4 See also](wcsrchr.html#See_also)

  
---  
  
### Parameters

str  |  \-  |  pointer to the null-terminated wide string to be analyzed   
---|---|---  
ch  |  \-  |  wide character to search for   
  
### Return value

Pointer to the found character in str, or a null pointer if no such character is found. 

### Example

Run this code
    
    
    #include <cwchar>
    #include <iostream>
    #include <locale>
     
    int main()
    {
        const wchar_t arr[] = L"白猫 黒猫 кошки";
        const wchar_t* cat = std::wcsrchr(arr, L'猫');
        const wchar_t* dog = std::wcsrchr(arr, L'犬');
     
        [std::cout](../../io/cout.html).imbue([std::locale](../../locale/locale.html)("en_US.utf8"));
     
        if (cat)
            [std::cout](../../io/cout.html) << "The character 猫 found at position " << cat - arr << '\n';
        else
            [std::cout](../../io/cout.html) << "The character 猫 not found\n";
     
        if (dog)
            [std::cout](../../io/cout.html) << "The character 犬 found at position " << dog - arr << '\n';
        else
            [std::cout](../../io/cout.html) << "The character 犬 not found\n";
    }

Output: 
    
    
    The character 猫 found at position 4
    The character 犬 not found
