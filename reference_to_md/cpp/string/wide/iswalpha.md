int iswalpha( [std::wint_t](../wide.html) ch ); |  |   
| |   
  
Checks if the given wide character is an alphabetic character, i.e. either an uppercase letter (`ABCDEFGHIJKLMNOPQRSTUVWXYZ`), a lowercase letter (`abcdefghijklmnopqrstuvwxyz`) or any alphabetic character specific to the current locale. 

If the value of ch is neither representable as a wchar_t nor equal to the value of the macro WEOF, the behavior is undefined. 

## Contents

  * [1 Parameters](iswalpha.html#Parameters)
  * [2 Return value](iswalpha.html#Return_value)
  * [3 Notes](iswalpha.html#Notes)
  * [4 Example](iswalpha.html#Example)
  * [5 See also](iswalpha.html#See_also)

  
---  
  
### Parameters

ch  |  \-  |  wide character   
---|---|---  
  
### Return value

Non-zero value if the wide character is an alphabetic character, ​0​ otherwise. 

### Notes

[ISO 30112](https://www.open-std.org/JTC1/SC35/WG5/docs/30112d10.pdf) specifies which Unicode characters are include in POSIX alpha category. 

### Example

Run this code
    
    
    #include <clocale>
    #include <cwctype>
    #include <iostream>
     
    int main()
    {
        wchar_t c = L'\u0b83'; // Tamil sign Visarga ('ஃ')
     
        [std::cout](../../io/cout.html) << [std::hex](../../io/manip/hex.html) << [std::showbase](../../io/manip/showbase.html) << [std::boolalpha](../../io/manip/boolalpha.html);
        [std::cout](../../io/cout.html) << "in the default locale, iswalpha(" << ([std::wint_t](../wide.html))c << ") = "
                  << (bool)std::iswalpha(c) << '\n';
     
        [std::setlocale](../../locale/setlocale.html)([LC_ALL](../../locale/LC_categories.html), "en_US.utf8");
        [std::cout](../../io/cout.html) << "in Unicode locale, iswalpha(" << ([std::wint_t](../wide.html))c << ") = "
                  << (bool)std::iswalpha(c) << '\n';
    }

Possible output: 
    
    
    in the default locale, iswalpha(0xb83) = false
    in Unicode locale, iswalpha(0xb83) = true
