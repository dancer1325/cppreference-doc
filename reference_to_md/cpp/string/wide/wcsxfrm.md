[std::size_t](../../types/size_t.html) wcsxfrm( wchar_t* dest, const wchar_t* src, [std::size_t](../../types/size_t.html) count ); |  |   
| |   
  
Transforms the null-terminated wide string pointed to by src into the implementation-defined form such that comparing two transformed strings with [std::wcscmp](wcscmp.html "cpp/string/wide/wcscmp") gives the same result as comparing the original strings with [std::wcscoll](wcscoll.html "cpp/string/wide/wcscoll"), in the current C locale. 

The first count characters of the transformed string are written to destination, including the terminating null character, and the length of the full transformed string is returned, excluding the terminating null character. 

If count is ​0​, then dest is allowed to be a null pointer.

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/string/wide/wcsxfrm&action=edit&section=1 "Edit section: Notes")] Notes

The correct length of the buffer that can receive the entire transformed string is 1 + std::wcsxfrm(nullptr, src, 0). 

This function is used when making multiple locale-dependent comparisons using the same wide string or set of wide strings, because it is more efficient to use `std::wcsxfrm` to transform all the strings just once, and subsequently compare the transformed wide strings with [std::wcscmp](wcscmp.html "cpp/string/wide/wcscmp"). 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/string/wide/wcsxfrm&action=edit&section=2 "Edit section: Parameters")] Parameters

dest  |  \-  |  pointer to the first element of a wide null-terminated string to write the transformed string to   
---|---|---  
src  |  \-  |  pointer to the null-terminated wide character string to transform   
count  |  \-  |  maximum number of characters to output   
  
### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/string/wide/wcsxfrm&action=edit&section=3 "Edit section: Return value")] Return value

The length of the transformed wide string, not including the terminating null-character. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/string/wide/wcsxfrm&action=edit&section=4 "Edit section: Example")] Example

Run this code
    
    
    #include <cwchar>
    #include <iostream>
     
    int main()
    {
        [std::setlocale](../../locale/setlocale.html)([LC_ALL](../../locale/LC_categories.html), "sv_SE.utf8");
     
        [std::wstring](../basic_string.html) in1 = L"\u00e5r";
        [std::wstring](../basic_string.html) out1(1 + std::wcsxfrm(nullptr, in1.c_str(), 0), L' ');
        [std::wstring](../basic_string.html) in2 = L"\u00e4ngel";
        [std::wstring](../basic_string.html) out2(1 + std::wcsxfrm(nullptr, in2.c_str(), 0), L' ');
     
        std::wcsxfrm(&out1[0], in1.c_str(), out1.size());
        std::wcsxfrm(&out2[0], in2.c_str(), out2.size());
     
        [std::wcout](../../io/cout.html) << "In the Swedish locale: ";
        if (out1 < out2)
            [std::wcout](../../io/cout.html) << in1 << " before " << in2 << '\n';
        else
            [std::wcout](../../io/cout.html) << in2 << " before " << in1 << '\n';
     
        [std::wcout](../../io/cout.html) << "In lexicographical comparison: ";
        if (in1 < in2)
            [std::wcout](../../io/cout.html) << in1 << " before " << in2 << '\n';
        else
            [std::wcout](../../io/cout.html) << in2 << " before " << in1 << '\n';
     
    }

Output: 
    
    
    In the Swedish locale: år before ängel
    In lexicographical comparison: ängel before år
