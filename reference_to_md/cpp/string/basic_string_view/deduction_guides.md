These [deduction guides](../../language/ctad.html "cpp/language/class template argument deduction") are provided for [std::basic_string_view](../basic_string_view.html "cpp/string/basic string view"). 

1) This deduction guide allow the character type to be deduced from the iterator-sentinel pair. This overload participates in overload resolution only if `It` satisfies [`contiguous_iterator`](../../iterator/contiguous_iterator.html "cpp/iterator/contiguous iterator") and `End` satisfies [`sized_sentinel_for`](../../iterator/sized_sentinel_for.html "cpp/iterator/sized sentinel for") for `It`.

2) This deduction guide allow the character type to be deduced from the range. This overload participates in overload resolution only if `R` satisfies [`contiguous_range`](../../ranges/contiguous_range.html "cpp/ranges/contiguous range").

### Example

Run this code
    
    
    #include <array>
    #include <iostream>
    #include <string_view>
     
    int main()
    {
        [std::array](../../container/array.html) a1{'n', 'u', 'c', 'l', 'e', 'o', 'n', 's', ':', '\n'};
        [std::basic_string_view](../basic_string_view.html) s1(a1.cbegin(), a1.cend()); // deduction: CharT -> char
        static_assert([std::is_same_v](../../types/is_same.html)<decltype(s1)::value_type, char>);
        [std::cout](../../io/cout.html) << s1;
     
        [std::array](../../container/array.html) a2{L'p', L'r', L'o', L't', L'o', L'n', L's', L'\n'};
        [std::basic_string_view](../basic_string_view.html) s2(a2.cbegin(), a2.cend()); // deduction: CharT -> wchar_t
        static_assert([std::is_same_v](../../types/is_same.html)<decltype(s2)::value_type, wchar_t>);
        [std::wcout](../../io/cout.html) << s2;
     
        [std::array](../../container/array.html)<long, 9> a3{'n', 'e', 'u', 't', 'r', 'o', 'n', 's', '\n'};
        [std::basic_string_view](../basic_string_view.html) s3(a3.cbegin(), a3.cend()); // deduction: CharT -> long
        static_assert([std::is_same_v](../../types/is_same.html)<decltype(s3)::value_type, long>);
        for (const auto e : s3)
            [std::cout](../../io/cout.html) << static_cast<char>(e);
    }

Output: 
    
    
    nucleons:
    protons
    neutrons
