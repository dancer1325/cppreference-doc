### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/string/basic_string_view/swap&action=edit&section=4 "Edit section: Example")] Example

Run this code
    
    
    #include <iostream>
    #include <string_view>
     
    int main() 
    {
        [std::string_view](../basic_string_view.html) a = "AAA";
        [std::string_view](../basic_string_view.html) b = "BBBB";
     
        [std::cout](../../io/cout.html) << "Before swap:\n"
                     "a = " << a << "\n"
                     "b = " << b << "\n\n";
     
        a.swap(b);
     
        [std::cout](../../io/cout.html) << "After swap:\n"
                     "a = " << a << "\n"
                     "b = " << b << '\n';
    }

Output: 
    
    
    Before swap:
    a = AAA
    b = BBBB
     
    After swap:
    a = BBBB
    b = AAA
