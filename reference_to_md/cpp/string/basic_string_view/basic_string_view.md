1) Default constructor. Constructs an empty `std::basic_string_view`. After construction, [data()](data.html "cpp/string/basic string view/data") is equal to nullptr, and [size()](size.html "cpp/string/basic string view/size") is equal to ​0​.

2) Copy constructor. Constructs a view of the same content as other. After construction, [data()](data.html "cpp/string/basic string view/data") is equal to other.data(), and [size()](size.html "cpp/string/basic string view/size") is equal to other.size().

3) Constructs a view of the first count characters of the character array starting with the element pointed by s. s can contain null characters. The behavior is undefined if `[`s`, `s + count`)` is not a valid range (even though the constructor may not access any of the elements of this range). After construction, [data()](data.html "cpp/string/basic string view/data") is equal to s, and [size()](size.html "cpp/string/basic string view/size") is equal to count.

4) Constructs a view of the null-terminated character string pointed to by s, not including the terminating null character. The length of the view is determined as if by Traits::length(s). The behavior is undefined if `[`s`, `s + Traits::length(s)`)` is not a valid range. After construction, [data()](data.html "cpp/string/basic string view/data") is equal to s, and [size()](size.html "cpp/string/basic string view/size") is equal to Traits::length(s).

5) Constructs a `std::basic_string_view` over the range `[`first`, `last`)`. The behavior is undefined if `[`first`, `last`)` is not a valid range, if `It` does not actually model [`contiguous_iterator`](../../iterator/contiguous_iterator.html "cpp/iterator/contiguous iterator"), or if `End` does not actually model [`sized_sentinel_for`](../../iterator/sized_sentinel_for.html "cpp/iterator/sized sentinel for") for `It`. After construction, [data()](data.html "cpp/string/basic string view/data") is equal to [std::to_address](../../memory/to_address.html)(first), and [size()](size.html "cpp/string/basic string view/size") is equal to last - first. 

This overload participates in overload resolution only if all following conditions are satisfied: 

    

  * `It` satisfies [`contiguous_iterator`](../../iterator/contiguous_iterator.html "cpp/iterator/contiguous iterator"), 
  * `End` satisfies [`sized_sentinel_for`](../../iterator/sized_sentinel_for.html "cpp/iterator/sized sentinel for") for `It`, 
  * [std::iter_value_t](../../iterator/iter_t.html)<It> and `CharT` are the same type, and 
  * `End` is not convertible to [std::size_t](../../types/size_t.html "cpp/types/size t").



6) Constructs a `std::basic_string_view` over the range r. After construction, [data()](data.html "cpp/string/basic string view/data") is equal to [ranges::data](../../ranges/data.html)(r), and [size()](size.html "cpp/string/basic string view/size") is equal to [ranges::size](../../ranges/size.html)(r). 

This overload participates in overload resolution only if all following conditions are satisfied: 

    

  * [std::remove_cvref_t](../../types/remove_cvref.html)<R> is not the same type as `std::basic_string_view`, 
  * `R` models [`contiguous_range`](../../ranges/contiguous_range.html "cpp/ranges/contiguous range") and [`sized_range`](../../ranges/sized_range.html "cpp/ranges/sized range"), 
  * [ranges::range_value_t](../../ranges/range_size_t.html)<R> and `CharT` are the same type, 
  * `R` is not convertible to const CharT*, and 
  * Let d be an lvalue of type [std::remove_cvref_t](../../types/remove_cvref.html)<R>, d.operator ::[std::basic_string_view](../basic_string_view.html)<CharT, Traits>() is not a valid expression.



7) `std::basic_string_view` cannot be constructed from nullptr.

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/string/basic_string_view/basic_string_view&action=edit&section=3 "Edit section: Example")] Example

Run this code
    
    
    #include <array>
    #include <iomanip>
    #include <iostream>
    #include <string>
    #include <string_view>
     
    int main()
    {
        [std::string](../basic_string.html) cppstr = "Foo";
        [std::string_view](../basic_string_view.html) cppstr_v(cppstr); // overload (2), after
                                           // std::string::operator string_view
        [std::cout](../../io/cout.html) << "1) cppstr_v: " << [std::quoted](../../io/manip/quoted.html)(cppstr_v) << '\n';
     
        char array[3] = {'B', 'a', 'r'};
        [std::string_view](../basic_string_view.html) array_v(array, [std::size](../../iterator/size.html)(array)); // overload (3)
        [std::cout](../../io/cout.html) << "2) array_v: " << [std::quoted](../../io/manip/quoted.html)(array_v) << '\n';
     
        const char* one_0_two = "One\0Two";
     
        [std::string_view](../basic_string_view.html) one_two_v{one_0_two, 7}; // overload (3)
        [std::cout](../../io/cout.html) << "3) one_two_v: \"";
        for (char c : one_two_v)
            [std::cout](../../io/cout.html) << (c != '\0' ? c : '?');
        [std::cout](../../io/cout.html) << "\", one_two_v.size(): " << one_two_v.size() << '\n';
     
        [std::string_view](../basic_string_view.html) one_v{one_0_two}; // overload (4)
        [std::cout](../../io/cout.html) << "4) one_v: " << [std::quoted](../../io/manip/quoted.html)(one_v) << ", one_v.size(): " 
                  << one_v.size() << '\n';
     
        constexpr [std::wstring_view](../basic_string_view.html) wcstr_v = L"xyzzy"; // overload (4)
        [std::cout](../../io/cout.html) << "5) wcstr_v.size(): " << wcstr_v.size() << '\n';
     
        [std::array](../../container/array.html) ar = {'P', 'u', 'b'};
        [std::string_view](../basic_string_view.html) ar_v(ar.begin(), ar.end()); // overload (5), C++20
        [std::cout](../../io/cout.html) << "6) ar_v: " << [std::quoted](../../io/manip/quoted.html)(ar_v) << '\n';
     
    //  std::string_view ar_v2{ar}; // overload (6), OK in C++23
    //  std::cout << "ar_v2: " << std::quoted(ar_v2) << '\n'; // ar_v2: "Pub"
     
        [[maybe_unused]] auto zero = [] { /* ... */ return nullptr; };
    //  std::string_view s{zero()}; // overload (7), won't compile since C++23
    }

Output: 
    
    
    1) cppstr_v: "Foo"
    2) array_v: "Bar"
    3) one_two_v: "One?Two", one_two_v.size(): 7
    4) one_v: "One", one_v.size(): 3
    5) wcstr_v.size(): 5
    6) ar_v: "Pub"
