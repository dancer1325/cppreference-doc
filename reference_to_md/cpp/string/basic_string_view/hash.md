Template specializations of [std::hash](../../utility/hash.html "cpp/utility/hash") for the various view classes for hashing views. 

These hashes equal the hashes of corresponding [std::basic_string](../basic_string.html "cpp/string/basic string") classes: If S is one of the standard basic_string types, SV is the corresponding string view type, and s is an object of type S, then [std::hash](../../utility/hash.html)<S>()(s) == [std::hash](../../utility/hash.html)<SV>()(SV(s)). 

### Example

Run this code
    
    
    #include <iostream>
    #include <string_view>
    #include <unordered_set>
    using namespace std::literals;
     
    int main()
    {
        [std::cout](../../io/cout.html) << "\"A\"   #: " << [std::hash](../../utility/hash.html)<[std::string_view](../basic_string_view.html)>{}("A"sv) << '\n';
        [std::cout](../../io/cout.html) << "L\"B\"  #: " << [std::hash](../../utility/hash.html)<[std::wstring_view](../basic_string_view.html)>{}(L"B"sv) << '\n';
        [std::cout](../../io/cout.html) << "u8\"C\" #: " << [std::hash](../../utility/hash.html)<[std::u8string_view](../basic_string_view.html)>{}(u8"C"sv) << '\n';
        [std::cout](../../io/cout.html) << "u\"D\"  #: " << [std::hash](../../utility/hash.html)<[std::u16string_view](../basic_string_view.html)>{}(u"D"sv) << '\n';
        [std::cout](../../io/cout.html) << "U\"E\"  #: " << [std::hash](../../utility/hash.html)<[std::u32string_view](../basic_string_view.html)>{}(U"E"sv) << '\n';
     
        // std::hash for string_view family makes it possible to keep these view-types
        // in unordered_* associative containers, such as unordered_set. But ensure
        // the lifespan of referenced strings is no less than lifespan of the container,
        // i.e. no dangling references occurred.
     
        [std::unordered_set](../../container/unordered_set.html) stars{"Rigel"sv, "Capella"sv, "Vega"sv, "Arcturus"sv};
     
        for ([std::string_view](../basic_string_view.html) const& s : stars)
            [std::cout](../../io/cout.html) << s << ' ';
        [std::cout](../../io/cout.html) << '\n';
    }

Possible output: 
    
    
    "A"   #: 6919333181322027406
    L"B"  #: 11959850520494268278
    u8"C" #: 12432341034569643010
    u"D"  #: 312659256970442235
    U"E"  #: 18073225910249204957
    Arcturus Vega Capella Rigel
