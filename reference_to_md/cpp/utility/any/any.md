Constructs a new `any` object. 

1) Constructs an empty object.

2,3) Copies (2) or moves (3) content of other into a new instance, so that any content is equivalent in both type and value to those of other prior to the constructor call, or empty if other is empty. Formally,

2) If other is empty, the constructed object is empty. Otherwise, equivalent to any([std::in_place_type](../in_place.html)<T>, [std::any_cast](any_cast.html)<const T&>(other)), where `T` is the type of the object contained in other.

3) If other is empty, the constructed object is empty. Otherwise, the constructed object contains either the object contained in other, or an object of the same type constructed from the object contained in other, considering that object as an rvalue.

4) Constructs an object with initial content an object of type [std::decay_t](../../types/decay.html)<ValueType>, [direct-initialized](../../language/direct_initialization.html "cpp/language/direct initialization") from [std::forward](../forward.html)<ValueType>(value). 

  * This overload participates in overload resolution only if [std::decay_t](../../types/decay.html)<ValueType> is not the same type as `any` nor a specialization of [std::in_place_type_t](../in_place.html "cpp/utility/in place"), and [std::is_copy_constructible_v](../../types/is_copy_constructible.html)<[std::decay_t](../../types/decay.html)<ValueType>> is true.



5) Constructs an object with initial content an object of type [std::decay_t](../../types/decay.html)<ValueType>, [direct-non-list-initialized](../../language/direct_initialization.html "cpp/language/direct initialization") from [std::forward](../forward.html)<Args>(args).... 

  * This overload participates in overload resolution only if [std::is_constructible_v](../../types/is_constructible.html)<[std::decay_t](../../types/decay.html)<ValueType>, Args...> and [std::is_copy_constructible_v](../../types/is_copy_constructible.html)<[std::decay_t](../../types/decay.html)<ValueType>> are both true.



6) Constructs an object with initial content an object of type [std::decay_t](../../types/decay.html)<ValueType>, [direct-non-list-initialized](../../language/direct_initialization.html "cpp/language/direct initialization") from il, [std::forward](../forward.html)<Args>(args).... 

  * This overload participates in overload resolution only if [std::is_constructible_v](../../types/is_constructible.html)<[std::decay_t](../../types/decay.html)<ValueType>, [std::initializer_list](../initializer_list.html)<U>&, Args...> and [std::is_copy_constructible_v](../../types/is_copy_constructible.html)<[std::decay_t](../../types/decay.html)<ValueType>> are both true.

Type requirements

### Parameters

other  |  \-  |  another `any` object to copy or move from   
---|---|---  
value  |  \-  |  value to initialize the contained value with   
il, args  |  \-  |  arguments to be passed to the constructor of the contained object   
  
### Exceptions

2,4-6) Throws any exception thrown by the constructor of the contained type.

### Notes

Because the default constructor is constexpr, static `std::any`s are initialized as part of [static non-local initialization](../../language/initialization.html#Non-local_variables "cpp/language/initialization"), before any dynamic non-local initialization begins. This makes it safe to use an object of type `std::any` in a constructor of any static object. 

### Example

Run this code
    
    
    #include <boost/core/demangle.hpp>
     
    #include <any>
    #include <initializer_list>
    #include <iostream>
    #include <memory>
    #include <set>
    #include <string>
    #include <utility>
     
    struct A
    {
        int age;
        [std::string](../../string/basic_string.html) name;
        double salary;
     
    #if __cpp_aggregate_paren_init < 201902L
        // Required before C++20 for in-place construction
        A(int age, [std::string](../../string/basic_string.html) name, double salary)
            : age(age), name(std::move(name)), salary(salary) {}
    #endif
    };
     
    // Using abi demangle to print nice type name of instance of any holding
    void printType(const [std::any](../any.html)& a)
    {
        [std::cout](../../io/cout.html) << boost::core::demangle(a.type().name()) << '\n';
    }
     
    int main()
    {
        // Constructor #4: std::any holding int
        [std::any](../any.html) a1{7};
     
        // Constructor #5: std::any holding A, constructed in place
        [std::any](../any.html) a2([std::in_place_type](../in_place.html)<A>, 30, "Ada", 1000.25);
     
        // Constructor #6: std::any holding a set of A with custom comparison
        auto lambda = [](auto&& l, auto&& r){ return l.age < r.age; };
        [std::any](../any.html) a3(
            [std::in_place_type](../in_place.html)<[std::set](../../container/set.html)<A, decltype(lambda)>>,
            {
                A{39, [std::string](../../string/basic_string.html){"Ada"}, 100.25},
                A{20, [std::string](../../string/basic_string.html){"Bob"}, 75.5}
            },
            lambda);
     
        printType(a1);
        printType(a2);
        printType(a3);
    }

Possible output: 
    
    
    int
    A
    std::set<A, main::{lambda(auto:1&&, auto:2&&)#1}, std::allocator<A> >
