### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/utility/expected/expected&action=edit&section=5 "Edit section: Constraints and supplement information")] Constraints and supplement information

#### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/utility/expected/expected&action=edit&section=6 "Edit section: Primary template constructors")] Primary template constructors

1) This overload participates in overload resolution only if [std::is_default_constructible_v](../../types/is_default_constructible.html)<T> is true.

2) This constructor is defined as deleted unless [std::is_copy_constructible_v](../../types/is_copy_constructible.html)<T> and [std::is_copy_constructible_v](../../types/is_copy_constructible.html)<E> are both true.

This constructor is trivial if [std::is_trivially_copy_constructible_v](../../types/is_copy_constructible.html)<T> and [std::is_trivially_copy_constructible_v](../../types/is_copy_constructible.html)<E> are both true.

3) This overload participates in overload resolution only if [std::is_move_constructible_v](../../types/is_move_constructible.html)<T> and [std::is_move_constructible_v](../../types/is_move_constructible.html)<E> are both true.

This constructor is trivial if [std::is_trivially_move_constructible_v](../../types/is_move_constructible.html)<T> and [std::is_trivially_move_constructible_v](../../types/is_move_constructible.html)<E> are both true.

4,5) These overloads participate in overload resolution only if all following conditions are satisfied: 

  * For overload (4), [std::is_constructible_v](../../types/is_constructible.html)<T, const U&> and [std::is_constructible_v](../../types/is_constructible.html)<E, const G&> are both true. 
  * For overload (5), [std::is_constructible_v](../../types/is_constructible.html)<T, U> and [std::is_constructible_v](../../types/is_constructible.html)<E, G> are both true. 
  * If `T` is not (possibly cv-qualified) bool, the following 8 values are all false: 
    * [std::is_constructible_v](../../types/is_constructible.html)<T, [std::expected](../expected.html)<U, G>&>
    * [std::is_constructible_v](../../types/is_constructible.html)<T, [std::expected](../expected.html)<U, G>>
    * [std::is_constructible_v](../../types/is_constructible.html)<T, const [std::expected](../expected.html)<U, G>&>
    * [std::is_constructible_v](../../types/is_constructible.html)<T, const [std::expected](../expected.html)<U, G>>
    * [std::is_convertible_v](../../types/is_convertible.html)<[std::expected](../expected.html)<U, G>&, T>
    * [std::is_convertible_v](../../types/is_convertible.html)<[std::expected](../expected.html)<U, G>, T>
    * [std::is_convertible_v](../../types/is_convertible.html)<const [std::expected](../expected.html)<U, G>&, T>
    * [std::is_convertible_v](../../types/is_convertible.html)<const [std::expected](../expected.html)<U, G>, T>
  * The following 4 values are all false: 
    * [std::is_constructible_v](../../types/is_constructible.html)<[std::unexpected](unexpected.html)<E>, [std::expected](../expected.html)<U, G>&>
    * [std::is_constructible_v](../../types/is_constructible.html)<[std::unexpected](unexpected.html)<E>, [std::expected](../expected.html)<U, G>>
    * [std::is_constructible_v](../../types/is_constructible.html)<[std::unexpected](unexpected.html)<E>, const [std::expected](../expected.html)<U, G>&>
    * [std::is_constructible_v](../../types/is_constructible.html)<[std::unexpected](unexpected.html)<E>, const [std::expected](../expected.html)<U, G>>



4) This constructor is explicit if [std::is_convertible_v](../../types/is_convertible.html)<const U&, T> or [std::is_convertible_v](../../types/is_convertible.html)<const G&, E> is false.

5) This constructor is explicit if [std::is_convertible_v](../../types/is_convertible.html)<U, T> or [std::is_convertible_v](../../types/is_convertible.html)<G, E> is false.

6) This overload participates in overload resolution only if all following conditions are satisfied: 

  * [std::is_same_v](../../types/is_same.html)<[std::remove_cvref_t](../../types/remove_cvref.html)<U>, [std::in_place_t](../in_place.html)> is false. 
  * [std::is_same_v](../../types/is_same.html)<[std::expected](../expected.html)<T, E>, [std::remove_cvref_t](../../types/remove_cvref.html)<U>> is false. 
  * [std::is_constructible_v](../../types/is_constructible.html)<T, U> is true. 
  * [std::remove_cvref_t](../../types/remove_cvref.html)<U> is not a specialization of [`std::unexpected`](unexpected.html "cpp/utility/expected/unexpected"). 
  * If `T` is (possibly cv-qualified) bool, [std::remove_cvref_t](../../types/remove_cvref.html)<U> is not a specialization of [`std::expected`](../expected.html "cpp/utility/expected").



7) This overload participates in overload resolution only if [std::is_constructible_v](../../types/is_constructible.html)<E, const G&> is true.

8) This overload participates in overload resolution only if [std::is_constructible_v](../../types/is_constructible.html)<E, G> is true.

9) This overload participates in overload resolution only if [std::is_constructible_v](../../types/is_constructible.html)<T, Args...> is true.

10) This overload participates in overload resolution only if [std::is_constructible_v](../../types/is_constructible.html)<T, [std::initializer_list](../initializer_list.html)<U>&, Args...> is true.

11) This overload participates in overload resolution only if [std::is_constructible_v](../../types/is_constructible.html)<E, Args...> is true.

12) This overload participates in overload resolution only if [std::is_constructible_v](../../types/is_constructible.html)<E, [std::initializer_list](../initializer_list.html)<U>&, Args...> is true.
