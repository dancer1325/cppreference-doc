(exposition only*)  
| |   
  
The class template /*range-default-formatter*/ for range types is specialized for formatting range as a string or an escaped string if [std::format_kind](../format_kind.html)<R> is either std::range_format::string or std::range_format::debug_string.

##  std::_range-default-formatter_ <std::range_format::string>::parse

template< class ParseContext >

---|---|---  
| |   
  
Equivalent to: 

auto i = underlying_.parse(ctx);  
if constexpr (K == std::range_format::debug_string)  
underlying_.set_debug_format();  
return i;

Returns an iterator past the end of the _[std-format-spec](../spec.html "cpp/utility/format/spec")_. 

##  std::_range-default-formatter_ <std::range_format::string>::format  
std::_range-default-formatter_ <std::range_format::debug_string>::format

template< class FormatContext >  
auto format( /* see below */& r, FormatContext& ctx ) const -> FormatContext::iterator; |  |   
---|---|---  
| |   
  
If [ranges::input_range](../../../ranges/input_range.html)<const R> is true, the type of r is const R&. Otherwise, the type is R&. 

Let s be a [std::basic_string](../../../string/basic_string.html)<CharT> as if by constructing s with [std::basic_string](../../../string/basic_string.html)<CharT>([std::from_range](../../../ranges/from_range.html), r) such that [ranges::equal](../../../algorithm/ranges/equal.html)(s, r) is true. 

Equivalent to return underlying_.format(s, ctx);. 

Returns an iterator past the end of the output range.
