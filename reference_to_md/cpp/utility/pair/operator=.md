Replaces the contents of the pair. 

1) Copy assignment operator. Replaces the contents with a copy of the contents of other.  The assignment operator is implicitly declared. Using this assignment operator makes the program ill-formed if either `T1` or `T2` is a const-qualified type, or a reference type, or a class type with an inaccessible copy assignment operator, or an array type of such class.  | (until C++11)  
---|---  
This overload is defined as deleted if either [std::is_copy_assignable](../../types/is_copy_assignable.html)<T1>::value or [std::is_copy_assignable](../../types/is_copy_assignable.html)<T2>::value is false.  | (since C++11)  
  
2) Copy assignment operator for const-qualified operand.

This overload participates in overload resolution only if [std::is_copy_assignable_v](../../types/is_copy_assignable.html)<const T1> and [std::is_copy_assignable_v](../../types/is_copy_assignable.html)<const T2> are both true.

3) Assigns other.first to `first` and other.second to `second`.  This overload participates in overload resolution only if [std::is_assignable](../../types/is_assignable.html)<T1&, const U1&>::value and [std::is_assignable](../../types/is_assignable.html)<T2&, const U2&>::value are both true.  | (since C++11)  
---|---  
  
4) Assigns other.first to `first` and other.second to `second`.

This overload participates in overload resolution only if [std::is_assignable_v](../../types/is_assignable.html)<const T1&, const U1&> and [std::is_assignable_v](../../types/is_assignable.html)<const T2&, const U2&> are both true.

5) Move assignment operator. Replaces the contents with those of other using move semantics.

This overload participates in overload resolution only if [std::is_move_assignable](../../types/is_move_assignable.html)<T1>::value and [std::is_move_assignable](../../types/is_move_assignable.html)<T2>::value are both true.

6) Move assignment operator for const-qualified operand.

This overload participates in overload resolution only if [std::is_assignable_v](../../types/is_assignable.html)<const T1&, T1> and [std::is_assignable_v](../../types/is_assignable.html)<const T2&, T2> are both true.

7) Assigns [std::forward](../forward.html)<U1>(p.first) to `first` and [std::forward](../forward.html)<U2>(p.second) to `second`.

This overload participates in overload resolution only if [std::is_assignable](../../types/is_assignable.html)<T1&, U1>::value and [std::is_assignable](../../types/is_assignable.html)<T2&, U2>::value are both true.

8) Assigns [std::forward](../forward.html)<U1>(p.first) to `first` and [std::forward](../forward.html)<U2>(p.second) to `second`.

This overload participates in overload resolution only if [std::is_assignable_v](../../types/is_assignable.html)<const T1&, U1> and [std::is_assignable_v](../../types/is_assignable.html)<const T2&, U2> are both true.

9) Assigns std::get<0>([std::forward](../forward.html)<P>(u)) to `first` and std::get<1>([std::forward](../forward.html)<P>(u)) to `second`.

This overload participates in overload resolution only if 

  * [std::same_as](../../concepts/same_as.html)<[std::remove_cvref_t](../../types/remove_cvref.html)<P>, [std::pair](../pair.html)> is false, 
  * [std::remove_cvref_t](../../types/remove_cvref.html)<P> is not a specialization of [std::ranges::subrange](../../ranges/subrange.html "cpp/ranges/subrange"), 
  * [std::is_assignable_v](../../types/is_assignable.html)<T1&, decltype(std::get<0>([std::forward](../forward.html)<P>(p)))> is true, and 
  * [std::is_assignable_v](../../types/is_assignable.html)<T1&, decltype(std::get<1>([std::forward](../forward.html)<P>(p)))> is true.



10) Assigns std::get<0>([std::forward](../forward.html)<P>(u)) to `first` and std::get<1>([std::forward](../forward.html)<P>(u)) to `second`.

This overload participates in overload resolution only if 

  * [std::same_as](../../concepts/same_as.html)<[std::remove_cvref_t](../../types/remove_cvref.html)<P>, [std::pair](../pair.html)> is false, 
  * [std::remove_cvref_t](../../types/remove_cvref.html)<P> is not a specialization of [std::ranges::subrange](../../ranges/subrange.html "cpp/ranges/subrange"), 
  * [std::is_assignable_v](../../types/is_assignable.html)<const T1&, decltype(std::get<0>([std::forward](../forward.html)<P>(p)))> is true, and 
  * [std::is_assignable_v](../../types/is_assignable.html)<const T1&, decltype(std::get<1>([std::forward](../forward.html)<P>(p)))> is true.

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/utility/pair/operator%3D&action=edit&section=4 "Edit section: Example")] Example

Run this code
    
    
    #include <cstddef>
    #include <iomanip>
    #include <iostream>
    #include <utility>
    #include <vector>
     
    template<class Os, class T>
    Os& operator<<(Os& os, const [std::vector](../../container/vector.html)<T>& v)
    {
        os << '{';
        for ([std::size_t](../../types/size_t.html) t = 0; t != v.size(); ++t)
            os << v[t] << (t + 1 < v.size() ? ", " : "");
        return os << '}';
    }
     
    template<class Os, class U1, class U2>
    Os& operator<<(Os& os, const [std::pair](../pair.html)<U1, U2>& pair)
    {
        return os << '{' << pair.first << ", " << pair.second << '}';
    }
     
    int main()
    {
        [std::pair](../pair.html)<int, [std::vector](../../container/vector.html)<int>> p{1, {2}}, q{2, {5, 6}};
     
        p = q; // (1) operator=(const pair& other);
        [std::cout](../../io/cout.html) << [std::setw](../../io/manip/setw.html)(23) << [std::left](../../io/manip/left.html)
                  << "(1) p = q;"
                  << "p: " << p << "     q: " << q << '\n';
     
        [std::pair](../pair.html)<short, [std::vector](../../container/vector.html)<int>> r{4, {7, 8, 9}};
        p = r; // (3) operator=(const pair<U1, U2>& other);
        [std::cout](../../io/cout.html) << [std::setw](../../io/manip/setw.html)(23)
                  << "(3) p = r;"
                  << "p: " << p << "  r: " << r << '\n';
     
        p = [std::pair](../pair.html)<int, [std::vector](../../container/vector.html)<int>>{3, {4}};
        p = std::move(q); // (5) operator=(pair&& other);
        [std::cout](../../io/cout.html) << [std::setw](../../io/manip/setw.html)(23)
                  << "(5) p = std::move(q);"
                  << "p: " << p << "     q: " << q << '\n';
     
        p = [std::pair](../pair.html)<int, [std::vector](../../container/vector.html)<int>>{5, {6}};
        p = std::move(r); // (7) operator=(pair<U1, U2>&& other);
        [std::cout](../../io/cout.html) << [std::setw](../../io/manip/setw.html)(23)
                  << "(7) p = std::move(r);"
                  << "p: " << p << "  r: " << r << '\n';
    }

Output: 
    
    
    (1) p = q;             p: {2, {5, 6}}     q: {2, {5, 6}}
    (3) p = r;             p: {4, {7, 8, 9}}  r: {4, {7, 8, 9}}
    (5) p = std::move(q);  p: {2, {5, 6}}     q: {2, {}}
    (7) p = std::move(r);  p: {4, {7, 8, 9}}  r: {4, {}}
