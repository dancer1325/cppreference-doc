### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/utility/basic_stacktrace/operator_ltlt&action=edit&section=3 "Edit section: Example")] Example

Run this code
    
    
    #include <stacktrace>
    #include <iostream>
     
    int main()
    {
        [std::cout](../../io/cout.html) << "The stacktrace obtained in the main function:\n";
        [std::cout](../../io/cout.html) << std::stacktrace::current() << '\n';
        []{
            [std::cout](../../io/cout.html) << "The stacktrace obtained in a nested lambda:\n";
            [std::cout](../../io/cout.html) << std::stacktrace::current() << '\n';
        }();
    }

Possible output: 
    
    
    The stacktrace obtained in the main function:
     0# 0x0000000000402E7B in ./prog.exe
     1# __libc_start_main in /lib/x86_64-linux-gnu/libc.so.6
     2# 0x0000000000402CD9 in ./prog.exe
     
    The stacktrace obtained in a nested lambda:
     0# 0x0000000000402DDA in ./prog.exe
     1# 0x0000000000402EB2 in ./prog.exe
     2# __libc_start_main in /lib/x86_64-linux-gnu/libc.so.6
     3# 0x0000000000402CD9 in ./prog.exe
