double expint( IntegralType arg ); |  (2)  |   
| |   
  
1) Computes the [exponential integral](https://en.wikipedia.org/wiki/Exponential_integral "enwiki:Exponential integral") of arg.

2) A set of overloads or a function template accepting an argument of any [integral type](../../types/is_integral.html "cpp/types/is integral"). Equivalent to (1) after casting the argument to double.

As all special functions, `expint` is only guaranteed to be available in `<cmath>` if `__STDCPP_MATH_SPEC_FUNCS__` is defined by the implementation to a value at least 201003L and if the user defines `__STDCPP_WANT_MATH_SPEC_FUNCS__` before including any standard library headers.

### Error handling

Errors may be reported as specified in [math_errhandling](../../numeric/math/math_errhandling.html "cpp/numeric/math/math errhandling"). 

  * If the argument is NaN, NaN is returned and domain error is not reported. 
  * If the argument is ±0, -∞ is returned. 



### Notes

Implementations that do not support TR 29124 but support TR 19768, provide this function in the header `tr1/cmath` and namespace `std::tr1`. 

An implementation of this function is also [available in boost.math](https://www.boost.org/doc/libs/release/libs/math/doc/html/math_toolkit/expint/expint_i.html). 

### Example

(works as shown with gcc 6.0)

Run this code
    
    
    #define __STDCPP_WANT_MATH_SPEC_FUNCS__ 1
    #include <cmath>
    #include <iostream>
     
    int main()
    {
        [std::cout](../../io/cout.html) << "Ei(0) = " << [std::expint](../../numeric/special_functions/expint.html)(0) << '\n'
                  << "Ei(1) = " << [std::expint](../../numeric/special_functions/expint.html)(1) << '\n'
                  << "Gompetz constant = " << -[std::exp](../../numeric/math/exp.html)(1) * [std::expint](../../numeric/special_functions/expint.html)(-1) << '\n';
    }

Output: 
    
    
    Ei(0) = -inf
    Ei(1) = 1.89512
    Gompetz constant = 0.596347
