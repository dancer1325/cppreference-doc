1) Computes the (physicist's) [Hermite polynomials](https://en.wikipedia.org/wiki/Hermite_polynomials "enwiki:Hermite polynomials") of the degree n and argument x.

2) A set of overloads or a function template accepting an argument of any [integral type](../../types/is_integral.html "cpp/types/is integral"). Equivalent to (1) after casting the argument to double.

As all special functions, `hermite` is only guaranteed to be available in `<cmath>` if `__STDCPP_MATH_SPEC_FUNCS__` is defined by the implementation to a value at least 201003L and if the user defines `__STDCPP_WANT_MATH_SPEC_FUNCS__` before including any standard library headers.

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/experimental/special_functions/hermite&action=edit&section=3 "Edit section: Error handling")] Error handling

Errors may be reported as specified in [math_errhandling](../../numeric/math/math_errhandling.html "cpp/numeric/math/math errhandling"). 

  * If the argument is NaN, NaN is returned and domain error is not reported. 
  * If n is greater or equal than 128, the behavior is implementation-defined. 



### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/experimental/special_functions/hermite&action=edit&section=4 "Edit section: Notes")] Notes

Implementations that do not support TR 29124 but support TR 19768, provide this function in the header `tr1/cmath` and namespace `std::tr1`. 

An implementation of this function is also [available in boost.math](https://www.boost.org/doc/libs/release/libs/math/doc/html/math_toolkit/sf_poly/hermite.html). 

The Hermite polynomials are the polynomial solutions of the equation u,,  
\- 2xu,  
= -2nu. 

The first few are: 

  * hermite(0, x) = 1. 
  * hermite(1, x) = 2x. 
  * hermite(2, x) = 4x2  
\- 2. 
  * hermite(3, x) = 8x3  
\- 12x. 
  * hermite(4, x) = 16x4  
\- 48x2  
\+ 12. 



### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/experimental/special_functions/hermite&action=edit&section=5 "Edit section: Example")] Example

(works as shown with gcc 6.0)

Run this code
    
    
    #define __STDCPP_WANT_MATH_SPEC_FUNCS__ 1
    #include <cmath>
    #include <iostream>
     
    double H3(double x)
    {
        return 8 * [std::pow](../../numeric/math/pow.html)(x, 3) - 12 * x;
    }
     
    double H4(double x)
    {
        return 16 * [std::pow](../../numeric/math/pow.html)(x, 4) - 48 * x * x + 12;
    }
     
    int main()
    {
        // spot-checks
        [std::cout](../../io/cout.html) << [std::hermite](../../numeric/special_functions/hermite.html)(3, 10) << '=' << H3(10) << '\n'
                  << [std::hermite](../../numeric/special_functions/hermite.html)(4, 10) << '=' << H4(10) << '\n';
    }

Output: 
    
    
    7880=7880
    155212=155212
