double assoc_legendre( unsigned int n, unsigned int m, IntegralType x ); |  (2)  |   
| |   
  
1) Computes the [associated Legendre polynomials](https://en.wikipedia.org/wiki/Associated_Legendre_polynomials "enwiki:Associated Legendre polynomials") of the degree n, order m, and argument x.

2) A set of overloads or a function template accepting an argument of any [integral type](../../types/is_integral.html "cpp/types/is integral"). Equivalent to (1) after casting the argument to double.

As all special functions, `assoc_legendre` is only guaranteed to be available in `<cmath>` if `__STDCPP_MATH_SPEC_FUNCS__` is defined by the implementation to a value at least 201003L and if the user defines `__STDCPP_WANT_MATH_SPEC_FUNCS__` before including any standard library headers.

### Error handling

Errors may be reported as specified in [math_errhandling](../../numeric/math/math_errhandling.html "cpp/numeric/math/math errhandling"). 

  * If the argument is NaN, NaN is returned and domain error is not reported. 
  * If |x| > 1, a domain error may occur. 
  * If `n` is greater or equal to 128, the behavior is implementation-defined. 



### Notes

Implementations that do not support TR 29124 but support TR 19768, provide this function in the header `tr1/cmath` and namespace `std::tr1`. 

An implementation of this function is also [available in boost.math](https://www.boost.org/doc/libs/release/libs/math/doc/html/math_toolkit/sf_poly/legendre.html).

### Example

(works as shown with gcc 6.0)

Run this code
    
    
    #define __STDCPP_WANT_MATH_SPEC_FUNCS__ 1
    #include <cmath>
    #include <iostream>
     
    double P20(double x)
    {
        return 0.5 * (3 * x * x - 1);
    }
     
    double P21(double x)
    {
        return -3.0 * x * [std::sqrt](../../numeric/math/sqrt.html)(1 - x * x);
    }
     
    double P22(double x)
    {
        return 3 * (1 - x * x);
    }
     
    int main()
    {
        // spot-checks
        [std::cout](../../io/cout.html) << [std::assoc_legendre](../../numeric/special_functions/assoc_legendre.html)(2, 0, 0.5) << '=' << P20(0.5) << '\n'
                  << [std::assoc_legendre](../../numeric/special_functions/assoc_legendre.html)(2, 1, 0.5) << '=' << P21(0.5) << '\n'
                  << [std::assoc_legendre](../../numeric/special_functions/assoc_legendre.html)(2, 2, 0.5) << '=' << P22(0.5) << '\n';
    }

Output: 
    
    
    -0.125=-0.125
    -1.29904=-1.29904
    2.25=2.25
