template< class T, class Abi >  
int popcount( const simd_mask<T, Abi>& k ) noexcept; |  |  (parallelism TS v2)  
| |   
  
Returns the number of true values in the mask k. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/experimental/simd/popcount&action=edit&section=1 "Edit section: Parameters")] Parameters

k  |  \-  |  the [`simd_mask`](simd_mask.html "cpp/experimental/simd/simd mask") to apply the reduction to   
---|---|---  
  
### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/experimental/simd/popcount&action=edit&section=2 "Edit section: Return value")] Return value

An int in the range `[`​0​`, `simd_size_v<T, Abi>`)`. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/experimental/simd/popcount&action=edit&section=3 "Edit section: Example")] Example

Run this code
    
    
    #include <cstddef>
    #include <experimental/simd>
    #include <iostream>
    namespace stdx = std::experimental;
     
    template<typename Abi>
    int count_zeros(stdx::simd<int, Abi> v)
    {
        return stdx::popcount(v == 0); // v == 0 returns a simd_mask
    }
     
    void println(auto rem, auto const v)
    {
        [std::cout](../../io/cout.html) << rem << ": ";
        for ([std::size_t](../../types/size_t.html) i = 0; i != v.size(); ++i)
            [std::cout](../../io/cout.html) << v[i] << ' ';
    }
     
    int main()
    {
        stdx::simd<int> x{8};
        println("x", x);
        [std::cout](../../io/cout.html) << "  zeros: " << count_zeros(x) << '\n';
     
        x[3] = x[1] = false;
        println("x", x);
        [std::cout](../../io/cout.html) << "  zeros: " << count_zeros(x) << '\n';
    }

Possible output: 
    
    
    x: 8 8 8 8   zeros: 0
    x: 8 0 8 0   zeros: 2
