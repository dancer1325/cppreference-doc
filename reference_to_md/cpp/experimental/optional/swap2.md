template< class T >  
void swap( optional<T>& lhs, optional<T>& rhs ) noexcept(/* see below */); |  |  (library fundamentals TS)  
| |   
  
Overloads the [std::swap](../../utility/swap.html "cpp/algorithm/swap") algorithm for [std::experimental::optional](../optional.html "cpp/experimental/optional"). Exchanges the state of lhs with that of rhs. Effectively calls lhs.swap(rhs).
