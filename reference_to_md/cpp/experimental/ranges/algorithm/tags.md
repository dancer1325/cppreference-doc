These classes are [`TagSpecifier`s](../utility/TagSpecifier.html "cpp/experimental/ranges/utility/TagSpecifier") for use with [`ranges::tagged`](../utility/tagged.html "cpp/experimental/ranges/utility/tagged"). 

The name of a specifier is also the element name it corresponds to. For instance, [ranges::tagged](../utility/tagged.html)<[std::pair](../../../utility/pair.html)<int, int>, tag::min, tag::max> provides a set of named accessors named `min` and a set of named accessors named `max`.
