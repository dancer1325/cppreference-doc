template< class F, class... Args >  
[std::result_of_t](../../../types/result_of.html)<F&&(Args&&...)> invoke( F&& f, Args&&... args ); |  |  (ranges TS)  
| |   
  
Invoke the [Callable](../../../named_req/Callable.html "cpp/named req/Callable") object f with the parameters args, and return the result, as if by return INVOKE([std::forward](../../../utility/forward.html)<F>(f), [std::forward](../../../utility/forward.html)<Args>(args)...);, where _INVOKE(f, t1, t2, ..., tN)_ is defined as follows: 

  * if f is a [pointer to member function](../../../language/pointer.html#Pointers_to_member_functions "cpp/language/pointer") of class `T`: 



    

  * If [std::is_base_of](../../../types/is_base_of.html)<T, [std::decay_t](../../../types/decay.html)<decltype(t1)>>::value is true, then INVOKE(f, t1, t2, ..., tN) is equivalent to (t1.*f)(t2, ..., tN), 
  * otherwise, INVOKE(f, t1, t2, ..., tN) is equivalent to ((*t1).*f)(t2, ..., tN). 



  * otherwise, if N == 1 and f is a [pointer to data member](../../../language/pointer.html#Pointers_to_data_members "cpp/language/pointer") of class `T`: 



    

  * If [std::is_base_of](../../../types/is_base_of.html)<T, [std::decay_t](../../../types/decay.html)<decltype(t1)>>::value is true, then INVOKE(f, t1) is equivalent to t1.*f, 
  * otherwise, then INVOKE(f, t1) is equivalent to (*t1).*f. 



  * otherwise, INVOKE(f, t1, t2, ..., tN) is equivalent to f(t1, t2, ..., tN) (that is, f is a [FunctionObject](../../../named_req/FunctionObject.html "cpp/named req/FunctionObject")). 



### Parameters

f  |  \-  |  [Callable](../../../named_req/Callable.html "cpp/named req/Callable") object to be invoked   
---|---|---  
args  |  \-  |  arguments to pass to f
