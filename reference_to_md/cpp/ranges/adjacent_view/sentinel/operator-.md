template< bool OtherConst >  
requires [std::sized_sentinel_for](../../../iterator/sized_sentinel_for.html)<[ranges::sentinel_t](../../iterator_t.html)<Base>,  
[ranges::iterator_t](../../iterator_t.html)</*maybe-const*/<OtherConst, V>>>  
friend constexpr [ranges::range_difference_t](../../range_size_t.html)</*maybe-const*/<OtherConst, V>>  
operator-( const /*sentinel*/& y, const /*iterator*/<OtherConst>& x ); |  (2)  |  (since C++23)  
| |   
  
Computes the distance between the underlying iterator of x and the underlying sentinel of y. 

Let [`_current__`](../iterator.html#Data_members "cpp/ranges/adjacent view/iterator") denote the underlying array of iterators in x, and [`_end__`](../sentinel.html#Data_members "cpp/ranges/adjacent view/sentinel") denote the underlying sentinel in y. 

1) Equivalent to: return x.current_.back() - y.end_;

2) Equivalent to: return y.end_ - x.current_.back();

These function templates are not visible to ordinary [unqualified](../../../language/unqualified_lookup.html "cpp/language/unqualified lookup") or [qualified lookup](../../../language/qualified_lookup.html "cpp/language/qualified lookup"), and can only be found by [argument-dependent lookup](../../../language/adl.html "cpp/language/adl") when `adjacent_view::_sentinel_` is an associated class of the arguments.
