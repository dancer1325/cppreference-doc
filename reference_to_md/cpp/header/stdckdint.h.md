This header was originally in the C standard library ([`<stdckdint.h>`](../../c/header/stdckdint.html "c/header/stdckdint")). 

This header is part of the [numeric](../numeric.html "cpp/numeric") library, in particular, it provides several function templates for [checked integer arithmetic](../numeric.html#Checked_integer_arithmetic "cpp/numeric"). 

It is unspecified whether **`< stdckdint.h>`** provides any declarations in namespace std.
