This header was originally in the C standard library as [`<wchar.h>`](../../c/header/wchar.html "c/header/wchar"). 

This header is part of the null-terminated [wide](../string/wide.html "cpp/string/wide") and [multibyte](../string/multibyte.html "cpp/string/multibyte") strings libraries. It also provides some [C-style I/O](../io/c.html "cpp/io/c") functions and conversion from [C-style Date](../chrono/c.html "cpp/chrono/c").
