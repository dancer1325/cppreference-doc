C++20 is a major version after C++17, featuring major features (concepts, modules, coroutines, and ranges) and other language and library features. The standard was published in December 2020.

## [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/20&action=edit&section=1 "Edit section: New language features")] New language features

  * [Feature test macros](experimental/feature_test.html "cpp/feature test")
  * [Three-way comparison](language/operator_comparison.html#Three-way_comparison "cpp/language/operator comparison") operator <=> and [operator==() = default](language/default_comparisons.html "cpp/language/default comparisons")
  * [Designated initializers](language/aggregate_initialization.html#Designated_initializers "cpp/language/aggregate initialization")
  * Init-statements and initializers in [range-`for`](language/range-for.html "cpp/language/range-for")
  * [`char8_t`](keyword/char8_t.html "cpp/keyword/char8 t")
  * New [attributes](language/attributes.html "cpp/language/attributes"): `[[[no_unique_address](language/attributes/no_unique_address.html "cpp/language/attributes/no unique address")]]`, `[[[likely](language/attributes/likely.html "cpp/language/attributes/likely")]]`, `[[[unlikely](language/attributes/likely.html "cpp/language/attributes/likely")]]`
  * [`Pack-expansions`](language/parameter_pack.html "cpp/language/parameter pack") in [lambda init-captures](language/lambda.html#Lambda_capture "cpp/language/lambda")
  * Removed the requirement to use `typename` to disambiguate types in many contexts 
  * [`consteval`](language/consteval.html "cpp/language/consteval"), [`constinit`](language/constinit.html "cpp/language/constinit")
  * Further relaxed constexpr 
  * Signed integers are 2's complement 
  * [Bitwise shift operators](language/operator_arithmetic.html "cpp/language/operator arithmetic") unified behavior 
  * [aggregate initialization](language/aggregate_initialization.html "cpp/language/aggregate initialization") using parentheses 
  * [coroutines](language/coroutines.html "cpp/language/coroutines")
  * [modules](language/modules.html "cpp/language/modules")
  * [Constraints and concepts](language/constraints.html "cpp/language/constraints")
  * [Abbreviated function template](language/function_template.html#Abbreviated_function_template "cpp/language/function template")
  * DR11: [array new](language/new.html#Defect_reports "cpp/language/new") can deduce array size
