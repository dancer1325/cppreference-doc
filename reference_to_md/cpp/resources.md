### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/links&action=edit&section=3 "Edit section: C++ standard documents and drafts")] C++ standard documents and drafts

  * [JTC1/SC22/WG21](https://open-std.org/jtc1/sc22/wg21/) — The C++ Standards Committee, includes draft C++ language standards, proposals, and defect reports. 
    * N1146 ([PDF](https://open-std.org/JTC1/SC22/WG21/docs/wp/pdf/nov97-2/)) — C++98 final working draft. 
    * N1804 ([PDF](https://wg21.link/n1804)) — A working draft in early years (2005-04-27). 
    * N3337 ([PDF](https://wg21.link/n3337), [HTML](https://timsong-cpp.github.io/cppwp/n3337/)) — C++11 first post-publication draft (2012-01-16). 
    * N4140 ([PDF](https://timsong-cpp.github.io/cppwp/n4140/draft.pdf), [HTML](https://timsong-cpp.github.io/cppwp/n4140/)) — C++14 final working draft (2014-10-07). 
    * N4659 ([PDF](https://wg21.link/n4659), [HTML](https://timsong-cpp.github.io/cppwp/n4659/)) — C++17 final working draft (2017-03-21). 
    * N4861 ([PDF](https://wg21.link/n4861), [HTML](https://timsong-cpp.github.io/cppwp/n4861/)) — C++20 final working draft (2020-04-01). 
    * N4950 ([PDF](https://wg21.link/n4950), [HTML](https://timsong-cpp.github.io/cppwp/n4950/)) — C++23 final working draft (2023-05-10). 
    * N5008 ([PDF](https://wg21.link/n5008)) — C++ latest working draft (2025-03-15). 
  * [cplusplus/draft](https://github.com/cplusplus/draft) — The official GitHub repository of current draft. 
  * [eel.is/c++draft](https://eel.is/c++draft/) — HTML version of current draft. 



### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/links&action=edit&section=4 "Edit section: C++ language and library references")] C++ language and library references

  * [Apache](https://stdcxx.apache.org/doc/stdlibref/index.html) — Reference manual for the Apache implementation of the C++ standard library. 
  * [IBM AIX 16.1](https://www.ibm.com/docs/en/xl-c-and-cpp-aix/16.1) [IBM zOS 3.1.0](https://www.ibm.com/docs/en/zos/3.1.0) — Reference manual for the IBM implementation of C and C++ core languages. 
  * [IBM AIX 16.1](https://www.ibm.com/docs/en/xl-c-and-cpp-aix/16.1?topic=reference-standard-c-library-header-files) — Reference manual for the IBM implementations of the C++ standard library. 
  * [MSDN](https://learn.microsoft.com/en-us/cpp/cpp/cpp-language-reference) — Reference manual for the Microsoft implementation of the C++ core language. 
  * [MSDN](https://learn.microsoft.com/en-us/cpp/standard-library/cpp-standard-library-reference) — Reference manual for the Microsoft implementation of the C++ library. 
    * [Dinkumware](https://web.archive.org/web/20120204111500/http://www.dinkumware.com/manuals/default.aspx) — C++ Standard Library implementation that ships with Microsoft Visual C++. 
    * [Open STL](https://github.com/microsoft/STL) — MSVC's implementation of modern C++ Standard Library. 
  * [GNU libstdc++](https://gcc.gnu.org/onlinedocs/libstdc++/index.html) — Reference manual for the GNU implementation of the C++ library. 
    * [libstdc++-v3](https://github.com/gcc-mirror/gcc/tree/master/libstdc%2B%2B-v3) — GNU implementation of the C++ library. 
  * [libc++ documentation](https://libcxx.llvm.org/) — clang/llvm project's documentation for the implementation of the C++ Standard Library. 
    * [libc++](https://github.com/llvm/llvm-project/tree/main/libcxx) — clang/llvm implementation of the C++ Standard Library. 
  * [HP](https://support.hpe.com/hpesc/public/docDisplay?docLocale=en_US&docId=emr_na-c02133105) — Reference manual for the Dinkumware implementation of the C++ library (as licensed to HP). 
  * [EWL](https://www.nxp.com.cn/docs/en/reference-manual/CWEWLCPPREF.pdf) — Reference manual for the Embedded Warrior implementation of the C++ library. 
  * [cplusplus.com](https://cplusplus.com/reference) — Unofficial C++ standard library reference. 
  * [SGI STL (hosted on rrsd.com)](https://www.rrsd.com/software_development/stl/stl/index.html) — Reference for STL, the 1994 library that introduced many concepts later adopted by the C++ standard library.
