## [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/23&action=edit&section=1 "Edit section: New language features")] New language features

  * New [language feature testing macros](experimental/feature_test.html#Language_features "cpp/feature test")
  * [Explicit object parameter](language/function.html#Explicit_object_parameter "cpp/language/function"), [explicit object member functions](language/member_functions.html#Explicit_object_member_functions "cpp/language/member functions"), a.k.a. "deducing this" ([P0847R7](https://wg21.link/P0847R7)) 
  * [Multidimensional subscript operator](language/operators.html#Array_subscript_operator "cpp/language/operators") (e.g. v[1, 3, 7] = 42;) ([P2128R6](https://wg21.link/P2128R6)) 
  * static operator[] ([P2589R1](https://wg21.link/P2589R1)), static operator(), and static lambdas ([P1169R4](https://wg21.link/P1169R4)) 
  * auto(x) and auto{x}: [decay-copy](language/explicit_cast.html#Syntax "cpp/language/explicit cast") in the language ([P0849R8](https://wg21.link/P0849R8)) 
  * Assumptions via new attribute `[[[assume](language/attributes/assume.html "cpp/language/attributes/assume")(_expression_)]]` ([P1774R8](https://wg21.link/P1774R8)) 
  * Attributes on [lambda-expressions](language/lambda.html#Syntax "cpp/language/lambda") ([P2173R1](https://wg21.link/P2173R1)) 
  * Optional [extended floating-point types](types/floating-point.html "cpp/types/floating-point"): std::float{16|32|64|128}_t, and [std::bfloat16_t](types/floating-point.html) ([P1467R9](https://wg21.link/P1467R9)) 
  * New preprocessor directives: [`#elifdef`, `#elifndef`](preprocessor/conditional.html "cpp/preprocessor/conditional") ([P2334R1](https://wg21.link/P2334R1)), and [` #warning`](preprocessor/warning.html "cpp/preprocessor/error") ([P2437R1](https://wg21.link/P2437R1)) 
  * [Literal suffix '`Z`'/'`z`'](language/integer_literal.html#The_type_of_the_literal "cpp/language/integer literal") for (signed) [std::size_t](types/size_t.html) literals ([P0330R8](https://wg21.link/P0330R8)), e.g. auto ouz = 0uz;
  * White-spaces trimming before line splicing. ([P2223R2](https://wg21.link/P2223R2)) 
  * Simpler implicit move ([P2266R3](https://wg21.link/P2266R3)) 
  * Extending the lifetime of temporaries in range-based for loop initializer ([P2718R0](https://wg21.link/P2718R0)) 
  * CTAD from inherited constructors ([P2582R1](https://wg21.link/P2582R1)) 
  * Labels at the end of compound statements ([P2324R2](https://wg21.link/P2324R2)), e.g.  




     void f(int& x)  
{  
if (x)  
goto END;  
x = 42;  
END:  
}

  * Alias declarations (using) in init-statements ([P2360R0](https://wg21.link/P2360R0)), e.g.  




     for (using T = int; T e : v)  
/* ... */

  * Make `**()**` more optional for lambda expressions ([P1102R2](https://wg21.link/P1102R2)) 
  * Narrowing contextual conversions to bool in static_assert and if constexpr ([P1401R5](https://wg21.link/P1401R5)) 
  * Make declaration order layout (of non-static class data members) mandated ([P1847R4](https://wg21.link/P1847R4)) 
  * Text encoding changes: 
    * [Character sets and encodings](language/charset.html "cpp/language/charset") ([P2314R4](https://wg21.link/P2314R4)) 
    * Consistent character literal encoding ([P2316R2](https://wg21.link/P2316R2)) 
    * [Named universal character escapes](language/escape.html#Named_universal_character_escapes "cpp/language/escape"), e.g. "\N{CAT FACE}" for "üê±" ([P2071R2](https://wg21.link/P2071R2)) 
    * [Delimited escape sequences](language/escape.html "cpp/language/escape"), e.g. "\o{7777}", "\x{C0DE}", "\u{CAFE}" ([P2290R3](https://wg21.link/P2290R3)) 
    * [Support for UTF-8 as a portable source file encoding](language/translation_phases.html#Phase_1 "cpp/language/translation phases") ([P2295R6](https://wg21.link/P2295R6)) 
  * [`if consteval`](language/if.html#Consteval_if "cpp/language/if") / [`if not consteval`](language/if.html#Consteval_if "cpp/language/if") ([P1938R3](https://wg21.link/P1938R3)) 
  * constexpr changes: 
    * Non-literal variables, labels, and gotos in constexpr functions ([P2242R3](https://wg21.link/P2242R3)) 
    * Permitting static and thread_local variables in constant expressions in constexpr functions ([P2647R1](https://wg21.link/P2647R1)) 
    * constexpr function does not need its return type and parameter types to be literal type ([P2448R2](https://wg21.link/P2448R2)) 
    * constexpr function for which no invocation satisfies the requirements of a core constant expression ([P2448R2](https://wg21.link/P2448R2))
