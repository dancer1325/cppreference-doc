[std::pmr::memory_resource](memory_resource.html)* null_memory_resource() noexcept; |  |  (since C++17)  
| |   
  
Returns a pointer to a `memory_resource` that doesn't perform any allocation. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/memory/null_memory_resource&action=edit&section=1 "Edit section: Return value")] Return value 

Returns a pointer `p` to a static storage duration object of a type derived from [std::pmr::memory_resource](memory_resource.html "cpp/memory/memory resource"), with the following properties: 

  * its `allocate()` function always throws [std::bad_alloc](new/bad_alloc.html "cpp/memory/new/bad alloc"); 
  * its `deallocate()` function has no effect; 
  * for any `memory_resource` `r`, p->is_equal(r) returns &r == p. 



The same value is returned every time this function is called. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/memory/null_memory_resource&action=edit&section=2 "Edit section: Example")] Example

The program demos the main usage of `null_memory_resource`: ensure that a memory pool which requires memory allocated on the stack will NOT allocate memory on the heap if it needs more memory.

Run this code
    
    
    #include <array>
    #include <cstddef>
    #include <iostream>
    #include <memory_resource>
    #include <string>
    #include <unordered_map>
     
    int main()
    {
        // allocate memory on the stack
        [std::array](../container/array.html)<[std::byte](../types/byte.html), 20000> buf;
     
        // without fallback memory allocation on heap
        [std::pmr::monotonic_buffer_resource](monotonic_buffer_resource.html) pool{buf.data(), buf.size(),
                                                 std::pmr::null_memory_resource()};
     
        // allocate too much memory
        [std::pmr::unordered_map](../container/unordered_map.html)<long, [std::pmr::string](../string/basic_string.html)> coll{&pool};
        try
        {
            for ([std::size_t](../types/size_t.html) i = 0; i < buf.size(); ++i)
            {
                coll.emplace(i, "just a string with number " + [std::to_string](../string/basic_string/to_string.html)(i));
     
                if (i && i % 50 == 0)
                    [std::clog](../io/clog.html) << "size: " << i << "...\n";
            }
        }
        catch (const [std::bad_alloc](new/bad_alloc.html)& e)
        {
            [std::cerr](../io/cerr.html) << e.what() << '\n';
        }
     
        [std::cout](../io/cout.html) << "size: " << coll.size() << '\n';
    }

Possible output: 
    
    
    size: 50...
    size: 100...
    size: 150...
    std::bad_alloc
    size: 183
