### Requirements

An UnformattedInputFunction is a stream input function that performs the following: 

1) Constructs an object of type [`basic_istream::sentry`](../io/basic_istream/sentry.html "cpp/io/basic istream/sentry") with automatic storage duration and with the `noskipws` argument set to true, which performs the following: 

  * If [`eofbit`](../io/ios_base/iostate.html "cpp/io/ios base/iostate") or [`badbit`](../io/ios_base/iostate.html "cpp/io/ios base/iostate") are set on the input stream, sets the [`failbit`](../io/ios_base/iostate.html "cpp/io/ios base/iostate") as well, and if exceptions on `failbit` are enabled in this input stream's [exception mask](../io/basic_ios/exceptions.html "cpp/io/basic ios/exceptions") ((exceptions() & failbit) != 0), throws [`ios_base::failure`](../io/ios_base/failure.html "cpp/io/ios base/failure"). 
  * Flushes the tie()'d output stream, if applicable.



2) Checks the status of the sentry by calling `sentry::operator bool()`, which is equivalent to [`basic_ios::good`](../io/basic_ios/good.html "cpp/io/basic ios/good"). 

  * If the operator returns false or sentry's constructor throws an exception: 



    

  * Sets the number of extracted characters (gcount) in the input stream to zero. 
  * If the function was called to write to an array of `CharT`, writes `CharT()` (the null character) to the first location of the array. 



  * If the operator returns true, performs the input as if by calling rdbuf()->sbumpc() or rdbuf()->sgetc(). 



    

  * If the end of the stream is reached (the call to rdbuf()->sbumpc() or rdbuf()->sgetc() returns Traits::eof()), sets `eofbit`. If exceptions on `eofbit` are enabled in this stream's [exception mask](../io/basic_ios/exceptions.html "cpp/io/basic ios/exceptions") ((exceptions() & eofbit) != 0), throws [`ios_base::failure`](../io/ios_base/failure.html "cpp/io/ios base/failure"). 
  * If an exception is thrown during input, sets `badbit` in the input stream. If exceptions on `badbit` are enabled in this stream's [exception mask](../io/basic_ios/exceptions.html "cpp/io/basic ios/exceptions") ((exceptions() & badbit) != 0), the exception is also rethrown. 



    

  * Exceptions thrown from [`basic_ios::clear`](../io/basic_ios/clear.html "cpp/io/basic ios/clear") are not caught or rethrown. 



  * If no exception was thrown during input, sets the number of extracted characters (gcount) in the input stream.



3) In any event, whether terminating by exception or returning, the sentry's destructor is called before leaving this function.

### Standard library

The following standard library functions are **UnformattedInputFunction s**. 

  * [std::getline](../string/basic_string/getline.html "cpp/string/basic string/getline"), except that it does not modify gcount. 
  * [`basic_istream::operator>>(basic_streambuf*)`](../io/basic_istream/operator_gtgt.html "cpp/io/basic istream/operator gtgt")
  * [`basic_istream::get`](../io/basic_istream/get.html "cpp/io/basic istream/get")
  * [`basic_istream::getline`](../io/basic_istream/getline.html "cpp/io/basic istream/getline")
  * [`basic_istream::ignore`](../io/basic_istream/ignore.html "cpp/io/basic istream/ignore")
  * [`basic_istream::peek`](../io/basic_istream/peek.html "cpp/io/basic istream/peek")
  * [`basic_istream::read`](../io/basic_istream/read.html "cpp/io/basic istream/read")
  * [`basic_istream::readsome`](../io/basic_istream/readsome.html "cpp/io/basic istream/readsome")
  * [`basic_istream::putback`](../io/basic_istream/putback.html "cpp/io/basic istream/putback"), except that it first clears `eofbit`
  * [`basic_istream::unget`](../io/basic_istream/unget.html "cpp/io/basic istream/unget"), except that it first clears `eofbit`
  * [`basic_istream::sync`](../io/basic_istream/sync.html "cpp/io/basic istream/sync"), except that it does not modify gcount 
  * [`basic_istream::tellg`](../io/basic_istream/tellg.html "cpp/io/basic istream/tellg"), except that it does not modify gcount 
  * [`basic_istream::seekg`](../io/basic_istream/seekg.html "cpp/io/basic istream/seekg"), except that it first clears `eofbit` and does not modify gcount 
  * [std::ws](../io/manip/ws.html "cpp/io/manip/ws"), except that it does not modify gcount 



### Defect reports

The following behavior-changing defect reports were applied retroactively to previously published C++ standards. 

DR  | Applied to  | Behavior as published  | Correct behavior   
---|---|---|---  
[LWG 61](https://cplusplus.github.io/LWG/issue61) | C++98  | it was unclear whether throwing an exception because of  
setting `eofbit` and/or `failbit` will result in setting `badbit` | exceptions thrown from [`basic_ios::clear`](../io/basic_ios/clear.html "cpp/io/basic ios/clear")  
are not caught or rethrown   
[LWG 160](https://cplusplus.github.io/LWG/issue160) | C++98  | the process of determining whether the exception caught  
is rethrown mentioned a non-existing function `exception()` | corrected to [`exceptions()`](../io/basic_ios/exceptions.html "cpp/io/basic ios/exceptions")  
[LWG 243](https://cplusplus.github.io/LWG/issue243) | C++98  | the behavior when `sentry::operator bool()` returns false  
or the sentry object fails to be constructed was not specified  | specified
