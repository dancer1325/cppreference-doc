### Requirements

A UnformattedOutputFunction is a stream output function that performs the following: 

1) Constructs an object of type [`basic_ostream::sentry`](../io/basic_ostream/sentry.html "cpp/io/basic ostream/sentry") with automatic storage duration, which performs the following: 

  * If [`eofbit`](../io/ios_base/iostate.html "cpp/io/ios base/iostate") or [`badbit`](../io/ios_base/iostate.html "cpp/io/ios base/iostate") are set on the output stream, sets the [`failbit`](../io/ios_base/iostate.html "cpp/io/ios base/iostate") as well, and if exceptions on `failbit` are enabled in this output stream's [exception mask](../io/basic_ios/exceptions.html "cpp/io/basic ios/exceptions") ((exceptions() & failbit) != 0), throws [`ios_base::failure`](../io/ios_base/failure.html "cpp/io/ios base/failure"). 
  * Flushes the tie()'d output stream, if applicable.



2) Checks the status of the sentry by calling `sentry::operator bool()`, which is equivalent to [`basic_ios::good`](../io/basic_ios/good.html "cpp/io/basic ios/good"). 

  * If the operator returns false or sentry's constructor throws an exception, no output takes place. 
  * If the operator returns true, attempts to perform the desired output by inserting the characters into the output stream as if by calling rdbuf()->sputc(). Other public members of [std::basic_ostream](../io/basic_ostream.html "cpp/io/basic ostream") may also be used, but virtual members of rdbuf() except [`overflow()`](../io/basic_streambuf/overflow.html "cpp/io/basic streambuf/overflow"), [`xsputn()`](../io/basic_streambuf/sputn.html "cpp/io/basic streambuf/sputn") and [`sync()`](../io/basic_streambuf/pubsync.html "cpp/io/basic streambuf/pubsync") will never be called. 



    

  * If an exception is thrown during output, sets `badbit` in the output stream. If exceptions on `badbit` are enabled in this stream's [exception mask](../io/basic_ios/exceptions.html "cpp/io/basic ios/exceptions") ((exceptions() & badbit) != 0), the exception is also rethrown. 
  * If no exception was thrown, returns the value specified by the function.



3) In any event, whether terminating by exception or returning, the sentry's destructor is called before leaving this function.

### Standard library

The following standard library functions are **UnformattedOutputFunction s**. 

  * [`basic_ostream::operator<<(basic_streambuf*)`](../io/basic_ostream/operator_ltlt.html "cpp/io/basic ostream/operator ltlt")
  * [`basic_ostream::put`](../io/basic_ostream/put.html "cpp/io/basic ostream/put")
  * [`basic_ostream::write`](../io/basic_ostream/write.html "cpp/io/basic ostream/write")
  * [`basic_ostream::flush`](../io/basic_ostream/flush.html "cpp/io/basic ostream/flush")



  * [`basic_ostream::tellp`](../io/basic_ostream/tellp.html "cpp/io/basic ostream/tellp") (except calls pubseekoff instead of output) 
  * [`basic_ostream::seekp`](../io/basic_ostream/seekp.html "cpp/io/basic ostream/seekp") (except calls pubseekoff or pubseekpos instead of output)

### Defect reports

The following behavior-changing defect reports were applied retroactively to previously published C++ standards. 

DR  | Applied to  | Behavior as published  | Correct behavior   
---|---|---|---  
[LWG 63](https://cplusplus.github.io/LWG/issue63) | C++98  | the exception-handling policy was missing  | added   
[LWG 160](https://cplusplus.github.io/LWG/issue160) | C++98  | the process of determining whether the exception caught  
is rethrown mentioned a non-existing function `exception()` | corrected to [`exceptions()`](../io/basic_ios/exceptions.html "cpp/io/basic ios/exceptions")  
[LWG 165](https://cplusplus.github.io/LWG/issue165) | C++98  | the only virtual member allowed to be  
called on rdbuf() was `overflow()` | also allowed  
`xsputn()` and `sync()`
