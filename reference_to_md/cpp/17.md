C++17 is a major version that follows the minor version C++14 and introduces new language and library features. The standard was published in December 2017. 

The following features were merged into C++17: 

  * From the [File System TS](experimental/fs.html "cpp/experimental/fs"): the [filesystem library](filesystem.html "cpp/filesystem"). 
  * From the [Library fundamentals v1 TS](experimental/memory.html "cpp/experimental/lib extensions"): features, including [std::any](utility/any.html "cpp/utility/any"), [std::optional](utility/optional.html "cpp/utility/optional"), [std::string_view](string/basic_string_view.html "cpp/string/basic string view"), [std::apply](utility/apply.html "cpp/utility/apply"), [polymorphic allocators](memory.html#Allocators "cpp/memory"), [searchers](functional.html#Searchers "cpp/utility/functional"). 
  * From [Library fundamentals v2 TS](experimental/lib_extensions_2.html "cpp/experimental/lib extensions 2"): [std::void_t](types/void_t.html "cpp/types/void t"), [std::conjunction](types/conjunction.html "cpp/types/conjunction"), [std::disjunction](types/disjunction.html "cpp/types/disjunction"), [std::negation](types/negation.html "cpp/types/negation"), [std::not_fn](utility/functional/not_fn.html "cpp/utility/functional/not fn"), [std::gcd](numeric/gcd.html "cpp/numeric/gcd"), [std::lcm](numeric/lcm.html "cpp/numeric/lcm"). 
  * From [Parallelism v1 TS](experimental/parallelism.html "cpp/experimental/parallelism"): features, including [execution policies](algorithm/execution_policy_tag_t.html "cpp/algorithm/execution policy tag t"), [std::reduce](algorithm/reduce.html "cpp/algorithm/reduce"), [std::inclusive_scan](algorithm/inclusive_scan.html "cpp/algorithm/inclusive scan"), [std::exclusive_scan](algorithm/exclusive_scan.html "cpp/algorithm/exclusive scan"), but removing `exception_list`. 
  * From [Mathematical special functions IS](experimental/special_math.html "cpp/experimental/special functions"): [mathematical special functions](numeric/special_math.html "cpp/numeric/special functions"). 
  * From C11: [std::aligned_alloc](memory/c/aligned_alloc.html "cpp/memory/c/aligned alloc"), [std::timespec_get](chrono/c/timespec_get.html "cpp/chrono/c/timespec get").

## New language features

  * [`u8` character literal](language/character_literal.html "cpp/language/character literal")

  * new [order of evaluation](language/eval_order.html "cpp/language/eval order") rules 
  * [lambda capture of *this](language/lambda.html#Lambda_capture "cpp/language/lambda")


  * **[`constexpr`](keyword/constexpr.html "cpp/keyword/constexpr")**

     compile-time [`if constexpr`](language/if.html "cpp/language/if")
     [constexpr lambda](language/lambda.html "cpp/language/lambda")


  * **Variables**

     [inline variables](language/inline.html "cpp/language/inline")
     [structured bindings](language/structured_binding.html "cpp/language/structured binding")
     initializers for [`if`](language/if.html "cpp/language/if") and [`switch`](language/switch.html "cpp/language/switch")
     guaranteed [copy elision](language/copy_elision.html "cpp/language/copy elision")
     [temporary materialization](language/implicit_cast.html#Temporary_materialization "cpp/language/implicit conversion")


  * **Templates**

     [fold-expressions](language/fold.html "cpp/language/fold") ( ... )

     constant [template parameters](language/template_parameters.html "cpp/language/template parameters") declared with `auto`


  * **[Namespaces](language/namespace.html "cpp/language/namespace")**

     simplified nested namespaces 
     `using`-declaration declaring multiple names 
     [attribute namespaces](language/attributes.html "cpp/language/attributes") don't have to repeat 
  * **new[attributes](language/attributes.html "cpp/language/attributes")**: 
    * `[[[fallthrough](language/attributes/fallthrough.html "cpp/language/attributes/fallthrough")]]`
    * `[[[maybe_unused](language/attributes/maybe_unused.html "cpp/language/attributes/maybe unused")]]`
