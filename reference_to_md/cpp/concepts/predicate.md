template< class F, class... Args >  
concept predicate =  
[std::regular_invocable](invocable.html)<F, Args...> &&  
[boolean-testable](boolean-testable.html)<[std::invoke_result_t](../types/result_of.html)<F, Args...>>; |  |  (since C++20)  
| |   
  
The concept std::predicate<F, Args...> specifies that `F` is a predicate that accepts arguments whose types and value categories are encoded by `Args...`, i.e., it can be invoked with these arguments to produce a [`_boolean-testable_`](boolean-testable.html "cpp/concepts/boolean-testable") result. 

Note that [`regular_invocable`](invocable.html "cpp/concepts/invocable") requires the invocation to not modify either the callable object or the arguments and be [equality-preserving](../concepts.html#Equality_preservation "cpp/concepts"). 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=Template:cpp/concepts/equality_preservation&action=edit&section=T-1 "Template:cpp/concepts/equality preservation")] Equality preservation

Expressions declared in [requires expressions](../language/requires.html "cpp/language/requires") of the standard library concepts are required to be [equality-preserving](../concepts.html#Equality_preservation "cpp/concepts") (except where stated otherwise).
