static T quiet_NaN() throw(); |  | (until C++11)  
---|---|---  
static constexpr T quiet_NaN() noexcept; |  |  (since C++11)  
| |   
  
Returns the special value "quiet [not-a-number](https://en.wikipedia.org/wiki/NaN "enwiki:NaN")", as represented by the floating-point type `T`. Only meaningful if [std::numeric_limits](../numeric_limits.html)<T>::has_quiet_NaN == true. In IEEE 754, the most common binary representation of floating-point numbers, any value with all bits of the exponent set and at least one bit of the fraction set represents a NaN. It is implementation-defined which values of the fraction represent quiet or signaling NaNs, and whether the sign bit is meaningful.

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/types/numeric_limits/quiet_NaN&action=edit&section=2 "Edit section: Notes")] Notes

A NaN never compares equal to itself. Copying a NaN may not preserve its bit representation. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/types/numeric_limits/quiet_NaN&action=edit&section=3 "Edit section: Example")] Example

Several ways to generate a NaN (the output string is compiler-specific):

Run this code
    
    
    #include <iostream>
    #include <limits>
    #include <cmath>
     
    int main()
    {
        [std::cout](../../io/cout.html) << [std::numeric_limits](../numeric_limits.html)<double>::quiet_NaN()     << ' ' // nan
                  << [std::numeric_limits](../numeric_limits.html)<double>::signaling_NaN() << ' ' // nan
                  << [std::acos](../../numeric/math/acos.html)(2)    << ' '   // nan
                  << [std::tgamma](../../numeric/math/tgamma.html)(-1) << ' '   // nan
                  << [std::log](../../numeric/math/log.html)(-1)    << ' '   // nan
                  << [std::sqrt](../../numeric/math/sqrt.html)(-1)   << ' '   // -nan
                  << 0 / 0.0         << '\n'; // -nan
     
        [std::cout](../../io/cout.html) << "NaN == NaN? " << [std::boolalpha](../../io/manip/boolalpha.html)
                  << ( [std::numeric_limits](../numeric_limits.html)<double>::quiet_NaN() ==
                       [std::numeric_limits](../numeric_limits.html)<double>::quiet_NaN() ) << '\n';
    }

Possible output: 
    
    
    nan nan nan nan nan -nan -nan
    NaN == NaN? false
