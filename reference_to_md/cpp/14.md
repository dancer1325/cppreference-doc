C++14 is a minor version after the major version C++11, featuring mainly minor improvements and defect fixes. Its approval was announced on August 18, 2014. It was released on December 15, 2014. 

Before its approval, C++1y is sometimes used to indicate its release in 2010s.

## [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/14&action=edit&section=1 "Edit section: New language features")] New language features

  * [variable templates](language/variable_template.html "cpp/language/variable template")
  * [generic lambdas](language/lambda.html "cpp/language/lambda")
  * lambda init-capture 
  * new/delete elision 
  * [relaxed restrictions on constexpr functions](language/constexpr.html "cpp/language/constexpr")
  * [binary literals](language/integer_literal.html "cpp/language/integer literal")
  * [digit separators](language/integer_literal.html#Single_quote "cpp/language/integer literal")
  * [return type deduction for functions](language/function.html#Return_type_deduction_.28since_C.2B.2B14.29 "cpp/language/function")
  * [aggregate classes](language/aggregate_initialization.html "cpp/language/aggregate initialization") with default non-static member initializers. 



## [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/14&action=edit&section=2 "Edit section: New library features")] New library features

  * [std::make_unique](memory/unique_ptr/make_unique.html "cpp/memory/unique ptr/make unique")
  * [std::shared_timed_mutex](thread/shared_timed_mutex.html "cpp/thread/shared timed mutex") and [std::shared_lock](thread/shared_lock.html "cpp/thread/shared lock")
  * [std::integer_sequence](utility/integer_sequence.html "cpp/utility/integer sequence")
  * [std::exchange](utility/exchange.html "cpp/utility/exchange")
  * [std::quoted](io/manip/quoted.html "cpp/io/manip/quoted")
  * and many small improvements to existing library facilities, such as 
    * two-range overloads for some algorithms 
    * type alias versions of type traits 
    * user-defined literals for [`basic_string`](string/basic_string.html "cpp/string/basic string"), [`duration`](chrono/duration.html "cpp/chrono/duration") and [`complex`](numeric/complex.html "cpp/numeric/complex")
    * etc.
