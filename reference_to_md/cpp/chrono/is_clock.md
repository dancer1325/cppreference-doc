template< class T >  
struct is_clock; |  |  (since C++20)  
| |   
  
If `T` satisfies the [Clock](../named_req/Clock.html "cpp/named req/Clock") requirements, provides the member constant value equal true. For any other type, value is false. 

For the purpose of this trait, the extent to which an implementation determines that a type cannot meet the [Clock](../named_req/Clock.html "cpp/named req/Clock") requirements is unspecified, except that a minimum `T` shall not qualify as a [Clock](../named_req/Clock.html "cpp/named req/Clock") unless it meets all following conditions: 

  * Each of the following [qualified identifiers](../language/name.html#Qualified_identifiers "cpp/language/identifiers") is valid and denotes a type: 



    

  * `T::rep`
  * `T::period`
  * `T::duration`
  * `T::time_point`



  * Each of the following expressions is well-formed when treated as an [unevaluated operand](../language/expressions.html#Potentially-evaluated_expressions "cpp/language/expressions"): 



    

  * `T::is_steady`
  * `T::now()`



If the program adds specializations for `std::is_clock` or `std::is_clock_v`, the behavior is undefined.

### Possible implementation
    
    
    template<class>
    struct is_clock : [std::false_type](../types/integral_constant.html) {};
     
    template<class T>
        requires
            requires
            {
                typename T::rep;
                typename T::period;
                typename T::duration;
                typename T::time_point;
                T::is_steady; // type is not checked
                T::now();     // return type is not checked
            }
    struct is_clock<T> : [std::true_type](../types/integral_constant.html) {};

### Notes

If `T` otherwise meets the [Clock](../named_req/Clock.html "cpp/named req/Clock") requirements, but `T::is_steady` is not of type const bool, or `T::now()` is not of type T::time_point, the result of `is_clock_v<T>` is unspecified. 

### Example

Run this code
    
    
    #include <chrono>
    #include <ratio>
     
    static_assert
    (
        std::chrono::is_clock_v<[std::chrono::utc_clock](utc_clock.html)> and
        not std::chrono::is_clock_v<[std::chrono::duration](duration.html)<int, [std::exa](../numeric/ratio/ratio.html)>>
    );
     
    int main() {}
