Defined in header `[<chrono>](../../header/chrono.html "cpp/header/chrono")` |  |   
---|---|---  
template<  
class CharT,  
class Traits,  
class Rep,  
class Period  
> [std::basic_ostream](../../io/basic_ostream.html)<CharT, Traits>&  
operator<<( [std::basic_ostream](../../io/basic_ostream.html)<CharT, Traits>& os,  
const [std::chrono::duration](../duration.html)<Rep, Period>& d ); |  |  (since C++20)  
| |   
  
Inserts a textual representation of d into os. 

Behaves as if it was implemented as 
    
    
    [std::basic_ostringstream](../../io/basic_ostringstream.html)<CharT, Traits> s;
    s.flags(os.flags());
    s.imbue(os.getloc());
    s.precision(os.precision());
    s << d.count() << units_suffix; // see below
    return os << s.str();

In other words, the stream flags, locale, and precision are determined by the stream, but any padding are determined using the entire output string. 

The `units_suffix` is determined based on `Period::type` according to the following table. 

`Period::type` |  Suffix   
---|---  
std::atto |  `as`  
[std::femto](../../numeric/ratio/ratio.html "cpp/numeric/ratio/ratio") |  `fs`  
[std::pico](../../numeric/ratio/ratio.html "cpp/numeric/ratio/ratio") |  `ps`  
[std::nano](../../numeric/ratio/ratio.html "cpp/numeric/ratio/ratio") |  `ns`  
[std::micro](../../numeric/ratio/ratio.html "cpp/numeric/ratio/ratio") |  `µs` (U+00B5) or `us`, it is implementation-defined which one is used   
[std::milli](../../numeric/ratio/ratio.html "cpp/numeric/ratio/ratio") |  `ms`  
[std::centi](../../numeric/ratio/ratio.html "cpp/numeric/ratio/ratio") |  `cs`  
[std::deci](../../numeric/ratio/ratio.html "cpp/numeric/ratio/ratio") |  `ds`  
[`std::ratio<1>`](../../numeric/ratio/ratio.html "cpp/numeric/ratio/ratio") |  `s`  
[std::deca](../../numeric/ratio/ratio.html "cpp/numeric/ratio/ratio") |  `das`  
[std::hecto](../../numeric/ratio/ratio.html "cpp/numeric/ratio/ratio") |  `hs`  
[std::kilo](../../numeric/ratio/ratio.html "cpp/numeric/ratio/ratio") |  `ks`  
[std::mega](../../numeric/ratio/ratio.html "cpp/numeric/ratio/ratio") |  `Ms`  
[std::giga](../../numeric/ratio/ratio.html "cpp/numeric/ratio/ratio") |  `Gs`  
[std::tera](../../numeric/ratio/ratio.html "cpp/numeric/ratio/ratio") |  `Ts`  
[std::peta](../../numeric/ratio/ratio.html "cpp/numeric/ratio/ratio") |  `Ps`  
[std::exa](../../numeric/ratio/ratio.html "cpp/numeric/ratio/ratio") |  `Es`  
[`std::ratio<60>`](../../numeric/ratio/ratio.html "cpp/numeric/ratio/ratio") |  `min`  
[`std::ratio<3600>`](../../numeric/ratio/ratio.html "cpp/numeric/ratio/ratio") |  `h`  
[`std::ratio<86400>`](../../numeric/ratio/ratio.html "cpp/numeric/ratio/ratio") |  `d`  
None of the above, and Period::type::den == 1 |  `[_num_]s`  
None of the above  |  `[_num_ /_den_]s`  
  
For the last two rows of the table, `_num_` and `_den_` in the suffix are `Period::type::num` and `Period::type::den` formatted as a decimal number with no leading zeroes, respectively. 

### Return value

A reference to the stream, i.e., os. 

### Example

This example shows the output of `std::chrono::operator<<` when given a duration:

Run this code
    
    
    #include <chrono>
    #include <iostream>
    using namespace std::chrono_literals;
     
    int main()
    {
        constexpr auto duration = 123ms;
        [std::cout](../../io/cout.html) << duration << '\n';
    }

Output: 
    
    
    123ms
