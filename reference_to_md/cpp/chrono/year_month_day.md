class year_month_day; |  |  (since C++20)  
| |   
  
The class `year_month_day` represents a specific year, month, and day. It is a field-based time point, with a resolution of std::chrono::days. std::chrono::years\- and std::chrono::months-oriented arithmetic are supported directly. An implicit conversion to and from std::chrono::sys_days allows std::chrono::days-oriented arithmetic to be performed efficiently. 

`year_month_day` is a [TriviallyCopyable](../named_req/TriviallyCopyable.html "cpp/named req/TriviallyCopyable") [StandardLayoutType](../named_req/StandardLayoutType.html "cpp/named req/StandardLayoutType").

### Example

Run this code
    
    
    #include <chrono>
    #include <iostream>
     
    int main()
    {
        const [std::chrono::time_point](time_point.html) now{[std::chrono::system_clock::now](system_clock/now.html)()};
     
        const std::chrono::year_month_day ymd{[std::chrono::floor](duration/floor.html)<[std::chrono::days](duration.html)>(now)};
     
        [std::cout](../io/cout.html) << "Current Year: " << static_cast<int>(ymd.year()) << ", "
                     "Month: " << static_cast<unsigned>(ymd.month()) << ", "
                     "Day: " << static_cast<unsigned>(ymd.day()) << "\n"
                     "ymd: " << ymd << '\n';
    }

Possible output: 
    
    
    Current Year: 2023, Month: 9, Day: 4
    ymd: 2023-09-04
