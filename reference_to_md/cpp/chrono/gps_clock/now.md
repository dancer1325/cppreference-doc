static [std::chrono::time_point](../time_point.html)<[std::chrono::gps_clock](../gps_clock.html)> now(); |  |  (since C++20)  
---|---|---  
| |   
  
Returns a time point representing the current point in time. The result is calculated as if by [std::chrono::gps_clock::from_utc](from_utc.html)([std::chrono::utc_clock::now](../utc_clock/now.html)()). Implementations may use a more accurate value of GPS time. 

### Return value

A time point representing the current time. 

### Example

Run this code
    
    
    #include <chrono>
    #include <cstddef>
    #include <iomanip>
    #include <iostream>
    #include <numeric>
    #include <vector>
     
    volatile int sink; // prevents optimization
     
    void do_some_work([std::size_t](../../types/size_t.html) size)
    {
        [std::vector](../../container/vector.html)<int> v(size, 42);
        sink = [std::accumulate](../../algorithm/accumulate.html)(v.begin(), v.end(), 0); // ensures side effect
    }
     
    int main()
    {
        [std::cout](../../io/cout.html) << [std::fixed](../../io/manip/fixed.html) << [std::setprecision](../../io/manip/setprecision.html)(9) << [std::left](../../io/manip/left.html);
        for (auto size{1ull}; size < 1000'000'000ull; size *= 100)
        {
            const auto start = std::chrono::gps_clock::now();
            do_some_work(size);
            const auto end = std::chrono::gps_clock::now();
            const [std::chrono::duration](../duration.html)<double> diff = end - start;
            [std::cout](../../io/cout.html) << "start = " << start << "; end = " << end << ";\n";
            [std::cout](../../io/cout.html) << "diff = " << diff << "; size = " << size << '\n';
        }
    }
