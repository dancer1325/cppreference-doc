template< class Dest, class Source, class Duration >  
auto clock_cast( const [std::chrono::time_point](time_point.html)<Source, Duration>& t ); |  |  (since C++20)  
| |   
  
Converts the time point `t` of a clock `Source` to an equivalent time point of the clock `Dest`, using [std::chrono::system_clock](system_clock.html "cpp/chrono/system clock") and/or std::chrono::utc_clock as intermediaries if necessary. 

  * If the expression [std::chrono::clock_time_conversion](clock_time_conversion.html)<Dest, Source>{}(t) is well-formed, returns the result of that expression. 
  * Otherwise, if at least one of the following two expressions are well-formed, then 



    

  * If both expressions are well-formed, the conversion is ambiguous, and the program is ill-formed. 
  * Otherwise, exactly one of the two expressions is well-formed; the result of that expression is returned. 



1) [std::chrono::clock_time_conversion](clock_time_conversion.html)<Dest, [std::chrono::system_clock](system_clock.html)>{}(  
[std::chrono::clock_time_conversion](clock_time_conversion.html)<[std::chrono::system_clock](system_clock.html), Source>{}(t))

2) [std::chrono::clock_time_conversion](clock_time_conversion.html)<Dest, [std::chrono::utc_clock](utc_clock.html)>{}(  
[std::chrono::clock_time_conversion](clock_time_conversion.html)<[std::chrono::utc_clock](utc_clock.html), Source>{}(t))

  * Otherwise, if at least one of the following two expressions are well-formed, then 



    

  * If both expressions are well-formed, the conversion is ambiguous, and the program is ill-formed. 
  * Otherwise, exactly one of the two expressions is well-formed; the result of that expression is returned. 



1) [std::chrono::clock_time_conversion](clock_time_conversion.html)<Dest, [std::chrono::utc_clock](utc_clock.html)>{}(  


[std::chrono::clock_time_conversion](clock_time_conversion.html)<[std::chrono::utc_clock](utc_clock.html), [std::chrono::system_clock](system_clock.html)>{}(  


[std::chrono::clock_time_conversion](clock_time_conversion.html)<[std::chrono::system_clock](system_clock.html), Source>{}(t)))

2) [std::chrono::clock_time_conversion](clock_time_conversion.html)<Dest, [std::chrono::system_clock](system_clock.html)>{}(  


[std::chrono::clock_time_conversion](clock_time_conversion.html)<[std::chrono::system_clock](system_clock.html), [std::chrono::utc_clock](utc_clock.html)>{}(  


[std::chrono::clock_time_conversion](clock_time_conversion.html)<[std::chrono::utc_clock](utc_clock.html), Source>{}(t)))

  * Otherwise, this function does not participate in overload resolution. 



### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/chrono/clock_cast&action=edit&section=1 "Edit section: Return value")] Return value

The result of the conversion, determined as described above.
