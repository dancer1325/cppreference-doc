[std::tm](tm.html)* gmtime( const [std::time_t](time_t.html)* time ); |  |   
| |   
  
Converts given time since epoch as [std::time_t](time_t.html "cpp/chrono/c/time t") value into calendar time, expressed in Coordinated Universal Time (UTC).

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/chrono/c/gmtime&action=edit&section=1 "Edit section: Parameters")] Parameters

time  |  \-  |  pointer to a time_t object to convert   
---|---|---  
  
### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/chrono/c/gmtime&action=edit&section=2 "Edit section: Return value")] Return value

Pointer to a static internal [std::tm](tm.html "cpp/chrono/c/tm") object on success, or null pointer otherwise. The structure may be shared between **std::gmtime** , [std::localtime](localtime.html "cpp/chrono/c/localtime"), and [std::ctime](ctime.html "cpp/chrono/c/ctime") and may be overwritten on each invocation. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/chrono/c/gmtime&action=edit&section=3 "Edit section: Notes")] Notes

This function may not be thread-safe. 

POSIX requires that [`gmtime`](https://pubs.opengroup.org/onlinepubs/9799919799/functions/gmtime.html) sets [errno](../../error/errno.html "cpp/error/errno") to [EOVERFLOW](../../error/errno_macros.html) if it fails because the argument is too large. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/chrono/c/gmtime&action=edit&section=4 "Edit section: Example")] Example

Run this code
    
    
    #include <ctime>
    #include <iomanip>
    #include <iostream>
    #include <sstream>
     
    int main()
    {
        setenv("TZ", "/usr/share/zoneinfo/Europe/London", 1); // POSIX-specific
     
        [std::tm](tm.html) tm{}; // get_time does not set all fields hence {}
        tm.tm_year = 2020 - 1900; // 2020
        tm.tm_mon = 7 - 1; // July
        tm.tm_mday = 15; // 15th
        tm.tm_hour = 10;
        tm.tm_min = 15;
        tm.tm_isdst = 1; // Daylight saving in London
        [std::time_t](time_t.html) t = [std::mktime](mktime.html)(&tm); 
     
        [std::cout](../../io/cout.html) << "UTC:   " << [std::put_time](../../io/manip/put_time.html)(std::gmtime(&t), "%c %Z") << '\n';
        [std::cout](../../io/cout.html) << "local: " << [std::put_time](../../io/manip/put_time.html)([std::localtime](localtime.html)(&t), "%c %Z") << '\n';
    }

Possible output: 
    
    
    UTC:   Wed Jul 15 09:15:00 2020 GMT
    local: Wed Jul 15 10:15:00 2020 BST
