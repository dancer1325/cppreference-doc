[std::time_t](time_t.html) mktime( [std::tm](tm.html)* time ); |  |   
| |   
  
Converts local calendar time to a time since epoch as a [std::time_t](time_t.html "cpp/chrono/c/time t") object. time->tm_wday and time->tm_yday are ignored. The values in time are permitted to be outside their normal ranges. 

A negative value of time->tm_isdst causes `mktime` to attempt to determine if Daylight Saving Time was in effect. 

If the conversion is successful, the time object is modified. All fields of time are updated to fit their proper ranges. time->tm_wday and time->tm_yday are recalculated using information available in other fields.

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/chrono/c/mktime&action=edit&section=2 "Edit section: Return value")] Return value

Time since epoch as a [std::time_t](time_t.html "cpp/chrono/c/time t") object on success or -1 if time cannot be represented as a [std::time_t](time_t.html "cpp/chrono/c/time t") object. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/chrono/c/mktime&action=edit&section=3 "Edit section: Notes")] Notes

If the [std::tm](tm.html "cpp/chrono/c/tm") object was obtained from [std::get_time](../../io/manip/get_time.html "cpp/io/manip/get time") or the POSIX [`strptime`](https://pubs.opengroup.org/onlinepubs/9799919799/functions/strptime.html), the value of `tm_isdst` is indeterminate, and needs to be set explicitly before calling `mktime`. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/chrono/c/mktime&action=edit&section=4 "Edit section: Example")] Example

Construct a local time explicitly.

Run this code
    
    
    #include <ctime>
    #include <iomanip>
    #include <iostream>
    #include <sstream>
     
    int main()
    {
        setenv("TZ", "/usr/share/zoneinfo/America/Los_Angeles", 1); // POSIX-specific
     
        [std::tm](tm.html) tm{}; // Zero initialise
        tm.tm_year = 2020 - 1900; // 2020
        tm.tm_mon = 2 - 1; // February
        tm.tm_mday = 15; // 15th
        tm.tm_hour = 10;
        tm.tm_min = 15;
        tm.tm_isdst = 0; // Not daylight saving
        [std::time_t](time_t.html) t = std::mktime(&tm); 
        [std::tm](tm.html) local = *[std::localtime](localtime.html)(&t);
     
        [std::cout](../../io/cout.html) << "local: " << [std::put_time](../../io/manip/put_time.html)(&local, "%c %Z") << '\n';
    }

Possible output: 
    
    
    local: Sat Feb 15 10:15:00 2020 PST
