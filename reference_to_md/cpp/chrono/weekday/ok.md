constexpr bool ok() const noexcept; |  |  (since C++20)  
---|---|---  
| |   
  
Checks if the weekday value stored in *this is in the valid range, i.e., `[`​0​`, `6`]`. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/chrono/weekday/ok&action=edit&section=1 "Edit section: Return value")] Return value

true if the weekday value stored in *this is in the range `[`​0​`, `6`]`. Otherwise false. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/chrono/weekday/ok&action=edit&section=2 "Edit section: Example")] Example

Run this code
    
    
    #include <chrono>
    #include <iomanip>
    #include <iostream>
    #include <locale>
    #include <string>
     
    struct weekday_ok : [std::numpunct](../../locale/numpunct.html)<char>
    {
        [std::string](../../string/basic_string.html) do_truename()  const override { return " (is valid weekday)"; }
        [std::string](../../string/basic_string.html) do_falsename() const override { return " (is not valid weekday)"; }
    };
     
    int main()
    {
        [std::cout](../../io/cout.html).imbue([std::locale](../../locale/locale.html)([std::cout](../../io/cout.html).getloc(), new weekday_ok));
        [std::cout](../../io/cout.html) << [std::boolalpha](../../io/manip/boolalpha.html);
     
        for (const unsigned u : {0 /* Sun */, 1 /* Mon */, 6, 7 /* Sun */, 8, 9})
        {
            const [std::chrono::weekday](../weekday.html) wd{u};
            [std::cout](../../io/cout.html) << "u: " << u << "; wd: " << wd.c_encoding() << wd.ok() << '\n';
        }
    }

Output: 
    
    
    u: 0; wd: 0 (is valid weekday)
    u: 1; wd: 1 (is valid weekday)
    u: 6; wd: 6 (is valid weekday)
    u: 7; wd: 0 (is valid weekday)
    u: 8; wd: 8 (is not valid weekday)
    u: 9; wd: 9 (is not valid weekday)
