1) The class template `std::generator` presents a [`view`](../ranges/view.html "cpp/ranges/view") of the elements yielded by the evaluation of a [coroutine](../language/coroutines.html "cpp/language/coroutines").

2) Convenience alias template for the `generator` using the [polymorphic allocator](../memory/polymorphic_allocator.html "cpp/memory/polymorphic allocator").

A `std::generator` generates a sequence of elements by repeatedly resuming the coroutine from which it was returned. Each time a co_yield statement is evaluated, the coroutine produces one element of the sequence. When the co_yield statement is of the form co_yield ranges::elements_of(rng), each element of the [`range`](../ranges/range.html "cpp/ranges/range") rng is successively produced as an element of the sequence. 

`std::generator` models [`view`](../ranges/view.html "cpp/ranges/view") and [`input_range`](../ranges/input_range.html "cpp/ranges/input range"). 

The behavior of a program that adds a specialization for `std::generator` is undefined.

  * When co_yield ranges::elements_of(rng) is evaluated in a generator body, rng is converted to a generator and added to the stack that contains the enclosing generator. 
  * When a generator iterator is [incremented](generator/iterator.html#increment "cpp/coroutine/generator/iterator"), the coroutine at the top of the associated stack is resumed. 
  * When a generator finishes (i.e. when [`promise_type::final_suspend`](generator/promise_type/final_suspend.html "cpp/coroutine/generator/promise type/final suspend") is called), it is removed from the stack.  
(exposition-only member object*)

### Example

Run this code
    
    
    #include <generator>
    #include <iostream>
     
    template<typename T>
    struct Tree
    {
        T value;
        Tree *left{}, *right{};
     
        std::generator<const T&> traverse_inorder() const
        {
            if (left)
                co_yield std::ranges::elements_of(left->traverse_inorder());
     
            co_yield value;
     
            if (right)
                co_yield std::ranges::elements_of(right->traverse_inorder());
        }
    };
     
    int main()
    {
        Tree<char> tree[]
        {
                                        {'D', tree + 1, tree + 2},
            //                            │
            //            ┌───────────────┴────────────────┐
            //            │                                │
                        {'B', tree + 3, tree + 4},       {'F', tree + 5, tree + 6},
            //            │                                │
            //  ┌─────────┴─────────────┐      ┌───────────┴─────────────┐
            //  │                       │      │                         │
              {'A'},                  {'C'}, {'E'},                    {'G'}
        };
     
        for (char x : tree->traverse_inorder())
            [std::cout](../io/cout.html) << x << ' ';
        [std::cout](../io/cout.html) << '\n';
    }

Output: 
    
    
    A B C D E F G
