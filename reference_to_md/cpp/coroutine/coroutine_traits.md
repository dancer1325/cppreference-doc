Defined in header `[<coroutine>](../header/coroutine.html "cpp/header/coroutine")` |  |   
---|---|---  
template< class R, class... Args >  
struct coroutine_traits; |  |  (since C++20)  
| |   
  
Determines the promise type from the return type and parameter types of a coroutine. The standard library implementation provides a publicly accessible member type `promise_type` same as `R::promise_type` if the qualified-id is valid and denotes a type. Otherwise, it has no such member. 

[Program-defined specializations](../language/type-id.html#Program-defined_type "cpp/language/type") of `coroutine_traits` must define a publicly accessible nested type `promise_type`, otherwise the program is ill-formed.

---  
  
### Template parameters

R  |  \-  |  return type of the coroutine   
---|---|---  
Args  |  \-  |  parameter types of the coroutine, including the [implicit object parameter](../language/member_functions.html#ref-qualified_member_functions "cpp/language/member functions") if the coroutine is a non-static member function   
  
### Nested types

Name  |  Definition   
---|---  
`promise_type` |  `R::promise_type` if it is valid, or provided by program-defined specializations   
  
### Possible implementation
    
    
    namespace detail {
    template<class, class...>
    struct coroutine_traits_base {};
     
    template<class R, class... Args>
    requires requires { typename R::promise_type; }
    struct coroutine_traits_base <R, Args...>
    {
        using promise_type = R::promise_type;
    };
    }
     
    template<class R, class... Args>
    struct coroutine_traits : detail::coroutine_traits_base<R, Args...> {};  
  
---  
  
### Notes

If the coroutine is a non-static member function, then the first type in `Args...` is the type of the implicit object parameter, and the rest are parameter types of the function (if any). 

If `std::coroutine_traits<R, Args...>::promise_type` does not exist or is not a class type, the corresponding coroutine definition is ill-formed. 

Users may define explicit or partial specializations of `coroutine_traits` dependent on program-defined types to avoid modification to return types. 

### Example

Run this code
    
    
    #include <chrono>
    #include <coroutine>
    #include <exception>
    #include <future>
    #include <iostream>
    #include <thread>
    #include <type_traits>
     
    // A program-defined type on which the coroutine_traits specializations below depend
    struct as_coroutine {};
     
    // Enable the use of std::future<T> as a coroutine type
    // by using a std::promise<T> as the promise type.
    template<typename T, typename... Args>
        requires(![std::is_void_v](../types/is_void.html)<T> && ![std::is_reference_v](../types/is_reference.html)<T>)
    struct std::coroutine_traits<[std::future](../thread/future.html)<T>, as_coroutine, Args...>
    {
        struct promise_type : [std::promise](../thread/promise.html)<T>
        {
            [std::future](../thread/future.html)<T> get_return_object() noexcept
            {
                return this->get_future();
            }
     
            [std::suspend_never](suspend_never.html) initial_suspend() const noexcept { return {}; }
            [std::suspend_never](suspend_never.html) final_suspend() const noexcept { return {}; }
     
            void return_value(const T& value)
                noexcept([std::is_nothrow_copy_constructible_v](../types/is_copy_constructible.html)<T>)
            {
                this->set_value(value);
            }
     
            void return_value(T&& value) noexcept([std::is_nothrow_move_constructible_v](../types/is_move_constructible.html)<T>)
            {
                this->set_value(std::move(value));
            }
     
            void unhandled_exception() noexcept
            {
                this->set_exception([std::current_exception](../error/current_exception.html)());
            }
        };
    };
     
    // Same for std::future<void>.
    template<typename... Args>
    struct std::coroutine_traits<[std::future](../thread/future.html)<void>, as_coroutine, Args...>
    {
        struct promise_type : [std::promise](../thread/promise.html)<void>
        {
            [std::future](../thread/future.html)<void> get_return_object() noexcept
            {
                return this->get_future();
            }
     
            [std::suspend_never](suspend_never.html) initial_suspend() const noexcept { return {}; }
            [std::suspend_never](suspend_never.html) final_suspend() const noexcept { return {}; }
     
            void return_void() noexcept
            {
                this->set_value();
            }
     
            void unhandled_exception() noexcept
            {
                this->set_exception([std::current_exception](../error/current_exception.html)());
            }
        };
    };
     
    // Allow co_await'ing std::future<T> and std::future<void>
    // by naively spawning a new thread for each co_await.
    template<typename T>
    auto operator co_await([std::future](../thread/future.html)<T> future) noexcept
        requires(![std::is_reference_v](../types/is_reference.html)<T>)
    {
        struct awaiter : [std::future](../thread/future.html)<T>
        {
            bool await_ready() const noexcept
            {
                using namespace std::chrono_literals;
                return this->wait_for(0s) != [std::future_status::timeout](../thread/future_status.html);
            }
     
            void await_suspend([std::coroutine_handle](coroutine_handle.html)<> cont) const
            {
                [std::thread](../thread/thread.html)([this, cont]
                {
                    this->wait();
                    cont();
                }).detach();
            }
     
            T await_resume() { return this->get(); }
        };
     
        return awaiter { std::move(future) };
    }
     
    // Utilize the infrastructure we have established.
    [std::future](../thread/future.html)<int> compute(as_coroutine)
    {
        int a = co_await [std::async](../thread/async.html)([] { return 6; });
        int b = co_await [std::async](../thread/async.html)([] { return 7; });
        co_return a * b;
    }
     
    [std::future](../thread/future.html)<void> fail(as_coroutine)
    {
        throw [std::runtime_error](../error/runtime_error.html)("bleah");
        co_return;
    }
     
    int main()
    {
        [std::cout](../io/cout.html) << compute({}).get() << '\n';
     
        try
        {
            fail({}).get();
        }
        catch (const [std::runtime_error](../error/runtime_error.html)& e)
        {
            [std::cout](../io/cout.html) << "error: " << e.what() << '\n';
        }
    }

Output: 
    
    
    42
    error: bleah
