/*iterator*/ begin(); |  |  (since C++23)  
---|---|---  
| |   
  
Pushes [`_coroutine__`](../generator.html#Data_members "cpp/coroutine/generator") into *[`_active__`](../generator.html#Data_members "cpp/coroutine/generator") stack, then evaluates coroutine_.resume(). 

Before invocation of `begin()` the [`_coroutine__`](../generator.html#Data_members "cpp/coroutine/generator") must refer to a coroutine suspended at its [initial suspend point](../../language/coroutines.html#co_await "cpp/language/coroutines"). 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/coroutine/generator/begin&action=edit&section=1 "Edit section: Return value")] Return value

An iterator whose `_coroutine__` member refers to the same coroutine as [`_coroutine__`](../generator.html#Data_members "cpp/coroutine/generator") does. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/coroutine/generator/begin&action=edit&section=2 "Edit section: Notes")] Notes

It is an undefined behavior to call `begin()` more than once on the same [`generator`](../generator.html "cpp/coroutine/generator") object.
