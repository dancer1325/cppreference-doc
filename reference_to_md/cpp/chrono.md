C++ includes support for two types of time manipulation: 

  * The [chrono library](chrono.html#chrono_library), a flexible collection of types that track time with varying degrees of precision (e.g., [std::chrono::time_point](chrono/time_point.html "cpp/chrono/time point")). 
  * [C-style date and time library](chrono.html#C-style_date_and_time_library) (e.g., [std::time](chrono/c/time.html "cpp/chrono/c/time")).

#### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/chrono&action=edit&section=4 "Edit section: Duration")] Duration

A duration consists of a span of time, defined as some number of ticks of some time unit. For example, "42 seconds" could be represented by a duration consisting of 42 ticks of a 1-second time unit.

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/chrono&action=edit&section=11 "Edit section: C-style date and time library")] [C-style date and time library](chrono/c.html "cpp/chrono/c")

Also provided are the C-style date and time functions, such as [std::time_t](chrono/c/time_t.html "cpp/chrono/c/time t"), [std::difftime](chrono/c/difftime.html "cpp/chrono/c/difftime"), and [CLOCKS_PER_SEC](chrono/c/CLOCKS_PER_SEC.html "cpp/chrono/c/CLOCKS PER SEC"). 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/chrono&action=edit&section=12 "Edit section: Example")] Example

Run this code
    
    
    #include <chrono>
    #include <iostream>
     
    long Fibonacci(unsigned n)
    {
        return n < 2 ? n : Fibonacci(n - 1) + Fibonacci(n - 2);
    }
     
    int main()
    {
        // Measures and displays an execution time of a function call.
        const auto start{[std::chrono::steady_clock::now](chrono/steady_clock/now.html)()};
        const auto fb{Fibonacci(42)};
        const auto finish{[std::chrono::steady_clock::now](chrono/steady_clock/now.html)()};
        const [std::chrono::duration](chrono/duration.html)<double> elapsed_seconds{finish - start};
     
        [std::cout](io/cout.html) << "Fibonacci(42): " << fb << "\nElapsed time: ";
    //  std::cout << elapsed_seconds.count() << "s\n"; // Before C++20
        [std::cout](io/cout.html) << elapsed_seconds << '\n'; // C++20's chrono::duration operator<<
     
        // Prints UTC and local time.
        const auto tp_utc{[std::chrono::system_clock::now](chrono/system_clock/now.html)()};
        [std::cout](io/cout.html) << "Current time 'UTC' is: " << tp_utc << "\n"
                     "Current time 'Local' is: "
                  << [std::chrono::current_zone](chrono/current_zone.html)()->to_local(tp_utc) << '\n';
    }

Possible output: 
    
    
    Fibonacci(42): 267914296
    Elapsed time: 0.728532s
    Current time 'UTC' is: 2025-02-10 06:22:39.420666960
    Current time 'Local' is: 2025-02-10 09:22:39.420666960

  *[Std]: Standard in which the feature is introduced; DR means defect report against that revision
