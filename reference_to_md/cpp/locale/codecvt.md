Class template `std::codecvt` encapsulates conversion of character strings, including wide and multibyte, from one encoding to another. All file I/O operations performed through [std::basic_fstream](../io/basic_fstream.html)<CharT> use the std::codecvt<CharT, char, [std::mbstate_t](../string/multibyte/mbstate_t.html)> facet of the locale imbued in the stream.

### Specializations

The standard library is guaranteed to provide the following specializations (they are [required to be implemented by any locale object](locale.html "cpp/locale/locale")): 

Defined in header `[<locale>](../header/locale.html "cpp/header/locale")`

std::codecvt<char16_t, char, [std::mbstate_t](../string/multibyte/mbstate_t.html)>  
(since C++11)(deprecated in C++20) |  conversion between UTF-16 and UTF-8

(since C++20)(deprecated) |  conversion between UTF-16 and UTF-8

(since C++11)(deprecated in C++20) |  conversion between UTF-32 and UTF-8

(since C++20)(deprecated) |  conversion between UTF-32 and UTF-8   
std::codecvt<wchar_t, char, [std::mbstate_t](../string/multibyte/mbstate_t.html)> |  conversion between the system's native wide and the single-byte narrow character sets

### Example

The following examples reads a UTF-8 file using a locale which implements UTF-8 conversion in codecvt<wchar_t, char, [std::mbstate_t](../string/multibyte/mbstate_t.html)> and converts a UTF-8 string to UTF-16 using one of the standard specializations of `std::codecvt`.

Run this code
    
    
    #include <codecvt>
    #include <cstdint>
    #include <fstream>
    #include <iomanip>
    #include <iostream>
    #include <locale>
    #include <string>
    ¬†
    // utility wrapper to adapt locale-bound facets for wstring/wbuffer convert
    template<class Facet>
    struct deletable_facet : Facet
    {
        template<class... Args>
        deletable_facet(Args&&... args) : Facet([std::forward](../utility/forward.html)<Args>(args)...) {}
        ~deletable_facet() {}
    };
    ¬†
    int main()
    {
        // UTF-8 narrow multibyte encoding
        [std::string](../string/basic_string.html) data = reinterpret_cast<const char*>(+u8"z\u00df\u6c34\U0001f34c");
                           // or reinterpret_cast<const char*>(+u8"z√üÊ∞¥üçå")
                           // or "\x7a\xc3\x9f\xe6\xb0\xb4\xf0\x9f\x8d\x8c"
    ¬†
        [std::ofstream](../io/basic_ofstream.html)("text.txt") << data;
    ¬†
        // using system-supplied locale's codecvt facet
        [std::wifstream](../io/basic_ifstream.html) fin("text.txt");
        // reading from wifstream will use codecvt<wchar_t, char, std::mbstate_t>
        // this locale's codecvt converts UTF-8 to UCS4 (on systems such as Linux)
        fin.imbue([std::locale](locale.html)("en_US.UTF-8"));
        [std::cout](../io/cout.html) << "The UTF-8 file contains the following UCS4 code units:\n" << [std::hex](../io/manip/hex.html);
        for (wchar_t c; fin >> c;)
            [std::cout](../io/cout.html) << "U+" << [std::setw](../io/manip/setw.html)(4) << [std::setfill](../io/manip/setfill.html)('0')
                      << static_cast<uint32_t>(c) << ' ';
    ¬†
        // using standard (locale-independent) codecvt facet
        [std::wstring_convert](wstring_convert.html)<
            deletable_facet<std::codecvt<char16_t, char, [std::mbstate_t](../string/multibyte/mbstate_t.html)>>, char16_t> conv16;
        [std::u16string](../string/basic_string.html) str16 = conv16.from_bytes(data);
    ¬†
        [std::cout](../io/cout.html) << "\n\nThe UTF-8 file contains the following UTF-16 code units:\n"
                  << [std::hex](../io/manip/hex.html);
        for (char16_t c : str16)
            [std::cout](../io/cout.html) << "U+" << [std::setw](../io/manip/setw.html)(4) << [std::setfill](../io/manip/setfill.html)('0')
                      << static_cast<uint16_t>(c) << ' ';
        [std::cout](../io/cout.html) << '\n';
    }

Output: 
    
    
    The UTF-8 file contains the following UCS4 code units:
    U+007a U+00df U+6c34 U+1f34c 
    ¬†
    The UTF-8 file contains the following UTF-16 code units:
    U+007a U+00df U+6c34 U+d83c U+df4c
