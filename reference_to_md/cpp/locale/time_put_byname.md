### Specializations

The standard library is guaranteed to provide every specialization that satisfies the following type requirements: 

  * `CharT` is one of char and wchar_t, and 
  * `OutputIt` must meet the requirements of [LegacyOutputIterator](../named_req/OutputIterator.html "cpp/named req/OutputIterator").

Constructs a new `std::time_put_byname` facet for a locale with name. 

refs is used for resource management: if refs == 0, the implementation destroys the facet, when the last [std::locale](locale.html "cpp/locale/locale") object holding it is destroyed. Otherwise, the object is not destroyed.

### Example

Prints current time using the "C" locale with the `time_put` facet replaced by various `std::time_put_byname` facets. The result shown was obtained using the clang compiler.

Run this code
    
    
    #include <codecvt>
    #include <ctime>
    #include <iomanip>
    #include <iostream>
     
    int main()
    {
        [std::time_t](../chrono/c/time_t.html) t = [std::time](../chrono/c/time.html)(nullptr);
        [std::wbuffer_convert](wbuffer_convert.html)<[std::codecvt_utf8](codecvt_utf8.html)<wchar_t>> conv([std::cout](../io/cout.html).rdbuf());
        [std::wostream](../io/basic_ostream.html) out(&conv);
     
        out.imbue([std::locale](locale.html)(out.getloc(),
                              new std::time_put_byname<wchar_t>("ja_JP.utf8")));
        out << [std::put_time](../io/manip/put_time.html)([std::localtime](../chrono/c/localtime.html)(&t), L"%A %c") << '\n';
     
        out.imbue([std::locale](locale.html)(out.getloc(),
                              new std::time_put_byname<wchar_t>("ru_RU.utf8")));
        out << [std::put_time](../io/manip/put_time.html)([std::localtime](../chrono/c/localtime.html)(&t), L"%A %c") << '\n';
     
        out.imbue([std::locale](locale.html)(out.getloc(),
                              new std::time_put_byname<wchar_t>("sv_SE.utf8")));
        out << [std::put_time](../io/manip/put_time.html)([std::localtime](../chrono/c/localtime.html)(&t), L"%A %c") << '\n';
    }

Possible output: 
    
    
    木曜日 2023年10月05日 19時44分51秒
    Четверг Чт 05 окт 2023 19:44:51
    torsdag tor  5 okt 2023 19:44:51
