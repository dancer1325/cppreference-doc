protected:  
virtual CharT do_decimal_point() const; |  (2)  |   
| |   
  
1) Public member function, calls the member function `do_decimal_point` of the most derived class.

2) Returns the character to use as the decimal point separator in monetary I/O if the format uses fractions (that is, if [do_frac_digits()](frac_digits.html "cpp/locale/moneypunct/frac digits") is greater than zero). For typical U.S. locales, it is the character '.' (or L'.').

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/locale/moneypunct/decimal_point&action=edit&section=1 "Edit section: Return value")] Return value

The object of type `CharT` holding the decimal point character. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/locale/moneypunct/decimal_point&action=edit&section=2 "Edit section: Example")] Example

Run this code
    
    
    #include <iomanip>
    #include <iostream>
    #include <locale>
     
    void show_dpt(const char* locname)
    {
        [std::locale](../locale.html) loc(locname);
        [std::cout](../../io/cout.html).imbue(loc);
        [std::cout](../../io/cout.html) << locname << " decimal point is '"
                  << [std::use_facet](../use_facet.html)<[std::moneypunct](../moneypunct.html)<char>>(loc).decimal_point()
                  << "' for example: " << [std::showbase](../../io/manip/showbase.html) << [std::put_money](../../io/manip/put_money.html)(123);
        if ([std::use_facet](../use_facet.html)<[std::moneypunct](../moneypunct.html)<char>>(loc).frac_digits() == 0)
            [std::cout](../../io/cout.html) << " (does not use frac digits)";
     
        [std::cout](../../io/cout.html) << '\n';
    }
     
    int main()
    {
        show_dpt("en_US.utf8");
        show_dpt("ja_JP.utf8");
        show_dpt("sv_SE.utf8");
        show_dpt("de_DE.utf8");
    }

Output: 
    
    
    en_US.utf8 decimal point is '.' for example: $1.23
    ja_JP.utf8 decimal point is '.' for example: ￥123 (does not use frac digits)
    sv_SE.utf8 decimal point is ',' for example: 1,23 kr
    de_DE.utf8 decimal point is ',' for example: 1,23 €
