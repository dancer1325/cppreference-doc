template< class CharT >  
class collate; |  |

Class `std::collate` encapsulates locale-specific collation (comparison) and hashing of strings. This facet is used by [std::basic_regex](../regex/basic_regex.html "cpp/regex/basic regex") and can be applied, by means of [`std::locale::operator()`](locale/operator\(\).html "cpp/locale/locale/operator\(\)"), directly to all standard algorithms that expect a string comparison predicate.

### Specializations

The standard library is guaranteed to provide the following specializations (they are [required to be implemented by any locale object](locale.html "cpp/locale/locale")): 

Defined in header `[<locale>](../header/locale.html "cpp/header/locale")`  
---  
std::collate<char> |  implements lexicographical ordering of byte strings   
std::collate<wchar_t> |  implements lexicographical ordering of wide strings

### Example

Run this code
    
    
    #include <algorithm>
    #include <iostream>
    #include <locale>
    #include <string>
    #include <vector>
     
    int main()
    {
        [std::locale::global](locale/global.html)([std::locale](locale.html)("en_US.utf8"));
        [std::wcout](../io/cout.html).imbue([std::locale](locale.html)(""));
        [std::vector](../container/vector.html)<[std::wstring](../string/basic_string.html)> v
        {
            L"ar", L"zebra", L"\u00f6grupp", L"Zebra",
            L"\u00e4ngel",L"\u00e5r", L"f\u00f6rnamn"
        };
     
        [std::wcout](../io/cout.html) << "Default locale collation order: ";
        [std::sort](../algorithm/sort.html)(v.begin(), v.end());
        for (auto s : v)
            [std::wcout](../io/cout.html) << s << ' ';
        [std::wcout](../io/cout.html) << '\n';
     
        [std::wcout](../io/cout.html) << "English locale collation order: ";
        [std::sort](../algorithm/sort.html)(v.begin(), v.end(), [std::locale](locale.html)("en_US.UTF-8"));
        for (auto s : v)
            [std::wcout](../io/cout.html) << s << ' ';
        [std::wcout](../io/cout.html) << '\n';
     
        [std::wcout](../io/cout.html) << "Swedish locale collation order: ";
        [std::sort](../algorithm/sort.html)(v.begin(), v.end(), [std::locale](locale.html)("sv_SE.UTF-8"));
        for (auto s : v)
            [std::wcout](../io/cout.html) << s << ' ';
        [std::wcout](../io/cout.html) << '\n';
    }

Output: 
    
    
    Default locale collation order: Zebra ar förnamn zebra ängel år ögrupp
    English locale collation order: ängel ar år förnamn ögrupp zebra Zebra
    Swedish locale collation order: ar förnamn zebra Zebra år ängel ögrupp
