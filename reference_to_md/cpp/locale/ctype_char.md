class ctype<char>; |  |   
| |   
  
This specialization of [std::ctype](ctype.html "cpp/locale/ctype") encapsulates character classification features for type char. Unlike general-purpose [std::ctype](ctype.html "cpp/locale/ctype"), which uses virtual functions, this specialization uses table lookup to classify characters (which is generally faster). 

The base class `std::ctype`<char> implements character classification equivalent to the minimal "C" locale. The classification rules can be extended or modified if constructed with a non-default classification table argument, if constructed as [std::ctype_byname](ctype_byname.html)<char> or as a user-defined derived facet. All [std::istream](../io/basic_istream.html "cpp/io/basic istream") formatted input functions are required to use `std::ctype`<char> for character classing during input parsing.

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/locale/ctype_char&action=edit&section=5 "Edit section: Example")] Example

The following example demonstrates modification of ctype<char> to tokenize comma-separated values:

Run this code
    
    
    #include <cstddef>
    #include <iostream>
    #include <locale>
    #include <sstream>
    #include <vector>
     
    // This ctype facet classifies commas and endlines as whitespace
    struct csv_whitespace : [std::ctype](ctype.html)<char>
    {
        static const mask* make_table()
        {
            // make a copy of the "C" locale table
            static [std::vector](../container/vector.html)<mask> v(classic_table(), classic_table() + table_size);
            v[','] |=  space; // comma will be classified as whitespace
            v[' '] &= ~space; // space will not be classified as whitespace
            return &v[0];
        }
     
        csv_whitespace([std::size_t](../types/size_t.html) refs = 0) : ctype(make_table(), false, refs) {}
    };
     
    int main()
    {
        [std::string](../string/basic_string.html) in = "Column 1,Column 2,Column 3\n123,456,789";
        [std::string](../string/basic_string.html) token;
     
        [std::cout](../io/cout.html) << "Default locale:\n";
        [std::istringstream](../io/basic_istringstream.html) s1(in);
        while (s1 >> token)
            [std::cout](../io/cout.html) << "  " << token << '\n';
     
        [std::cout](../io/cout.html) << "Locale with modified ctype:\n";
        [std::istringstream](../io/basic_istringstream.html) s2(in);
        s2.imbue([std::locale](locale.html)(s2.getloc(), new csv_whitespace));
        while (s2 >> token)
            [std::cout](../io/cout.html) << "  " << token << '\n';
    }

Output: 
    
    
    Default locale:
      Column
      1,Column
      2,Column
      3
      123,456,789
    Locale with modified ctype:
      Column 1
      Column 2
      Column 3
      123
      456
      789
