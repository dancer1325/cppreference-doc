`std::codecvt_utf8` is a [std::codecvt](codecvt.html "cpp/locale/codecvt") facet which encapsulates conversion between a UTF-8 encoded byte string and UCS-2 or UTF-32 character string (depending on the type of `Elem`). This [std::codecvt](codecvt.html "cpp/locale/codecvt") facet can be used to read and write UTF-8 files, both text and binary. 

UCS-2 is an archaic encoding that is a subset of UTF-16, which encodes scalar values in the range U+0000-U+FFFF (Basic Multilingual Plane) only.

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/locale/codecvt_utf8&action=edit&section=4 "Edit section: Notes")] Notes

Although the standard requires that this facet works with UCS-2 when the size of `Elem` is 16 bits, some implementations use UTF-16 instead. The term "UCS-2" was deprecated and removed from ISO 10646. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/locale/codecvt_utf8&action=edit&section=5 "Edit section: Example")] Example

The following example demonstrates the difference between UCS-2/UTF-8 and UTF-16/UTF-8 conversions: the third character in the string is not a valid UCS-2 character.

Run this code
    
    
    #include <codecvt>
    #include <cstdint>
    #include <iostream>
    #include <locale>
    #include <string>
     
    int main()
    {
        // UTF-8 data. The character U+1d10b, musical sign segno, does not fit in UCS-2
        [std::string](../string/basic_string.html) utf8 = "z\u6c34\U0001d10b";
     
        // the UTF-8 / UTF-16 standard conversion facet
        [std::wstring_convert](wstring_convert.html)<[std::codecvt_utf8_utf16](codecvt_utf8_utf16.html)<char16_t>, char16_t> utf16conv;
        [std::u16string](../string/basic_string.html) utf16 = utf16conv.from_bytes(utf8);
        [std::cout](../io/cout.html) << "UTF-16 conversion produced " << utf16.size() << " code units:\n"
                  << [std::showbase](../io/manip/showbase.html) << [std::hex](../io/manip/hex.html);
        for (char16_t c : utf16)
            [std::cout](../io/cout.html) << static_cast<[std::uint16_t](../types/integer.html)>(c) << ' ';
     
        // the UTF-8 / UCS-2 standard conversion facet
        [std::wstring_convert](wstring_convert.html)<std::codecvt_utf8<char16_t>, char16_t> ucs2conv;
        try
        {
            [std::u16string](../string/basic_string.html) ucs2 = ucs2conv.from_bytes(utf8);
        }
        catch(const [std::range_error](../error/range_error.html)& e)
        {
            [std::u16string](../string/basic_string.html) ucs2 = ucs2conv.from_bytes(utf8.substr(0, ucs2conv.converted()));
            [std::cout](../io/cout.html) << "\nUCS-2 failed after producing " << [std::dec](../io/manip/hex.html) << ucs2.size()
                      << " characters:\n" << [std::showbase](../io/manip/showbase.html) << [std::hex](../io/manip/hex.html);
            for (char16_t c : ucs2)
                [std::cout](../io/cout.html) << static_cast<[std::uint16_t](../types/integer.html)>(c) << ' ';
            [std::cout](../io/cout.html) << '\n';
        }
    }

Output: 
    
    
    UTF-16 conversion produced 4 code units:
    0x7a 0x6c34 0xd834 0xdd0b
    UCS-2 failed after producing 2 characters:
    0x7a 0x6c34
