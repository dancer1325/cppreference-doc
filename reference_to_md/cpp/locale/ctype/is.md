1,2) Public member function, calls the protected virtual member function `do_is` of the most derived class.

3) Checks if the character c is classified by the mask m.

4) For every character in the character array `[`low`, `high`)`, identifies the complete classification mask (e.g. digit|xdigit|alnum|print|graph for the digit '0' in the default locale), and stores the masks in the corresponding elements of the array pointed to by vec.

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/locale/ctype/is&action=edit&section=1 "Edit section: Parameters")] Parameters

c  |  \-  |  character to classify   
---|---|---  
m  |  \-  |  mask to use for classifying a single character   
low  |  \-  |  pointer to the first character in an array of characters to classify   
high  |  \-  |  one past the end pointer for the array of characters to classify   
vec  |  \-  |  pointer to the first element of the array of masks to fill   
  
### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/locale/ctype/is&action=edit&section=2 "Edit section: Return value")] Return value

1,3) true if c is classified by m.

2,4) high

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/locale/ctype/is&action=edit&section=3 "Edit section: Example")] Example

Run this code
    
    
    #include <cstddef>
    #include <iostream>
    #include <locale>
    #include <utility>
    #include <vector>
     
    // utility wrapper to make locale-bound facets destructible
    template<class Facet>
    struct deletable_facet : Facet
    {
        template<class ...Args>
        deletable_facet(Args&& ...args) : Facet([std::forward](../../utility/forward.html)<Args>(args)...) {}
        ~deletable_facet() {}
    };
     
    int main()
    {
        // classify a single character using the default locale
        auto& f = [std::use_facet](../use_facet.html)<[std::ctype](../ctype.html)<char>>([std::locale](../locale.html)());
        char c = '0';
        if (f.is([std::ctype_base::digit](../ctype_base.html), c)) // or isdigit(c, locale());
            [std::cout](../../io/cout.html) << '\'' << c << "' is a digit\n";
     
        // classify every character in a string using a named locale
        deletable_facet<[std::ctype_byname](../ctype_byname.html)<wchar_t>> f2("en_US.utf8");
        [std::wstring](../../string/basic_string.html) str = L"z\u00df\u6c34\U0001d10b";
        [std::vector](../../container/vector.html)<[std::ctype_base::mask](../ctype_base.html)> vec(str.size());
        f2.is(&str[0], &str[0] + str.size(), &vec[0]);
     
        for ([std::size_t](../../types/size_t.html) n = 0; n < str.size(); ++n)
        {
            [std::cout](../../io/cout.html) << [std::hex](../../io/manip/hex.html) << "U+" << static_cast<wint_t>(str[n]) << " is: ";
            if (vec[n] & [std::ctype_base::alnum](../ctype_base.html)) 
                [std::cout](../../io/cout.html) << "alnum";
            if (vec[n] & [std::ctype_base::punct](../ctype_base.html)) 
                [std::cout](../../io/cout.html) << "punct";
            [std::cout](../../io/cout.html) << '\n';
        }
    }

Output: 
    
    
    '0' is a digit
    U+7a is: alnum 
    U+df is: alnum 
    U+6c34 is: alnum 
    U+1d10b is: punct
