`std::messages_byname` is a [std::messages](messages.html "cpp/locale/messages") facet which encapsulates retrieval of strings from message catalogs of the locale specified at its construction.

##  std::messages_byname::messages_byname

| |   
  
Constructs a new `std::messages_byname` facet for a locale with name. 

refs is used for resource management: if refs == 0, the implementation destroys the facet, when the last [std::locale](locale.html "cpp/locale/locale") object holding it is destroyed. Otherwise, the object is not destroyed. 

###  Parameters

### Example

Run this code
    
    
    #include <iostream>
    #include <locale>
     
    void try_with(const [std::locale](locale.html)& loc)
    {
        const [std::messages](messages.html)<char>& facet = [std::use_facet](use_facet.html)<[std::messages](messages.html)<char>>(loc);
     
        [std::messages](messages.html)<char>::catalog cat = facet.open("sed", [std::cout](../io/cout.html).getloc());
        if (cat < 0)
            [std::cout](../io/cout.html) << "Could not open \"sed\" message catalog\n";
        else
            [std::cout](../io/cout.html) << "\"No match\" "
                      << facet.get(cat, 0, 0, "No match") << '\n'
                      << "\"Memory exhausted\" " 
                      << facet.get(cat, 0, 0, "Memory exhausted") << '\n';
     
        facet.close(cat);
    }
     
    int main()
    {
        [std::locale](locale.html) loc("en_US.utf8");
        [std::cout](../io/cout.html).imbue(loc);
     
        try_with([std::locale](locale.html)(loc, new std::messages_byname<char>("de_DE.utf8")));
        try_with([std::locale](locale.html)(loc, new std::messages_byname<char>("fr_FR.utf8")));
        try_with([std::locale](locale.html)(loc, new std::messages_byname<char>("ja_JP.utf8")));
    }

Possible output: 
    
    
    "No match" Keine Übereinstimmung
    "Memory exhausted" Speicher erschöpft
    "No match" Pas de concordance
    "Memory exhausted" Mémoire épuisée
    "No match" 照合しません
    "Memory exhausted" メモリーが足りません
