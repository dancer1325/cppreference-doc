Constructs a new `std::numpunct_byname` facet for a locale with name. 

refs is used for resource management: if refs == 0, the implementation destroys the facet, when the last [std::locale](locale.html "cpp/locale/locale") object holding it is destroyed. Otherwise, the object is not destroyed.

### Example

This example demonstrates how to apply numeric punctuation rules of another language without changing the rest of the locale.

Run this code
    
    
    #include <iostream>
    #include <locale>
    Â 
    int main()
    {
        const double number = 1000.25;
        [std::wcout](../io/cout.html) << L"default locale: " << number << L'\n';
        [std::wcout](../io/cout.html).imbue([std::locale](locale.html)([std::wcout](../io/cout.html).getloc(),
                                     new std::numpunct_byname<wchar_t>("ru_RU.UTF8")));
        [std::wcout](../io/cout.html) << L"default locale with russian numpunct: " << number << L'\n';
    }

Output: 
    
    
    default locale: 1000.25
    default locale with russian numpunct: 1 000,25
