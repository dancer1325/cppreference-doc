Converts a byte sequence to a wide string using the facet pointed to by `_[cvtptr](../wstring_convert.html#cvtptr "cpp/locale/wstring convert")_` Ôªø. 

1) The byte sequence only consists of one element byte.

2) The byte sequence is the null-terminated sequence beginning at ptr.

3) The byte sequence is the sequence contained in str.

4) The byte sequence is the range `[`first`, `last`)`.

Before the conversion begins, if *this was **not** constructed with constructor overload [(3)](wstring_convert.html#Version_3 "cpp/locale/wstring convert/wstring convert"), `_[cvtstate](../wstring_convert.html#cvtstate "cpp/locale/wstring convert")_` will be set to its default value (the initial conversion state). 

The number of input elements successfully converted will be stored in `_[cvtcount](../wstring_convert.html#cvtcount "cpp/locale/wstring convert")_` Ôªø.

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/locale/wstring_convert/from_bytes&action=edit&section=1 "Edit section: Return value")] Return value

If the conversion succeeds, returns the conversion result. Otherwise, if *this is constructed with constructor overload [(4)](wstring_convert.html#Version_4 "cpp/locale/wstring convert/wstring convert"), returns `_[wide_err_string](../wstring_convert.html#wide_err_string "cpp/locale/wstring convert")_`. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/locale/wstring_convert/from_bytes&action=edit&section=2 "Edit section: Exceptions")] Exceptions

If the conversion fails and *this was **not** constructed with constructor overload [(4)](wstring_convert.html#Version_4 "cpp/locale/wstring convert/wstring convert"), throws [std::range_error](../../error/range_error.html "cpp/error/range error"). 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/locale/wstring_convert/from_bytes&action=edit&section=3 "Edit section: Example")] Example

Run this code
    
    
    #include <codecvt>
    #include <cstdint>
    #include <iostream>
    #include <locale>
    #include <string>
    ¬†
    int main()
    {
        [std::string](../../string/basic_string.html) utf8 = "z\u00df\u6c34\U0001d10b"; // or u8"z√üÊ∞¥ùÑã"
                     // or "\x7a\xc3\x9f\xe6\xb0\xb4\xf0\x9d\x84\x8b";
    ¬†
        // the UTF-8 / UTF-16 standard conversion facet
        [std::u16string](../../string/basic_string.html) utf16 = 
            [std::wstring_convert](../wstring_convert.html)<[std::codecvt_utf8_utf16](../codecvt_utf8_utf16.html)<char16_t>,
                                 char16_t>{}.from_bytes(utf8.data());
        [std::cout](../../io/cout.html) << "UTF-16 conversion produced " << utf16.size()
                  << " code units: " << [std::showbase](../../io/manip/showbase.html);
        for (char16_t c : utf16)
            [std::cout](../../io/cout.html) << [std::hex](../../io/manip/hex.html) << static_cast<[std::uint16_t](../../types/integer.html)>(c) << ' ';
    ¬†
        // the UTF-8 / UTF-32 standard conversion facet
        [std::u32string](../../string/basic_string.html) utf32 =
            [std::wstring_convert](../wstring_convert.html)<[std::codecvt_utf8](../codecvt_utf8.html)<char32_t>, char32_t>{}.from_bytes(utf8);
        [std::cout](../../io/cout.html) << "\nUTF-32 conversion produced " << [std::dec](../../io/manip/hex.html)
                  << utf32.size() << " code units: ";
        for (char32_t c : utf32)
            [std::cout](../../io/cout.html) << [std::hex](../../io/manip/hex.html) << static_cast<[std::uint32_t](../../types/integer.html)>(c) << ' ';
        [std::cout](../../io/cout.html) << '\n';
    }

Output: 
    
    
    UTF-16 conversion produced 5 code units: 0x7a 0xdf 0x6c34 0xd834 0xdd0b
    UTF-32 conversion produced 4 code units: 0x7a 0xdf 0x6c34 0x1d10b
