C++ includes a variety of utility libraries that provide functionality ranging from [bit-counting](utility/bitset.html "cpp/utility/bitset") to [partial function application](utility/functional/bind.html "cpp/utility/functional/bind"). These libraries can be broadly divided into two groups: 

  * language support libraries, and 
  * general-purpose libraries.

## [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/utility&action=edit&section=1 "Edit section: Language support")] Language support

Language support libraries provide classes and functions that interact closely with language features and support common language idioms. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/utility&action=edit&section=2 "Edit section: Implementation properties \(since C++20\)")] [Implementation properties](utility/feature_test.html "cpp/utility/feature test") (since C++20)

The header [`<version>`](header/version.html "cpp/header/version") supplies implementation-dependent information about the C++ standard library (such as the version number and release date). It also defines the [library feature-test macros](utility/feature_test.html "cpp/utility/feature test"). 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/utility&action=edit&section=3 "Edit section: Type support")] [Type support](utility/rtti.html "cpp/types")

Basic types (e.g. [std::size_t](types/size_t.html "cpp/types/size t"), [std::nullptr_t](types/nullptr_t.html "cpp/types/nullptr t")), RTTI (e.g. [std::type_info](types/type_info.html "cpp/types/type info")) 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/utility&action=edit&section=4 "Edit section: Program utilities")] [Program utilities](utility/program.html "cpp/utility/program")

Termination (e.g. [std::abort](utility/program/abort.html "cpp/utility/program/abort"), [std::atexit](utility/program/atexit.html "cpp/utility/program/atexit")), environment (e.g. [std::system](utility/program/system.html "cpp/utility/program/system")), signals (e.g. [std::raise](utility/program/raise.html "cpp/utility/program/raise")). 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/utility&action=edit&section=5 "Edit section: Dynamic memory management")] [Dynamic memory management](memory.html "cpp/memory")

Smart pointers (e.g. [std::shared_ptr](memory/shared_ptr.html "cpp/memory/shared ptr")), allocators (e.g. [std::allocator](memory/allocator.html "cpp/memory/allocator") or [std::pmr::memory_resource](memory/memory_resource.html "cpp/memory/memory resource")), C-style memory management (e.g. [std::malloc](memory/c/malloc.html "cpp/memory/c/malloc")). 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/utility&action=edit&section=6 "Edit section: Error handling")] [Error handling](error.html "cpp/error")

Exceptions (e.g. [std::exception](error/exception.html "cpp/error/exception"), [std::terminate](error/terminate.html "cpp/error/terminate")), assertions (e.g. [assert](error/assert.html "cpp/error/assert")). 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/utility&action=edit&section=7 "Edit section: Variadic functions")] [Variadic functions](utility/variadic.html "cpp/utility/variadic")

Support for functions that take an arbitrary number of parameters (via e.g. [va_start](utility/variadic/va_start.html "cpp/utility/variadic/va start"), [va_arg](utility/variadic/va_arg.html "cpp/utility/variadic/va arg"), [va_end](utility/variadic/va_end.html "cpp/utility/variadic/va end")).
