/*unspecified*/ setiosflags( [std::ios_base::fmtflags](../ios_base/fmtflags.html) mask ); |  |   
| |   
  
When used in an expression out << setiosflags(mask) or in >> setiosflags(mask), sets all format flags of the stream out or in as specified by the mask. 

## Contents

  * [1 Parameters](setiosflags.html#Parameters)
  * [2 Return value](setiosflags.html#Return_value)
  * [3 Example](setiosflags.html#Example)
  * [4 Defect reports](setiosflags.html#Defect_reports)
  * [5 See also](setiosflags.html#See_also)

  
---  
  
### Parameters

mask  |  \-  |  bitmask of the flags to set   
---|---|---  
  
### Return value

An object of unspecified type such that 

  * if out is an object of type [std::basic_ostream](../basic_ostream.html)<CharT, Traits>, the expression out << setiosflags(mask)
    * has type [std::basic_ostream](../basic_ostream.html)<CharT, Traits>&
    * has value out
    * behaves as if it called f(out, mask)
  * if in is an object of type [std::basic_istream](../basic_istream.html)<CharT, Traits>, the expression in >> setiosflags(mask)
    * has type [std::basic_istream](../basic_istream.html)<CharT, Traits>&
    * has value in
    * behaves as if it called f(in, mask)



where the function f is defined as: 
    
    
    void f([std::ios_base](../ios_base.html)& str, [std::ios_base::fmtflags](../ios_base/fmtflags.html) mask)
    {
        // set specified flags
        str.setf(mask);
    }

### Example

Run this code
    
    
    #include <iomanip>
    #include <iostream>
    Â 
    int main()
    {
        [std::cout](../cout.html) << [std::resetiosflags](resetiosflags.html)([std::ios_base::dec](../ios_base/fmtflags.html)) 
                  << std::setiosflags(  [std::ios_base::hex](../ios_base/fmtflags.html)
                                      | [std::ios_base::uppercase](../ios_base/fmtflags.html)
                                      | [std::ios_base::showbase](../ios_base/fmtflags.html)) << 42 << '\n';
    }

Output: 
    
    
    0X2A

### Defect reports

The following behavior-changing defect reports were applied retroactively to previously published C++ standards. 

DR  | Applied to  | Behavior as published  | Correct behavior   
---|---|---|---  
[LWG 183](https://cplusplus.github.io/LWG/issue183) | C++98  | `setiosflags` could only be used with streams  
of type [std::ostream](../basic_ostream.html "cpp/io/basic ostream") or [std::istream](../basic_istream.html "cpp/io/basic istream") | usable with any  
character stream
