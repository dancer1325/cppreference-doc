Modifies the default formatting for floating-point output. 

1) Sets the `floatfield` of the stream str to `fixed` as if by calling str.setf([std::ios_base::fixed](../ios_base/fmtflags.html), [std::ios_base::floatfield](../ios_base/fmtflags.html)).

2) Sets the `floatfield` of the stream str to `scientific` as if by calling str.setf([std::ios_base::scientific](../ios_base/fmtflags.html), [std::ios_base::floatfield](../ios_base/fmtflags.html)).

3) Sets the `floatfield` of the stream str to `fixed` and `scientific` simultaneously as if by calling str.setf([std::ios_base::fixed](../ios_base/fmtflags.html) | [std::ios_base::scientific](../ios_base/fmtflags.html), [std::ios_base::floatfield](../ios_base/fmtflags.html)). This enables hexadecimal floating-point formatting.

4) Sets the `floatfield` of the stream str to zero, as if by calling str.unsetf([std::ios_base::floatfield](../ios_base/fmtflags.html)). This enables the default floating-point formatting, which is different from fixed and scientific.

This is an I/O manipulator, it may be called with an expression such as out << std::fixed for any `out` of type [std::basic_ostream](../basic_ostream.html "cpp/io/basic ostream") (or with an expression such as in >> std::scientific for any `in` of type [std::basic_istream](../basic_istream.html "cpp/io/basic istream")).

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/io/manip/fixed&action=edit&section=3 "Edit section: Notes")] Notes

Hexadecimal floating-point formatting ignores the stream precision specification, as required by the specification of [std::num_put::do_put](../../locale/num_put/put.html "cpp/locale/num put/put"). 

These manipulators do not affect floating-point parsing. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/io/manip/fixed&action=edit&section=4 "Edit section: Example")] Example

Run this code
    
    
    #include <iomanip>
    #include <iostream>
    #include <sstream>
     
    enum class cap { title, middle, end };
     
    void print(const char* text, double num, cap c)
    {
        if (c == cap::title)
            [std::cout](../cout.html) <<
                "┌──────────┬────────────┬──────────────────────────┐\n"
                "│  number  │   iomanip  │      representation      │\n"
                "├──────────┼────────────┼──────────────────────────┤\n";
        [std::cout](../cout.html) << [std::left](left.html)
             << "│ " << [std::setw](setw.html)(8) << text <<      " │ fixed      │ "
             << [std::setw](setw.html)(24) << std::fixed  << num <<            " │\n"
             << "│ " << [std::setw](setw.html)(8) << text <<      " │ scientific │ "
             << [std::setw](setw.html)(24) << std::scientific << num <<        " │\n"
             << "│ " << [std::setw](setw.html)(8) << text <<      " │ hexfloat   │ "
             << [std::setw](setw.html)(24) << std::hexfloat << num <<          " │\n"
             << "│ " << [std::setw](setw.html)(8) << text <<      " │ default    │ "
             << [std::setw](setw.html)(24) << std::defaultfloat << num <<      " │\n";
        [std::cout](../cout.html) << (c != cap::end ?
                "├──────────┼────────────┼──────────────────────────┤\n" :
                "└──────────┴────────────┴──────────────────────────┘\n");
    }
     
    int main()
    {
        print("0.0", 0.0, cap::title);
        print("0.01", 0.01, cap::middle);
        print("0.00001", 0.00001, cap::end);
     
        // Note; choose clang for correct output
        double f;
        [std::istringstream](../basic_istringstream.html)("0x1.8p+0") >> f;
        [std::cout](../cout.html) << "Parsing 0x1.8p+0 gives " << f << '\n';
     
        [std::istringstream](../basic_istringstream.html)("0x1P-1022") >> f;
        [std::cout](../cout.html) << "Parsing 0x1P-1022 gives " << f << '\n';
    }
