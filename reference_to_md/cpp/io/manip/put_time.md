template< class CharT >  
/*unspecified*/ put_time( const [std::tm](../../chrono/c/tm.html)* tmb, const CharT* fmt ); |  |  (since C++11)  
| |   
  
When used in an expression out << put_time(tmb, fmt), converts the date and time information from a given calendar time tmb to a character string according to [format string](put_time.html#Format_string) fmt, as if by calling [std::strftime](../../chrono/c/strftime.html "cpp/chrono/c/strftime"), [std::wcsftime](../../chrono/c/wcsftime.html "cpp/chrono/c/wcsftime"), or analog (depending on `CharT`), according to the [std::time_put](../../locale/time_put.html "cpp/locale/time put") facet of the locale currently imbued in the output stream out.

fmt  |  \-  |  pointer to a null-terminated `CharT` string specifying the [format of conversion](put_time.html#Format_string)  
  
### Format string

The format string consists of zero or more conversion specifiers and ordinary characters (except `%`). All ordinary characters, including the terminating null character, are copied to the output string without modification. Each conversion specification begins with `%` character, optionally followed by `E` or `O` modifier (ignored if unsupported by the locale), followed by the character that determines the behavior of the specifier. ### Return value

An object of unspecified type such that 

  * if out is an object of type [std::basic_ostream](../basic_ostream.html)<CharT, Traits>, the expression out << put_time(tmb, fmt)
    * has type [std::basic_ostream](../basic_ostream.html)<CharT, Traits>&
    * has value out
    * behaves as if it called f(out, tmb, fmt)



where the function f is defined as: 
    
    
    template<class CharT, class Traits>
    void f([std::basic_ios](../basic_ios.html)<CharT, Traits>& str, const [std::tm](../../chrono/c/tm.html)* tmb, const CharT* fmt)
    {
        using Iter = [std::ostreambuf_iterator](../../iterator/ostreambuf_iterator.html)<CharT, Traits>;
        using TimePut = [std::time_put](../../locale/time_put.html)<CharT, Iter>;
     
        const TimePut& tp = [std::use_facet](../../locale/use_facet.html)<TimePut>(str.getloc());
        const Iter end = tp.put(Iter(str.rdbuf()), str, str.fill(), tmb,
            fmt, fmt + Traits::length(fmt));
     
        if (end.failed())
            str.setstate([std::ios_base::badbit](../ios_base/iostate.html));
    }

### Example

Run this code
    
    
    #include <ctime>
    #include <iomanip>
    #include <iostream>
     
    int main()
    {
        [std::time_t](../../chrono/c/time_t.html) t = [std::time](../../chrono/c/time.html)(nullptr);
        [std::tm](../../chrono/c/tm.html) tm = *[std::localtime](../../chrono/c/localtime.html)(&t);
     
        [std::cout](../cout.html).imbue([std::locale](../../locale/locale.html)("ru_RU.utf8"));
        [std::cout](../cout.html) << "ru_RU: " << std::put_time(&tm, "%c %Z") << '\n';
     
        [std::cout](../cout.html).imbue([std::locale](../../locale/locale.html)("ja_JP.utf8"));
        [std::cout](../cout.html) << "ja_JP: " << std::put_time(&tm, "%c %Z") << '\n';
    }

Possible output: 
    
    
    ru_RU: Ср. 28 дек. 2011 10:21:16 EST
    ja_JP: 2011年12月28日 10時21分16秒 EST
