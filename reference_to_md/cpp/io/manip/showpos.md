| |   
  
Enables or disables the display of the plus sign '+' in non-negative integer output. Has no effect on input. 

1) Enables the `showpos` flag in the stream str as if by calling str.setf([std::ios_base::showpos](../ios_base/fmtflags.html)).

2) Disables the `showpos` flag in the stream str as if by calling str.unsetf([std::ios_base::showpos](../ios_base/fmtflags.html)).

This is an I/O manipulator, it may be called with an expression such as out << std::showpos for any `out` of type [std::basic_ostream](../basic_ostream.html "cpp/io/basic ostream") or with an expression such as in >> std::showpos for any `in` of type [std::basic_istream](../basic_istream.html "cpp/io/basic istream").
