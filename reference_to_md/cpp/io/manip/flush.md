Flushes the output sequence os as if by calling os.flush(). 

This is an output-only I/O manipulator, it may be called with an expression such as out << std::flush for any `out` of type [std::basic_ostream](../basic_ostream.html "cpp/io/basic ostream").

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/io/manip/flush&action=edit&section=1 "Edit section: Notes")] Notes

This manipulator may be used to produce an incomplete line of output immediately, e.g. when displaying output from a long-running process, logging activity of multiple threads or logging activity of a program that may crash unexpectedly. An explicit flush of [std::cout](../cout.html "cpp/io/cout") is also necessary before a call to [std::system](../../utility/program/system.html "cpp/utility/program/system"), if the spawned process performs any screen I/O (a common example is [std::system](../../utility/program/system.html)("pause") on Windows). In most other usual interactive I/O scenarios, [std::endl](endl.html "cpp/io/manip/endl") is redundant when used with [std::cout](../cout.html "cpp/io/cout") because any input from [std::cin](../cin.html "cpp/io/cin"), output to [std::cerr](../cerr.html "cpp/io/cerr"), or program termination forces a call to [std::cout](../cout.html).flush(). 

When a complete line of output needs to be flushed, the [std::endl](endl.html "cpp/io/manip/endl") manipulator may be used. 

When every output operation needs to be flushed, the [std::unitbuf](unitbuf.html "cpp/io/manip/unitbuf") manipulator may be used.

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/io/manip/flush&action=edit&section=4 "Edit section: Example")] Example

Without `std::flush`, the output would be the same, but may not appear in real time.

Run this code
    
    
    #include <chrono>
    #include <iostream>
     
    template<typename Diff>
    void log_progress(Diff d)
    {
        [std::cout](../cout.html) << [std::chrono::duration_cast](../../chrono/duration/duration_cast.html)<[std::chrono::milliseconds](../../chrono/duration.html)>(d) << " ... "
                  << std::flush;
    }
     
    int main()
    {
        volatile int sink = 0;
     
        auto t1 = [std::chrono::high_resolution_clock::now](../../chrono/high_resolution_clock/now.html)();
        for (int j = 0; j < 5; ++j)
        {
            for (int n = 0; n < 10000; ++n)
                for (int m = 0; m < 20000; ++m)
                    sink += m * n; // do some work
            auto now = [std::chrono::high_resolution_clock::now](../../chrono/high_resolution_clock/now.html)();
            log_progress(now - t1);
        }
        [std::cout](../cout.html) << '\n';
    }

Possible output: 
    
    
    567ms ... 1137ms ... 1707ms ... 2269ms ... 2842ms ...
