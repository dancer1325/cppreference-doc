Defined in header `[<iomanip>](../../header/iomanip.html "cpp/header/iomanip")` |  |   
---|---|---  
template< class MoneyT >  
/*unspecified*/ get_money( MoneyT& mon, bool intl = false ); |  |  (since C++11)  
| |   
  
When used in an expression in >> get_money(mon, intl), parses the character input as a monetary value, as specified by the [std::money_get](../../locale/money_get.html "cpp/locale/money get") facet of the locale currently imbued in in, and stores the value in mon. 

The extraction operation in in >> get_money(mon, intl) behaves as a [FormattedInputFunction](../../named_req/FormattedInputFunction.html "cpp/named req/FormattedInputFunction").

### Parameters

mon  |  \-  |  variable where monetary value will be written. Can be either long double or [std::basic_string](../../string/basic_string.html "cpp/string/basic string")  
---|---|---  
intl  |  \-  |  expects to find required international currency strings if true, expects optional currency symbols otherwise   
  
### Return value

An object of unspecified type such that 

  * if in is an object of type [std::basic_istream](../basic_istream.html)<CharT, Traits>, the expression in >> get_money(mon, intl)
    * has type [std::basic_istream](../basic_istream.html)<CharT, Traits>&
    * has value in
    * behaves as if it called f(in, mon, intl)



where the function f is defined as: 
    
    
    template<class CharT, class Traits, class MoneyT>
    void f([std::basic_ios](../basic_ios.html)<CharT, Traits>& str, MoneyT& mon, bool intl)
    {
        using Iter = [std::istreambuf_iterator](../../iterator/istreambuf_iterator.html)<CharT, Traits>;
        using MoneyGet = [std::money_get](../../locale/money_get.html)<CharT, Iter>;
     
        [std::ios_base::iostate](../ios_base/iostate.html) err = [std::ios_base::goodbit](../ios_base/iostate.html);
        const MoneyGet& mg = [std::use_facet](../../locale/use_facet.html)<MoneyGet>(str.getloc());
     
        mg.get(Iter(str.rdbuf()), Iter(), intl, str, err, mon);
     
        if (err != [std::ios_base::goodbit](../ios_base/iostate.html))
            str.setstate(err);
    }

### Example

Run this code
    
    
    #include <iomanip>
    #include <iostream>
    #include <locale>
    #include <sstream>
     
    int main()
    {
        [std::istringstream](../basic_istringstream.html) in("$1,234.56 2.22 USD  3.33");
        long double v1, v2;
        [std::string](../../string/basic_string.html) v3;
     
        in.imbue([std::locale](../../locale/locale.html)("en_US.UTF-8"));
        in >> std::get_money(v1) >> std::get_money(v2) >> std::get_money(v3, true);
     
        if (in)
            [std::cout](../cout.html) << [std::quoted](quoted.html)(in.str()) << " parsed as: "
                      << v1 << ", " << v2 << ", " << v3 << '\n';
        else
            [std::cout](../cout.html) << "Parse failed";
    }

Output: 
    
    
    "$1,234.56 2.22 USD  3.33" parsed as: 123456, 222, 333
