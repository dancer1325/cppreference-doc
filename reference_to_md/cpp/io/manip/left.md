[std::ios_base](../ios_base.html)& right( [std::ios_base](../ios_base.html)& str ); |  (2)  |   
[std::ios_base](../ios_base.html)& internal( [std::ios_base](../ios_base.html)& str ); |  (3)  |   
| |   
  
Modifies the positioning of the fill characters in an output stream. `left` and `right` apply to any type being output, `internal` applies to integer, floating-point, and monetary output. Has no effect on input. 

1) Sets the `adjustfield` of the stream str to `left` as if by calling str.setf([std::ios_base::left](../ios_base/fmtflags.html), [std::ios_base::adjustfield](../ios_base/fmtflags.html)).

2) Sets the `adjustfield` of the stream str to `right` as if by calling str.setf([std::ios_base::right](../ios_base/fmtflags.html), [std::ios_base::adjustfield](../ios_base/fmtflags.html)).

3) Sets the `adjustfield` of the stream str to `internal` as if by calling str.setf([std::ios_base::internal](../ios_base/fmtflags.html), [std::ios_base::adjustfield](../ios_base/fmtflags.html)).

The initial default for standard streams is equivalent to `right`. 

This is an I/O manipulator. It may be called with an expression such as out << std::left for any `out` of type [std::basic_ostream](../basic_ostream.html "cpp/io/basic ostream") or with an expression such as in >> std::left for any `in` of type [std::basic_istream](../basic_istream.html "cpp/io/basic istream").

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/io/manip/left&action=edit&section=3 "Edit section: Example")] Example

Run this code
    
    
    #include <iomanip>
    #include <iostream>
    #include <locale>
     
    int main()
    {
        [std::cout](../cout.html).imbue([std::locale](../../locale/locale.html)("en_US.utf8"));
     
        [std::cout](../cout.html) << "Default positioning:\n" << [std::setfill](setfill.html)('*')
                  << [std::setw](setw.html)(12) << -1.23  << '\n'
                  << [std::setw](setw.html)(12) << [std::hex](hex.html) << [std::showbase](showbase.html) << 42 << '\n'
                  << [std::setw](setw.html)(12) << [std::put_money](put_money.html)(123, true) << "\n\n";
     
        [std::cout](../cout.html) << "Left positioning:\n" << std::left
                  << [std::setw](setw.html)(12) << -1.23  << '\n'
                  << [std::setw](setw.html)(12) << 42 << '\n'
                  << [std::setw](setw.html)(12) << [std::put_money](put_money.html)(123, true) << "\n\n";
     
        [std::cout](../cout.html) << "Internal positioning:\n" << std::internal
                  << [std::setw](setw.html)(12) << -1.23  << '\n'
                  << [std::setw](setw.html)(12) << 42 << '\n'
                  << [std::setw](setw.html)(12) << [std::put_money](put_money.html)(123, true) << "\n\n";
     
        [std::cout](../cout.html) << "Right positioning:\n" << std::right
                  << [std::setw](setw.html)(12) << -1.23  << '\n'
                  << [std::setw](setw.html)(12) << 42 << '\n'
                  << [std::setw](setw.html)(12) << [std::put_money](put_money.html)(123, true) << '\n';
    }

Output: 
    
    
    Default positioning:
    *******-1.23
    ********0x2a
    ***USD *1.23
     
    Left positioning:
    -1.23*******
    0x2a********
    USD *1.23***
     
    Internal positioning:
    -*******1.23
    0x********2a
    USD ****1.23
     
    Right positioning:
    *******-1.23
    ********0x2a
    ***USD *1.23
