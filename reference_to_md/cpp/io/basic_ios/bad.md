bool bad() const; |  |   
---|---|---  
| |   
  
Returns true if non-recoverable error has occurred on the associated stream. Specifically, returns true if [`badbit`](../ios_base/iostate.html "cpp/io/ios base/iostate") is set in rdstate(). 

See [`ios_base::iostate`](../ios_base/iostate.html "cpp/io/ios base/iostate") for the list of conditions that set `badbit`. 

## Contents

  * [1 Parameters](bad.html#Parameters)
  * [2 Return value](bad.html#Return_value)
  * [3 Example](bad.html#Example)
  * [4 See also](bad.html#See_also)

  
---  
  
### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/io/basic_ios/bad&action=edit&section=1 "Edit section: Parameters")] Parameters

(none) 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/io/basic_ios/bad&action=edit&section=2 "Edit section: Return value")] Return value

true if a non-recoverable error has occurred, false otherwise. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/io/basic_ios/bad&action=edit&section=3 "Edit section: Example")] Example

Run this code
    
    
    #include <cstdlib>
    #include <fstream>
    #include <iostream>
     
    int main()
    {
        [std::ifstream](../basic_ifstream.html) file("test.txt");
        if (!file) // operator! is used here
        {  
            [std::cout](../cout.html) << "File opening failed\n";
            return [EXIT_FAILURE](../../utility/program/EXIT_status.html);
        }
     
        // typical C++ I/O loop uses the return value of the I/O function
        // as the loop controlling condition, operator bool() is used here
        for (int n; file >> n;)
           [std::cout](../cout.html) << n << ' ';
        [std::cout](../cout.html) << '\n';
     
        if (file.bad())
            [std::cout](../cout.html) << "I/O error while reading\n";
        else if (file.eof())
            [std::cout](../cout.html) << "End of file reached successfully\n";
        else if (file.fail())
            [std::cout](../cout.html) << "Non-integer data encountered\n";
    }
