streamsize width() const; |  (1)  |   
---|---|---  
streamsize width( streamsize new_width ); |  (2)  |   
| |   
  
Manages the minimum number of characters to generate on certain output operations and the maximum number of characters to generate on certain input operations. 

1) Returns the current field width.

2) Sets the field width to the given one. Returns the previous field width.

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/io/ios_base/width&action=edit&section=2 "Edit section: Return value")] Return value

The field width before the call to the function. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/io/ios_base/width&action=edit&section=3 "Edit section: Notes")] Notes

Some I/O functions call width(0) before returning, see [std::setw](../manip/setw.html "cpp/io/manip/setw") (this results in this field having effect on the next I/O function only, and not on any subsequent I/O). 

The exact effects this modifier has on the input and output vary between the individual I/O functions and are described at each `operator<<` and `operator>>` overload page individually. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/io/ios_base/width&action=edit&section=4 "Edit section: Example")] Example

Run this code
    
    
    #include <algorithm>
    #include <array>
    #include <iomanip>
    #include <iostream>
    #include <span>
    #include <string_view>
    using namespace std::string_view_literals;
     
    constexpr [std::array](../../container/array.html) table_header =
    {
        "Language"sv, "Author"sv, "Birthdate"sv, "RIP date"sv
    };
     
    using row_t = [std::array](../../container/array.html)<[std::string_view](../../string/basic_string_view.html), table_header.size()>;
    using widths_t = [std::array](../../container/array.html)<[std::size_t](../../types/size_t.html), table_header.size()>;
     
    constexpr [std::array](../../container/array.html) table_body = [std::to_array](../../container/array/to_array.html)<row_t>
    ({
        {"C", "Dennis Ritchie", "1941-09-09", "2011-10-12"},
        {"C++", "Bjarne Stroustrup", "1950-12-30"},
        {"C#", "Anders Hejlsberg", "1960-12-02"},
        {"Python", "Guido van Rossum", "1956-01-31"},
        {"Javascript", "Brendan Eich", "1961-07-04"}
    });
     
    constexpr widths_t calculate_column_widths([std::span](../../container/span.html)<const row_t> table)
    {
        widths_t widths{};
        for (const row_t& row : table)
            for (size_t i = 0; i != row.size(); ++i)
                widths[i] = [std::max](../../algorithm/max.html)(widths[i], row[i].size());
        return widths;
    }
     
    void print_row(const row_t& row, const widths_t& widths)
    {
        [std::cout](../cout.html) << '|';
        for (size_t i = 0; i != row.size(); ++i)
        {
            [std::cout](../cout.html) << ' ';
            [std::cout](../cout.html).width(widths[i]);
            [std::cout](../cout.html) << row[i] << " |";
        }
        [std::cout](../cout.html) << '\n';
    };
     
    void print_break(const widths_t& widths)
    {
        const [std::size_t](../../types/size_t.html) margin = 1;
        [std::cout](../cout.html).put('+').fill('-');
        for ([std::size_t](../../types/size_t.html) w : widths)
        {
            [std::cout](../cout.html).width(w + margin * 2);
            [std::cout](../cout.html) << '-' << '+';
        }
        [std::cout](../cout.html).put('\n').fill(' ');
    };
     
    int main()
    {
        constexpr widths_t widths = calculate_column_widths(table_body);
     
        [std::cout](../cout.html).setf(std::ios::left, std::ios::adjustfield);
        print_break(widths);
        print_row(table_header, widths);
        print_break(widths);
        for (const row_t& row : table_body)
            print_row(row, widths);
        print_break(widths);
    }

    | C#         | Anders Hejlsberg  | 1960-12-02 |            |
    | Python     | Guido van Rossum  | 1956-01-31 |            |
    | Javascript | Brendan Eich      | 1961-07-04 |            |
