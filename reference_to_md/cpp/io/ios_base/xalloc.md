static int xalloc(); |  |   
---|---|---  
| |   
  
Returns a unique (program-wide) index value that can be used to access one long and one void* elements in the private storage of `std::ios_base` by calling [iword()](iword.html "cpp/io/ios base/iword") and [pword()](pword.html "cpp/io/ios base/pword"). The call to `xalloc` does not allocate memory. 

This function is thread-safe: concurrent access by multiple threads does not result in a data race.  | (since C++11)  
---|---  
  
Effectively increments the next available unique index.

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/io/ios_base/xalloc&action=edit&section=2 "Edit section: Example")] Example

[[edit]](https://en.cppreference.com/mwiki/index.php?title=Template:cpp/io/ios_base/xalloc_example&action=edit)

Uses base class pword storage for runtime type identification of derived stream objects.

Run this code
    
    
    #include <iostream>
     
    template<class CharT, class Traits = [std::char_traits](../../string/char_traits.html)<CharT>>
    class mystream : public [std::basic_ostream](../basic_ostream.html)<CharT, Traits>
    {
    public:
        static const int xindex;
     
        mystream([std::basic_ostream](../basic_ostream.html)<CharT, Traits>& ostr) :
            [std::basic_ostream](../basic_ostream.html)<CharT, Traits>(ostr.rdbuf())
        {
            this->pword(xindex) = this;
        }
     
        void myfn()
        {
            *this << "[special handling for mystream]";
        }
    };
     
    // Each specialization of mystream obtains a unique index from xalloc()
    template<class CharT, class Traits>
    const int mystream<CharT, Traits>::xindex = std::ios_base::xalloc();
     
    // This I/O manipulator will be able to recognize ostreams that are mystreams
    // by looking up the pointer stored in pword
    template<class CharT, class Traits>
    [std::basic_ostream](../basic_ostream.html)<CharT, Traits>& mymanip([std::basic_ostream](../basic_ostream.html)<CharT, Traits>& os)
    {
        if (os.pword(mystream<CharT, Traits>::xindex) == &os)
            static_cast<mystream<CharT, Traits>&>(os).myfn();
        return os;
    }
     
    int main()
    {
        [std::cout](../cout.html) << "cout, narrow-character test " << mymanip << '\n';
     
        mystream<char> myout([std::cout](../cout.html));
        myout << "myout, narrow-character test " << mymanip << '\n';
     
        [std::wcout](../cout.html) << "wcout, wide-character test " << mymanip << '\n';
     
        mystream<wchar_t> mywout([std::wcout](../cout.html));
        mywout << "mywout, wide-character test " << mymanip << '\n';
    }

Output: 
    
    
    cout, narrow-character test
    myout, narrow-character test [special handling for mystream]
    wcout, wide-character test
    mywout, wide-character test [special handling for mystream]
