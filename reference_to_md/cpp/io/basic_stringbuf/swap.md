void swap( basic_stringbuf& rhs ) noexcept(/* see below */); |  |  (since C++20)  
| |   
  
Swaps the state and the contents of *this and rhs. 

The behavior is undefined if `Allocator` does not propagate on swap and the allocators of *this and other are unequal.  | (since C++11)

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/io/basic_stringbuf/swap&action=edit&section=4 "Edit section: Notes")] Notes

This function is called automatically when swapping [std::stringstream](../basic_stringstream.html "cpp/io/basic stringstream") objects. It is rarely necessary to call it directly. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/io/basic_stringbuf/swap&action=edit&section=5 "Edit section: Example")] Example

Run this code
    
    
    #include <iomanip>
    #include <iostream>
    #include <sstream>
    #include <string>
     
    int main()
    {
        [std::istringstream](../basic_istringstream.html) one("one");
        [std::ostringstream](../basic_ostringstream.html) two("two");
     
        [std::cout](../cout.html) << "Before swap: one = " << [std::quoted](../manip/quoted.html)(one.str())
                  << ", two = " << [std::quoted](../manip/quoted.html)(two.str()) << ".\n";
     
        one.rdbuf()->swap(*two.rdbuf());
     
        [std::cout](../cout.html) << "After  swap: one = " << [std::quoted](../manip/quoted.html)(one.str())
                  << ", two = " << [std::quoted](../manip/quoted.html)(two.str()) << ".\n";
    }

Output: 
    
    
    Before swap: one = "one", two = "two".
    After  swap: one = "two", two = "one".
