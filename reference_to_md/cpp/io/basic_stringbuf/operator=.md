[std::basic_stringbuf](../basic_stringbuf.html)& operator=( [std::basic_stringbuf](../basic_stringbuf.html)&& rhs ); |  (1)  |  (since C++11)  
---|---|---  
[std::basic_stringbuf](../basic_stringbuf.html)& operator=( const [std::basic_stringbuf](../basic_stringbuf.html)& rhs ) = delete; |  (2)  |   
| |   
  
1) Move assignment operator: Moves the contents of rhs into *this. After the move, *this has the associated string, the open mode, the locale, and all other state formerly held by rhs. The six pointers of [std::basic_streambuf](../basic_streambuf.html "cpp/io/basic streambuf") in *this are guaranteed to be different from the corresponding pointers in the moved-from rhs unless null.

2) The copy assignment operator is deleted; `basic_stringbuf` is not [CopyAssignable](../../named_req/CopyAssignable.html "cpp/named req/CopyAssignable").

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/io/basic_stringbuf/operator%3D&action=edit&section=3 "Edit section: Example")] Example

Run this code
    
    
    #include <iostream>
    #include <sstream>
    #include <string>
     
    int main()
    {
        [std::istringstream](../basic_istringstream.html) one("one");
        [std::ostringstream](../basic_ostringstream.html) two("two");
     
        [std::cout](../cout.html) << "Before move, one = \"" << one.str() << '"'
                  << " two = \"" << two.str() << "\"\n";
     
        *one.rdbuf() = std::move(*two.rdbuf());
     
        [std::cout](../cout.html) << "After move, one = \"" << one.str() << '"'
                  << " two = \"" << two.str() << "\"\n";
    }

Output: 
    
    
    Before move, one = "one" two = "two"
    After move, one = "two" two = ""
