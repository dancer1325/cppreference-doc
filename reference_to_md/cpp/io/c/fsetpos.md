int fsetpos( [std::FILE](FILE.html)* stream, const [std::fpos_t](fpos_t.html)* pos ); |  |   
| |   
  
Sets the file position indicator and the multibyte parsing state (if any) for the C file stream `stream` according to the value pointed to by `pos`. 

Besides establishing new parse state and position, a call to this function undoes the effects of [std::ungetc](ungetc.html "cpp/io/c/ungetc") and clears the end-of-file state, if it is set. 

If a read or write error occurs, the error indicator ([std::ferror](ferror.html "cpp/io/c/ferror")) for the stream is set.

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/io/c/fsetpos&action=edit&section=1 "Edit section: Parameters")] Parameters

stream  |  \-  |  file stream to modify   
---|---|---  
pos  |  \-  |  pointer to a fpos_t object obtained from [std::fgetpos](fgetpos.html "cpp/io/c/fgetpos") called on a stream associated with the same file   
  
### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/io/c/fsetpos&action=edit&section=2 "Edit section: Return value")] Return value

​0​ upon success, nonzero value otherwise. Also, sets [errno](../../error/errno.html "cpp/error/errno") on failure. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/io/c/fsetpos&action=edit&section=3 "Edit section: Notes")] Notes

After seeking to a non-end position in a wide stream, the next call to any output function may render the remainder of the file undefined, e.g. by outputting a multibyte sequence of a different length. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/io/c/fsetpos&action=edit&section=4 "Edit section: Example")] Example

Run this code
    
    
    #include <cstdio>
    #include <cstdlib>
     
    int main()
    {
        // Prepare an array of floating-point values.
        const int SIZE = 5;
        double A[SIZE] = {1., 2., 3., 4., 5.};
        // Write array to a file.
        [std::FILE](FILE.html) * fp = [std::fopen](fopen.html)("test.bin", "wb");
        [std::fwrite](fwrite.html)(A, sizeof(double), SIZE, fp);
        [std::fclose](fclose.html)(fp);
     
        // Read the values into array B.
        double B[SIZE];
        fp = [std::fopen](fopen.html)("test.bin", "rb");
        [std::fpos_t](fpos_t.html) pos;
        if ([std::fgetpos](fgetpos.html)(fp, &pos) != 0)      // current position: start of file
        {
           [std::perror](perror.html)("fgetpos()");
           [std::fprintf](fprintf.html)(stderr, "fgetpos() failed in file %s at line # %d\n",
                        __FILE__, __LINE__-3);
           [std::exit](../../utility/program/exit.html)([EXIT_FAILURE](../../utility/program/EXIT_status.html));
        }
     
        int ret_code = [std::fread](fread.html)(B, sizeof(double), 1, fp);      // read one value
        // current position: after reading one value
        [std::printf](fprintf.html)("%.1f; read count = %d\n", B[0], ret_code);   // print one value and ret_code
     
        if (std::fsetpos(fp, &pos) != 0)   // reset current position to start of file
        {
           if ([std::ferror](ferror.html)(fp))
           {
              [std::perror](perror.html)("fsetpos()");
              [std::fprintf](fprintf.html)(stderr, "fsetpos() failed in file %s at line # %d\n",
                           __FILE__, __LINE__-5);
              [std::exit](../../utility/program/exit.html)([EXIT_FAILURE](../../utility/program/EXIT_status.html));
           }
        }
     
        ret_code = [std::fread](fread.html)(B, sizeof(double), 1, fp);         // re-read first value
        [std::printf](fprintf.html)("%.1f; read count = %d\n", B[0], ret_code);  // print one value and ret_code
        [std::fclose](fclose.html)(fp);
     
        return [EXIT_SUCCESS](../../utility/program/EXIT_status.html); 
    }

Output: 
    
    
    1.0; read count = 1
    1.0; read count = 1
