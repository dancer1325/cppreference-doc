Returns an iterator past the last element of *this. 

This returned iterator only acts as a sentinel. It is not guaranteed to be [dereferenceable](../../iterator.html#Dereferenceability_and_validity "cpp/iterator").

### Return value

Iterator past the last element. 

### Complexity

Constant. 

### Example

Run this code
    
    
    #include <algorithm>
    #include <cassert>
    #include <iostream>
    #include <string>
    #include <utility>
    #include <unordered_map>
     
    int main()
    {
        auto show_node = [](const [std::pair](../../utility/pair.html)<[std::string](../../string/basic_string.html), [std::string](../../string/basic_string.html)>& node)
        {
            [std::cout](../../io/cout.html) << node.first << " : " << node.second << '\n';
        };
     
        [std::unordered_multimap](../unordered_multimap.html)<[std::string](../../string/basic_string.html), [std::string](../../string/basic_string.html)> lemmas;
        [assert](../../error/assert.html)(lemmas.begin() == lemmas.end());
        [assert](../../error/assert.html)(lemmas.cbegin() == lemmas.cend());
     
        lemmas.insert({ "1. ∀x ∈ N ∃y ∈ N", "x ≤ y" });
        show_node(*lemmas.cbegin());
        [assert](../../error/assert.html)(lemmas.begin() != lemmas.end());
        [assert](../../error/assert.html)(lemmas.cbegin() != lemmas.cend());
     
        lemmas.begin()->second = "x < y";
        show_node(*lemmas.cbegin());
     
        lemmas.insert({ "2. ∀x, y ∈ N    ", "x = y V x ≠ y" });
        show_node(*lemmas.cbegin());
     
        lemmas.insert({ "3. ∀x ∈ N ∃y ∈ N", "y = x + 1" });
        show_node(*lemmas.cbegin());
     
        [std::cout](../../io/cout.html) << "Lemmas: \n";
        [std::for_each](../../algorithm/for_each.html)(lemmas.cbegin(), lemmas.cend(), [&](const auto& n)
        {
            show_node(n);
        });
        [std::cout](../../io/cout.html) << '\n';
    }

Possible output: 
    
    
    1. ∀x ∈ N ∃y ∈ N : x ≤ y
    1. ∀x ∈ N ∃y ∈ N : x < y
    2. ∀x, y ∈ N     : x = y V x ≠ y
    3. ∀x ∈ N ∃y ∈ N : y = x + 1
    Lemmas: 
    3. ∀x ∈ N ∃y ∈ N : y = x + 1
    1. ∀x ∈ N ∃y ∈ N : x < y
    2. ∀x, y ∈ N     : x = y V x ≠ y
