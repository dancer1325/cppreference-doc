Constructs a new `inplace_vector` from a variety of data sources. 

1) Constructs an empty `inplace_vector` whose data() == nullptr and size() == 0.

2) Constructs an `inplace_vector` with count default-inserted elements.

3) Constructs an `inplace_vector` with count copies of elements with value value.

4) Constructs an `inplace_vector` with the contents of the range `[`first`, `last`)`.

5) Constructs an `inplace_vector` with the contents of the range rg.

6) A [copy constructor](../../language/copy_constructor.html "cpp/language/copy constructor"). Constructs an `inplace_vector` with the copy of the contents of other.

The constructor is a [trivial copy constructor](../../language/copy_constructor.html#Trivial_copy_constructor "cpp/language/copy constructor") if N > 0 and [std::is_trivially_copy_constructible_v](../../types/is_copy_constructible.html)<T> are both true.

7) A [move constructor](../../language/move_constructor.html "cpp/language/move constructor"). Constructs an `inplace_vector` with the contents of other using move semantics.

The constructor is a [trivial move constructor](../../language/move_constructor.html#Trivial_move_constructor "cpp/language/move constructor") if N > 0 and [std::is_trivially_move_constructible_v](../../types/is_move_constructible.html)<T> are both true.

8) Constructs an `inplace_vector` with the contents of the initializer list init.

### Parameters

count  |  \-  |  the size of the container   
---|---|---  
value  |  \-  |  the value to initialize elements of the container with   
first, last  |  \-  |  the pair of iterators defining the source [range](../../iterator.html#Ranges "cpp/iterator") of elements to copy   
rg  |  \-  |  the range of values to initialize elements of the container with   
other  |  \-  |  another `inplace_vector` to be used as source to initialize the elements of the container with   
init  |  \-  |  initializer list to initialize the elements of the container with   
Type requirements   
-`T` must meet the requirements of [DefaultInsertable](../../named_req/DefaultInsertable.html "cpp/named req/DefaultInsertable") in order to use overloads (2,3).   
  
### Complexity

1) Constant.

2,3) Linear in count.

4) Linear in [std::distance](../../iterator/distance.html)(first, last).

5) Linear in std::[ranges::distance](../../iterator/ranges/distance.html)(rg).

6,7) Linear in size of other.

8) Linear in size of init.

### Exceptions

2,3) Throws [std::bad_alloc](../../memory/new/bad_alloc.html "cpp/memory/new/bad alloc") if count > N.

5) Throws [std::bad_alloc](../../memory/new/bad_alloc.html "cpp/memory/new/bad alloc") if std::[ranges::size](../../ranges/size.html)(rg) > N.

8) Throws [std::bad_alloc](../../memory/new/bad_alloc.html "cpp/memory/new/bad alloc") if init.size() > N.

### Example

Run this code
    
    
    #include <cassert>
    #include <initializer_list>
    #include <inplace_vector>
    #include <new>
    #include <print>
    #include <ranges>
     
    int main()
    {
        [std::inplace_vector](../inplace_vector.html)<int, 4> v1; // overload (1)
        [assert](../../error/assert.html)(v1.size() == 0 && v1.capacity() == 4);
     
        [std::inplace_vector](../inplace_vector.html)<int, 0> v2; // overload (1), N == 0 is allowed
        [assert](../../error/assert.html)(v2.size() == 0 && v2.capacity() == 0);
     
        [std::inplace_vector](../inplace_vector.html)<int, 5> v3(3); // overload (2)
        [assert](../../error/assert.html)(v3.size() == 3 && v3.capacity() == 5);
        [std::println](../../io/println.html)("v3 = {}", v3);
     
        try
        {
            [std::inplace_vector](../inplace_vector.html)<int, 3> v(4); // overload (2), throws: count > N
        }
        catch(const [std::bad_alloc](../../memory/new/bad_alloc.html)& ex1)
        {
            [std::println](../../io/println.html)("ex1.what(): {}", ex1.what());
        }
     
        [std::inplace_vector](../inplace_vector.html)<int, 5> v4(3, 8); // overload (3)
        [assert](../../error/assert.html)(v4.size() == 3 && v4.capacity() == 5);
        [std::println](../../io/println.html)("v4 = {}", v4);
     
        try
        {
            [std::inplace_vector](../inplace_vector.html)<int, 3> v(4, 2); // overload (3), throws: count > N
        }
        catch(const [std::bad_alloc](../../memory/new/bad_alloc.html)& ex2)
        {
            [std::println](../../io/println.html)("ex2.what(): {}", ex2.what());
        }
     
        const auto init = {1, 2, 3};
     
        [std::inplace_vector](../inplace_vector.html)<int, 4> v5(init.begin(), init.end()); // overload (4)
        [assert](../../error/assert.html)(v5.size() == 3 && v5.capacity() == 4);
        [std::println](../../io/println.html)("v5 = {}", v5);
     
        [std::inplace_vector](../inplace_vector.html)<int, 4> v6([std::from_range](../../ranges/from_range.html), init); // overload (5)
        [assert](../../error/assert.html)(v6.size() == 3 && v6.capacity() == 4);
        [std::println](../../io/println.html)("v6 = {}", v6);
     
        [std::inplace_vector](../inplace_vector.html)<int, 4> v7(v6); // overload (6)
        [assert](../../error/assert.html)(v7.size() == 3 && v7.capacity() == 4);
        [std::println](../../io/println.html)("v7 = {}", v7);
        [assert](../../error/assert.html)(v6.size() == 3);
     
        [std::inplace_vector](../inplace_vector.html)<int, 4> v8(std::move(v6)); // overload (7)
        // Note that after the move v6 is left in valid but indeterminate state.
        [assert](../../error/assert.html)(v8.size() == 3 && v8.capacity() == 4);
        [std::println](../../io/println.html)("v8 = {}", v8);
     
        [std::inplace_vector](../inplace_vector.html)<int, 4> v9(init); // overload (8)
        [assert](../../error/assert.html)(v9.size() == 3 && v9.capacity() == 4);
        [std::println](../../io/println.html)("v9 = {}", v9);
     
        try
        {
            [std::inplace_vector](../inplace_vector.html)<int, 2> v(init); // overload (8), throws: init.size() > N
        }
        catch(const [std::bad_alloc](../../memory/new/bad_alloc.html)& ex3)
        {
            [std::println](../../io/println.html)("ex3.what(): {}", ex3.what());
        }
    }

Possible output: 
    
    
    v3 = [0, 0, 0]
    ex1.what(): std::bad_alloc
    v4 = [42, 42, 42]
    ex2.what(): std::bad_alloc
    v5 = [1, 2, 3]
    v6 = [1, 2, 3]
    v7 = [1, 2, 3]
    v8 = [1, 2, 3]
    v9 = [1, 2, 3]
    ex3.what(): std::bad_alloc
