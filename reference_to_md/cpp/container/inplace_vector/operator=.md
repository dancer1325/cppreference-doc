Replaces the contents of the `inplace_vector`. 

1) [Copy assignment operator](../../language/as_operator.html#Copy_assignment_operator "cpp/language/copy assignment"). Also a [trivial copy assignment operator](../../language/as_operator.html#Trivial_copy_assignment_operator "cpp/language/copy assignment"), if [std::inplace_vector](../inplace_vector.html)<T, N> has a [trivial destructor](../../language/destructor.html#Trivial_destructor "cpp/language/destructor"), and [std::is_trivially_copy_constructible_v](../../types/is_copy_constructible.html)<T> && [std::is_trivially_copy_assignable_v](../../types/is_copy_assignable.html)<T> is true. Replaces the contents with a copy of the contents of other.

2) [Move assignment operator](../../language/move_operator.html#Move_assignment_operator "cpp/language/move assignment"). Also a [trivial move assignment operator](../../language/move_operator.html#Trivial_move_assignment_operator "cpp/language/move assignment"), if [std::inplace_vector](../inplace_vector.html)<T, N> has a [trivial destructor](../../language/destructor.html#Trivial_destructor "cpp/language/destructor"), and [std::is_trivially_move_constructible_v](../../types/is_move_constructible.html)<T> && [std::is_trivially_move_assignable_v](../../types/is_move_assignable.html)<T> is true. Replaces the contents with those of other using move semantics (i.e., the data in other is moved from other into this container). other is in a valid but unspecified state afterwards.

3) Replaces the contents with those identified by initializer list init.

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/container/inplace_vector/operator%3D&action=edit&section=1 "Edit section: Parameters")] Parameters

other  |  \-  |  another `inplace_vector` to be used as source to initialize the elements of the container with   
---|---|---  
init  |  \-  |  initializer list to initialize the elements of the container with   
  
### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/container/inplace_vector/operator%3D&action=edit&section=2 "Edit section: Complexity")] Complexity

1,2) Linear in the size of *this and other.

3) Linear in the size of *this and init.

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/container/inplace_vector/operator%3D&action=edit&section=3 "Edit section: Exceptions")] Exceptions

2)

[`noexcept`](../../language/noexcept_spec.html "cpp/language/noexcept spec") specification: 

noexcept(N == 0 ||  


([std::is_nothrow_move_assignable_v](../../types/is_move_assignable.html)<T> &&  


[std::is_nothrow_move_constructible_v](../../types/is_move_constructible.html)<T>))

3) Throws [std::bad_alloc](../../memory/new/bad_alloc.html "cpp/memory/new/bad alloc") if init.size() > N.

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/container/inplace_vector/operator%3D&action=edit&section=4 "Edit section: Example")] Example

Run this code
    
    
    #include <initializer_list>
    #include <inplace_vector>
    #include <new>
    #include <print>
    #include <ranges>
    #include <string>
    ¬†
    int main()
    {
        [std::inplace_vector](../inplace_vector.html)<int, 4> x({1, 2, 3}), y;
        [std::println](../../io/println.html)("Initially:");
        [std::println](../../io/println.html)("x = {}", x);
        [std::println](../../io/println.html)("y = {}", y);
    ¬†
        [std::println](../../io/println.html)("Copy assignment copies data from x to y:");
        y = x; // overload (1)
        [std::println](../../io/println.html)("x = {}", x);
        [std::println](../../io/println.html)("y = {}", y);
    ¬†
        [std::inplace_vector](../inplace_vector.html)<[std::string](../../string/basic_string.html), 3> z, w{"\N{CAT}", "\N{GREEN HEART}"};
        [std::println](../../io/println.html)("Initially:");
        [std::println](../../io/println.html)("z = {}", z);
        [std::println](../../io/println.html)("w = {}", w);
    ¬†
        [std::println](../../io/println.html)("Move assignment moves data from w to z:");
        z = std::move(w); // overload (2)
        [std::println](../../io/println.html)("z = {}", z);
        [std::println](../../io/println.html)("w = {}", w); // w is in valid but unspecified state
    ¬†
        auto l = {4, 5, 6, 7};
        [std::println](../../io/println.html)("Assignment of initializer_list {} to x:", l);
        x = l; // overload (3)
        [std::println](../../io/println.html)("x = {}", x);
    ¬†
        [std::println](../../io/println.html)("Assignment of initializer_list with size bigger than N throws:");
        try
        {
            x = {1, 2, 3, 4, 5}; // throws: (initializer list size == 5) > (capacity N == 4)
        }
        catch(const [std::bad_alloc](../../memory/new/bad_alloc.html)& ex)
        {
            [std::println](../../io/println.html)("ex.what(): {}", ex.what());
        }
    }
