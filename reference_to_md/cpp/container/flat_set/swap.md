void swap( flat_set& other ) noexcept; |  |  (since C++23)  
---|---|---  
| |   
Exchanges the contents of the container adaptor with those of `other`. Effectively calls 
    
    
    [ranges::swap](http://en.cppreference.com/w/cpp/ranges-utility-placeholder/swap)(compare, other.compare);
    [ranges::swap](http://en.cppreference.com/w/cpp/ranges-utility-placeholder/swap)(c, other.c);
