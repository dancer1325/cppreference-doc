value_compare value_comp() const; |  | (constexpr since C++26)  
---|---|---  
| |   
  
Returns a function object that compares objects of type `value_type` (key-value pairs) by using [key_comp()](key_comp.html "cpp/container/map/key comp") to compare the first components of the pairs. 

## Contents

  * [2 Complexity](value_comp.html#Complexity)
  * [3 Example](value_comp.html#Example)
  * [4 See also](value_comp.html#See_also)

  
---  
  
### [[edit](https://en.cppreference.com/mwiki/index.php?title=Template:cpp/container/value_comp&action=edit&section=T-1 "Template:cpp/container/value comp")] Return value

The value comparison function object. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=Template:cpp/container/value_comp&action=edit&section=T-2 "Template:cpp/container/value comp")] Complexity

Constant. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=Template:cpp/container/value_comp&action=edit&section=T-3 "Template:cpp/container/value comp")] Example

Run this code
    
    
    #include <iostream>
    #include <map>
    #include <utility>
     
    // Example module 97 key compare function
    struct ModCmp
    {
        bool operator()(int lhs, int rhs) const
        {
            return (lhs % 97) < (rhs % 97);
        }
    };
     
    int main()
    {
        [std::map](../map.html)<int, char, ModCmp> cont;
        cont = {{1, 'a'}, {2, 'b'}, {3, 'c'}, {4, 'd'}, {5, 'e'}};
     
        auto comp_func = cont.value_comp();
     
        for (const [std::pair](../../utility/pair.html)<int, char> val = {100, 'a'}; auto it : cont)
        {
            const bool before = comp_func(it, val);
            const bool after = comp_func(val, it);
     
            [std::cout](../../io/cout.html) << '(' << it.first << ',' << it.second << ") ";
            if (!before && !after)
                [std::cout](../../io/cout.html) << "equivalent to key (" << val.first << ")\n";
            else if (before)
                [std::cout](../../io/cout.html) << "goes before key (" << val.first << ")\n";
            else if (after)
                [std::cout](../../io/cout.html) << "goes after key (" << val.first << ")\n";
            else
                [std::unreachable](../../utility/unreachable.html)();
        }
    }

Output: 
    
    
    (1,a) goes before key (100)
    (2,b) goes before key (100)
    (3,c) equivalent to key (100)
    (4,d) goes after key (100)
    (5,e) goes after key (100)
