| |   
  
1,4) If a key equivalent to k already exists in the container, assigns [std::forward](../../utility/forward.html)<M>(obj) to the [`mapped_type`](../map.html#Member_types "cpp/container/map") corresponding to the key k. If the key does not exist, inserts the new value as if by [`insert`](insert.html "cpp/container/map/insert"), constructing it from value_type(k, [std::forward](../../utility/forward.html)<M>(obj)).

2,5) Same as (1,4), except the mapped value is constructed from value_type(std::move(k), [std::forward](../../utility/forward.html)<M>(obj)).

3,6) If a key equivalent to k already exists in the container, assigns [std::forward](../../utility/forward.html)<M>(obj) to the [`mapped_type`](../map.html#Member_types "cpp/container/map") corresponding to the key k. If the key does not exist, constructs an object `u` of `value_type` with [std::forward](../../utility/forward.html)<K>(k), [std::forward](../../utility/forward.html)<M>(obj)), then inserts `u` into *this. If equal_range(u.first) == equal_range(k) is false, the behavior is undefined. The `value_type` must be [EmplaceConstructible](../../named_req/EmplaceConstructible.html "cpp/named req/EmplaceConstructible") into `map` from [std::forward](../../utility/forward.html)<K>(k), [std::forward](../../utility/forward.html)<M>(obj). This overload participates in overload resolution only if `Compare` is [transparent](../../functional.html#Transparent_function_objects "cpp/utility/functional"). It allows calling this function without constructing an instance of `Key`.

The behavior is undefined(until C++20)The program is ill-formed(since C++20) if [std::is_assignable_v](../../types/is_assignable.html)<mapped_type&, M&&> is false. 

No iterators or references are invalidated.

### [[edit](https://en.cppreference.com/mwiki/index.php?title=Template:cpp/container/insert_or_assign_assoc&action=edit&section=T-1 "Template:cpp/container/insert or assign assoc")] Parameters

k  |  \-  |  the key used both to look up and to insert if not found   
---|---|---  
hint  |  \-  |  iterator to the position before which the new element will be inserted   
obj  |  \-  |  the value to insert or assign   
  
### [[edit](https://en.cppreference.com/mwiki/index.php?title=Template:cpp/container/insert_or_assign_assoc&action=edit&section=T-2 "Template:cpp/container/insert or assign assoc")] Return value

1-3) The bool component is true if the insertion took place and false if the assignment took place. The iterator component is pointing at the element that was inserted or updated.

4-6) Iterator pointing at the element that was inserted or updated.

### [[edit](https://en.cppreference.com/mwiki/index.php?title=Template:cpp/container/insert_or_assign_assoc&action=edit&section=T-3 "Template:cpp/container/insert or assign assoc")] Complexity

1-3) Same as for [`emplace`](emplace.html "cpp/container/map/emplace").

4-6) Same as for [`emplace_hint`](emplace_hint.html "cpp/container/map/emplace hint").

### [[edit](https://en.cppreference.com/mwiki/index.php?title=Template:cpp/container/insert_or_assign_assoc&action=edit&section=T-4 "Template:cpp/container/insert or assign assoc")] Notes

`insert_or_assign` returns more information than [`operator`](operator_at.html "cpp/container/map/operator at")[] and does not require default-constructibility of the mapped type.

### [[edit](https://en.cppreference.com/mwiki/index.php?title=Template:cpp/container/insert_or_assign_assoc&action=edit&section=T-5 "Template:cpp/container/insert or assign assoc")] Example

Run this code
    
    
    #include <iostream>
    #include <string>
    #include <map>
     
    void print_node(const auto& node)
    {
        [std::cout](../../io/cout.html) << '[' << node.first << "] = " << node.second << '\n';
    }
     
    void print_result(auto const& pair)
    {
        [std::cout](../../io/cout.html) << (pair.second ? "inserted: " : "assigned: ");
        print_node(*pair.first);
    }
     
    int main()
    {
        [std::map](../map.html)<[std::string](../../string/basic_string.html), [std::string](../../string/basic_string.html)> myMap;
     
        print_result(myMap.insert_or_assign("a", "apple"));
        print_result(myMap.insert_or_assign("b", "banana"));
        print_result(myMap.insert_or_assign("c", "cherry"));
        print_result(myMap.insert_or_assign("c", "clementine"));
     
        for (const auto& node : myMap)
            print_node(node);
    }

Output: 
    
    
    inserted: [a] = apple
    inserted: [b] = banana
    inserted: [c] = cherry
    assigned: [c] = clementine
    [a] = apple
    [b] = banana
    [c] = clementine
