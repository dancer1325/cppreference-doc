Returns a range containing all elements with the given key in the container. The range is defined by two iterators, one pointing to the first element that is **not** less than the given key and another pointing to the first element greater than the given key. 

Alternatively, the first iterator may be obtained with [lower_bound()](lower_bound.html "cpp/container/set/lower bound"), and the second with [upper_bound()](upper_bound.html "cpp/container/set/upper bound"). 

1,2) Compares the keys to key.

3,4) Compares the keys to the value x.

This overload participates in overload resolution only if `Compare` is [transparent](../../functional.html#Transparent_function_objects "cpp/utility/functional"). It allows calling this function without constructing an instance of `Key`.

### Parameters

key  |  \-  |  key value to compare the elements to   
---|---|---  
x  |  \-  |  alternative value that can be compared to `Key`  
  
### Return value

[std::pair](../../utility/pair.html "cpp/utility/pair") containing a pair of iterators defining the wanted range: 

  * The first iterator points to the first element not less than the given key, or end() if no such element exists. 
  * The second iterator points to the first element greater than the given key, or end() if no such element exists. 



### Complexity

Logarithmic in the size of the container.

### Example

Run this code
    
    
    #include <set>
    #include <functional>
    #include <print>
    #include <ranges>
    #include <string>
    #include <string_view>
    #include <tuple>
     
    struct Names
    {
        [std::string](../../string/basic_string.html) forename, surname;
        friend auto operator<(const Names& lhs, const Names& rhs)
        {
            return [std::tie](../../utility/tuple/tie.html)(lhs.surname, lhs.forename) < [std::tie](../../utility/tuple/tie.html)(rhs.surname, rhs.forename);
        }
    };
     
    struct SurnameCompare
    {
        [std::string_view](../../string/basic_string_view.html) surname;
     
        friend bool operator<(const Names& lhs, const SurnameCompare& rhs)
        {
            return lhs.surname < rhs.surname;
        }
     
        friend bool operator<(const SurnameCompare& lhs, const Names& rhs)
        {
            return lhs.surname < rhs.surname;
        }
    }; 
     
    [std::set](../set.html)<Names, [std::less](../../utility/functional/less.html)<>> characters
    {
        {"Homer", "Simpson"},
        {"Marge", "Simpson"},
        {"Lisa", "Simpson"},
        {"Ned", "Flanders"},
        {"Joe", "Quimby"}
    };
     
    void print_unique(const Names& names)
    {
        auto [begin, end] = characters.equal_range(names);
        [std::print](../../io/print.html)
        (
            "Found {} characters with name \"{} {}\"\n", 
            [std::distance](../../iterator/distance.html)(begin, end), 
            names.forename, names.surname
        );
    }
     
    void print_by_surname([std::string_view](../../string/basic_string_view.html) surname)
    {
        auto [begin, end] = characters.equal_range(SurnameCompare{surname});
        [std::print](../../io/print.html)("Found {} characters with surname \"{}\":\n", [std::distance](../../iterator/distance.html)(begin, end), surname);
        for (const Names& names : std::[ranges::subrange](../../ranges/subrange.html)(begin, end))
            [std::print](../../io/print.html)("    {} {}\n", names.forename, names.surname);
    }
     
    int main()
    {
        print_unique({"Maude", "Flanders"});
        print_unique({"Lisa", "Simpson"});
        print_by_surname("Simpson");
    }

Output: 
    
    
    Found 0 characters with name "Maude Flanders"
    Found 1 characters with name "Lisa Simpson"
    Found 3 characters with surname "Simpson":
        Homer Simpson
        Lisa Simpson
        Marge Simpson
