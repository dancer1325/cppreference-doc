1,2) Returns an iterator pointing to the first element that is **not** less than key.

3,4) Returns an iterator pointing to the first element that compares **not** less than x.

This overload participates in overload resolution only if `Compare` is [transparent](../../functional.html#Transparent_function_objects "cpp/utility/functional"). It allows calling this function without constructing an instance of `Key`.

key  |  \-  |  key value to compare the elements to   
---|---|---  
x  |  \-  |  alternative value that can be compared to `Key`  
  
### [[edit](https://en.cppreference.com/mwiki/index.php?title=Template:cpp/container/lower_bound&action=edit&section=T-2 "Template:cpp/container/lower bound")] Return value

An iterator pointing to the first element that is not less than the given key, or end() is no such element exists. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=Template:cpp/container/lower_bound&action=edit&section=T-3 "Template:cpp/container/lower bound")] Complexity

Logarithmic in the size of the container.
