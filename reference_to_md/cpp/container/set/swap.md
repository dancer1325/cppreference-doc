void swap( set& other ); |  |  (until C++17)  
---|---|---  
void swap( set& other ) noexcept(/* see below */); |  | (since C++17)   
(constexpr since C++26)  
| |   
  
Exchanges the contents of the container with those of other. Does not invoke any move, copy, or swap operations on individual elements. 

All iterators and references remain valid. The [`end()`](end.html "cpp/container/set/end") iterator is invalidated. `Compare` must be [Swappable](../../named_req/Swappable.html "cpp/named req/Swappable"), and objects of this type are exchanged using unqualified call to non-member `swap`. 

If [std::allocator_traits](../../memory/allocator_traits.html)<allocator_type>::propagate_on_container_swap::value is true, then the allocators are exchanged using an unqualified call to non-member `swap`. Otherwise, they are not swapped (and if get_allocator() != other.get_allocator(), the behavior is undefined).  | (since C++11)

### [[edit](https://en.cppreference.com/mwiki/index.php?title=Template:cpp/container/swap&action=edit&section=T-4 "Template:cpp/container/swap")] Example

Run this code
    
    
    #include <functional>
    #include <iostream>
    #include <set>
     
    template<class Os, class Co>
    Os& operator<<(Os& os, const Co& co)
    {
        os << '{';
        for (const auto& i : co)
            os << ' ' << i;
        return os << " } ";
    }
     
    int main()
    {
        [std::set](../set.html)<int> a1{3, 1, 3, 2}, a2{5, 4, 5};
     
        auto it1 = [std::next](../../iterator/next.html)(a1.begin());
        auto it2 = [std::next](../../iterator/next.html)(a2.begin());
     
        const int& ref1 = *(a1.begin());
        const int& ref2 = *(a2.begin());
     
        [std::cout](../../io/cout.html) << a1 << a2 << *it1 << ' ' << *it2 << ' ' << ref1 << ' ' << ref2 << '\n';
        a1.swap(a2);
        [std::cout](../../io/cout.html) << a1 << a2 << *it1 << ' ' << *it2 << ' ' << ref1 << ' ' << ref2 << '\n';
     
        // Note that every iterator referring to an element in one container before the swap
        // refers to the same element in the other container after the swap. Same is true
        // for references.
     
        struct Cmp : [std::less](../../utility/functional/less.html)<int>
        {
            int id{};
            Cmp(int i) : id{i} {}
        };
     
        [std::set](../set.html)<int, Cmp> s1{{2, 2, 1, 1}, Cmp{6}}, s2{{4, 4, 3, 3}, Cmp{9}};
     
        [std::cout](../../io/cout.html) << s1 << s2 << s1.key_comp().id << ' ' << s2.key_comp().id << '\n';
        s1.swap(s2);
        [std::cout](../../io/cout.html) << s1 << s2 << s1.key_comp().id << ' ' << s2.key_comp().id << '\n';
     
        // So, comparator objects (Cmp) are also exchanged after the swap.
    }

Output: 
    
    
    { 1 2 3 } { 4 5 } 2 5 1 4
    { 4 5 } { 1 2 3 } 2 5 1 4
    { 1 2 } { 3 4 } 6 9
    { 3 4 } { 1 2 } 9 6
