Returns an iterator to the first element of *this. 

If *this is empty, the returned iterator will be equal to [end()](end.html "cpp/container/deque/end"). 

![range-begin-end.svg](https://upload.cppreference.com/mwiki/images/1/1b/range-begin-end.svg)

### Return value

Iterator to the first element. 

### Complexity

Constant. 

###  Notes

libc++ backports `cbegin()` to C++98 mode.

### Example

Run this code
    
    
    #include <algorithm>
    #include <iostream>
    #include <numeric>
    #include <string>
    #include <deque>
     
    int main()
    {
        [std::deque](../deque.html)<int> nums{1, 2, 4, 8, 16};
        [std::deque](../deque.html)<[std::string](../../string/basic_string.html)> fruits{"orange", "apple", "raspberry"};
        [std::deque](../deque.html)<char> empty;
     
        // Print deque.
        [std::for_each](../../algorithm/for_each.html)(nums.begin(), nums.end(), [](const int n) { [std::cout](../../io/cout.html) << n << ' '; });
        [std::cout](../../io/cout.html) << '\n';
     
        // Sums all integers in the deque nums (if any), printing only the result.
        [std::cout](../../io/cout.html) << "Sum of nums: "
                  << [std::accumulate](../../algorithm/accumulate.html)(nums.begin(), nums.end(), 0) << '\n';
     
        // Prints the first fruit in the deque fruits, checking if there is any.
        if (!fruits.empty())
            [std::cout](../../io/cout.html) << "First fruit: " << *fruits.begin() << '\n';
     
        if (empty.begin() == empty.end())
            [std::cout](../../io/cout.html) << "deque 'empty' is indeed empty.\n";
    }

Output: 
    
    
    1 2 4 8 16
    Sum of nums: 31
    First fruit: orange
    deque 'empty' is indeed empty.
