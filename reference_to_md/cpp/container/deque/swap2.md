Specializes the [std::swap](../../utility/swap.html "cpp/algorithm/swap") algorithm for [std::deque](../deque.html "cpp/container/deque"). Swaps the contents of lhs and rhs. Calls lhs.swap(rhs).

### Example

Run this code
    
    
    #include <algorithm>
    #include <iostream>
    #include <deque>
     
    int main()
    {
        [std::deque](../deque.html)<int> alice{1, 2, 3};
        [std::deque](../deque.html)<int> bob{7, 8, 9, 10};
     
        auto print = [](const int& n) { [std::cout](../../io/cout.html) << ' ' << n; };
     
        // Print state before swap
        [std::cout](../../io/cout.html) << "Alice:";
        [std::for_each](../../algorithm/for_each.html)(alice.begin(), alice.end(), print);
        [std::cout](../../io/cout.html) << "\nBobby:";
        [std::for_each](../../algorithm/for_each.html)(bob.begin(), bob.end(), print);
        [std::cout](../../io/cout.html) << '\n';
     
        [std::cout](../../io/cout.html) << "-- SWAP\n";
        [std::swap](../../algorithm/swap.html)(alice, bob);
     
        // Print state after swap
        [std::cout](../../io/cout.html) << "Alice:";
        [std::for_each](../../algorithm/for_each.html)(alice.begin(), alice.end(), print);
        [std::cout](../../io/cout.html) << "\nBobby:";
        [std::for_each](../../algorithm/for_each.html)(bob.begin(), bob.end(), print);
        [std::cout](../../io/cout.html) << '\n';
    }

Output: 
    
    
    Alice: 1 2 3
    Bobby: 7 8 9 10
    -- SWAP
    Alice: 7 8 9 10
    Bobby: 1 2 3
