Specializes the [std::swap](../../utility/swap.html "cpp/algorithm/swap") algorithm for [std::unordered_multiset](../unordered_multiset.html "cpp/container/unordered multiset"). Swaps the contents of lhs and rhs. Calls lhs.swap(rhs).

### [[edit](https://en.cppreference.com/mwiki/index.php?title=Template:cpp/container/swap2&action=edit&section=T-5 "Template:cpp/container/swap2")] Example

Run this code
    
    
    #include <algorithm>
    #include <iostream>
    #include <unordered_set>
     
    int main()
    {
        [std::unordered_multiset](../unordered_multiset.html)<int> alice{1, 2, 3};
        [std::unordered_multiset](../unordered_multiset.html)<int> bob{7, 8, 9, 10};
     
        auto print = [](const int& n) { [std::cout](../../io/cout.html) << ' ' << n; };
     
        // Print state before swap
        [std::cout](../../io/cout.html) << "Alice:";
        [std::for_each](../../algorithm/for_each.html)(alice.begin(), alice.end(), print);
        [std::cout](../../io/cout.html) << "\nBobby:";
        [std::for_each](../../algorithm/for_each.html)(bob.begin(), bob.end(), print);
        [std::cout](../../io/cout.html) << '\n';
     
        [std::cout](../../io/cout.html) << "-- SWAP\n";
        [std::swap](../../algorithm/swap.html)(alice, bob);
     
        // Print state after swap
        [std::cout](../../io/cout.html) << "Alice:";
        [std::for_each](../../algorithm/for_each.html)(alice.begin(), alice.end(), print);
        [std::cout](../../io/cout.html) << "\nBobby:";
        [std::for_each](../../algorithm/for_each.html)(bob.begin(), bob.end(), print);
        [std::cout](../../io/cout.html) << '\n';
    }

Possible output: 
    
    
    Alice: 1 2 3
    Bobby: 7 8 9 10
    -- SWAP
    Alice: 7 8 9 10
    Bobby: 1 2 3
