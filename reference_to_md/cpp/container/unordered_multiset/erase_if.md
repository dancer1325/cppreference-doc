template< class Key, class Hash, class KeyEqual, class Alloc,  
class Pred >  
[std::unordered_multiset](../unordered_multiset.html)<Key, Hash, KeyEqual, Alloc>::size_type  
erase_if( [std::unordered_multiset](../unordered_multiset.html)<Key, Hash, KeyEqual, Alloc>& c,  
Pred pred ); |  | (since C++20)   
(constexpr since C++26)  
| |   
  
Erases all elements that satisfy the predicate pred from c. 

Equivalent to 
    
    
    auto old_size = c.size();
    for (auto first = c.begin(), last = c.end(); first != last;)
    {
        if (pred(*first))
            first = c.erase(first);
        else
            ++first;
    }
    return old_size - c.size();

### [[edit](https://en.cppreference.com/mwiki/index.php?title=Template:cpp/container/erase_if&action=edit&section=T-6 "Template:cpp/container/erase if")] Example

Run this code
    
    
    #include <iostream>
    #include <unordered_set>
     
    void println(auto rem, const auto& container)
    {
        [std::cout](../../io/cout.html) << rem << '{';
        for (char sep[]{0, ' ', 0}; const auto& item : container)
            [std::cout](../../io/cout.html) << sep << item, *sep = ',';
        [std::cout](../../io/cout.html) << "}\n";
    }
     
    int main()
    {
        [std::unordered_multiset](../unordered_multiset.html) data{3, 3, 4, 5, 5, 6, 6, 7, 2, 1, 0};
        println("Original:\n", data);
     
        auto divisible_by_3 = [](const auto& x) { return (x % 3) == 0; };
     
        const auto count = [std::erase_if](../vector/erase2.html)(data, divisible_by_3);
     
        println("Erase all items divisible by 3:\n", data);
        [std::cout](../../io/cout.html) << count << " items erased.\n";
    }

Possible output: 
    
    
    Original:
    {0, 1, 2, 7, 6, 6, 5, 5, 4, 3, 3}
    Erase all items divisible by 3:
    {1, 2, 7, 5, 5, 4}
    5 items erased.
