float load_factor() const; |  |  (since C++11)  
---|---|---  
| |   
  
Returns the average number of elements per bucket, that is, [size()](size.html "cpp/container/unordered multiset/size") divided by [bucket_count()](bucket_count.html "cpp/container/unordered multiset/bucket count").
