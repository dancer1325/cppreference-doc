constexpr pointer data() const noexcept; |  |  (since C++20)  
---|---|---  
| |   
  
Returns a pointer to the beginning of the underlying sequence. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/container/span/data&action=edit&section=1 "Edit section: Return value")] Return value

`_[data_](../span.html#data "cpp/container/span")_`

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/container/span/data&action=edit&section=2 "Edit section: Example")] Example

Run this code
    
    
    #include <iostream>
    #include <span>
     
    int main()
    {
        constexpr char str[] = "ABCDEF\n";
     
        const [std::span](../span.html) sp{str};
     
        for (auto n{sp.size()}; n != 2; --n)
            [std::cout](../../io/cout.html) << sp.last(n).data();
    }

Output: 
    
    
    ABCDEF
    BCDEF
    CDEF
    DEF
    EF
    F
