**Deduction guides**

The following [deduction guides](../../language/ctad.html "cpp/language/class template argument deduction") are provided for `span`. 

1) Allows the element type to be deduced from the iterator-sentinel pair. It also allows the static extent to be deduced if `EndOrSize` satisfies [`_integral-constant-like_`](../../header/span.html#Concept_integral-constant-like "cpp/header/span").(since C++26). This overload participates in overload resolution only if `It` satisfies [`contiguous_iterator`](../../iterator/contiguous_iterator.html "cpp/iterator/contiguous iterator").

2-4) Allows the static extent to be deduced from built-in arrays and [std::array](../array.html "cpp/container/array").

5) Allows the element type to be deduced from ranges. This overload participates in overload resolution only if `R` satisfies [`contiguous_range`](../../ranges/contiguous_range.html "cpp/ranges/contiguous range").

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/container/span/deduction_guides&action=edit&section=1 "Edit section: Example")] Example

Run this code
    
    
    #include <array>
    #include <cstddef>
    #include <iomanip>
    #include <iostream>
    #include <span>
    #include <string_view>
    #include <vector>
     
    void print([std::string_view](../../string/basic_string_view.html) rem = "", [std::size_t](../../types/size_t.html) size_of = 0, [std::size_t](../../types/size_t.html) extent = 0)
    {
        if (rem.empty())
        {
            [std::cout](../../io/cout.html) << "name │ sizeof │ extent\n"
                         "─────┼────────┼────────\n";
            return;
        }
        [std::cout](../../io/cout.html) << [std::setw](../../io/manip/setw.html)(4) << rem << " │ " << [std::setw](../../io/manip/setw.html)(6) << size_of << " │ ";
        if (extent == [std::dynamic_extent](dynamic_extent.html))
            [std::cout](../../io/cout.html) << "dynamic";
        else
            [std::cout](../../io/cout.html) << extent;
        [std::cout](../../io/cout.html) << '\n';
    }
     
    int main()
    {
        int a[]{1, 2, 3, 4, 5};
     
        print();
        [std::span](../span.html) s1{[std::begin](../../iterator/begin.html)(a), [std::end](../../iterator/end.html)(a)}; // guide (1)
        print("s1", sizeof s1, s1.extent);
     
        [std::span](../span.html) s2{[std::begin](../../iterator/begin.html)(a), 3}; // guide (1)
        print("s2", sizeof s2, s2.extent);
     
    #if __cplusplus > 202302L
        [std::span](../span.html) s3{[std::begin](../../iterator/begin.html)(a), [std::integral_constant](../../types/integral_constant.html)<[std::size_t](../../types/size_t.html), 2>{}}; // guide (1)
        print("s3", sizeof s3, s3.extent);
    #endif // C++26
     
        [std::span](../span.html) s4{a}; // guide (2)
        print("s4", sizeof s4, s4.extent);
     
        [std::span](../span.html)<int> s5{a}; // does not use a guide, makes a dynamic span
        print("s5", sizeof s5, s5.extent);
     
        [std::array](../array.html) arr{6, 7, 8};
        [std::span](../span.html) s6{arr}; // guide (3)
        print("s6", sizeof s6, s6.extent);
        s6[0] = 42; // OK, element_type is 'int'
     
        const [std::array](../array.html) arr2{9, 10, 11};
        [std::span](../span.html) s7{arr2}; // guide (4)
        print("s7", sizeof s7, s7.extent);
        // s7[0] = 42; // Error: element_type is 'const int'
     
        [std::vector](../vector.html) v{66, 69, 99};
        [std::span](../span.html) s8{v}; // guide (5)
        print("s8", sizeof s8, s8.extent);
    }

Possible output: 
    
    
    name │ sizeof │ extent
    ─────┼────────┼────────
      s1 │     16 │ dynamic
      s2 │     16 │ dynamic
      s3 │      8 │ 2
      s4 │      8 │ 5
      s5 │     16 │ dynamic
      s6 │      8 │ 3
      s7 │      8 │ 3
      s8 │     16 │ dynamic
