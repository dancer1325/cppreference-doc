Obtains a view to the object representation of the elements of the span s. 

If `N` is std::dynamic_extent, the extent of the returned span `S` is also std::dynamic_extent; otherwise it is sizeof(T) * N. 

`as_writable_bytes` only participates in overload resolution if [std::is_const_v](../../types/is_const.html)<T> is false. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/container/span/as_bytes&action=edit&section=1 "Edit section: Return value")] Return value

1) A span constructed with {reinterpret_cast<const [std::byte](../../types/byte.html)*>(s.data()), s.size_bytes()}.

2) A span constructed with {reinterpret_cast<[std::byte](../../types/byte.html)*>(s.data()), s.size_bytes()}.

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/container/span/as_bytes&action=edit&section=2 "Edit section: Example")] Example

Run this code
    
    
    #include <cstddef>
    #include <iomanip>
    #include <iostream>
    #include <span>
     
    void print(float const x, [std::span](../span.html)<const [std::byte](../../types/byte.html)> const bytes)
    {
        [std::cout](../../io/cout.html) << [std::setprecision](../../io/manip/setprecision.html)(6) << [std::setw](../../io/manip/setw.html)(8) << x << " = { "
                  << [std::hex](../../io/manip/hex.html) << [std::uppercase](../../io/manip/uppercase.html) << [std::setfill](../../io/manip/setfill.html)('0');
        for (auto const b : bytes)
            [std::cout](../../io/cout.html) << [std::setw](../../io/manip/setw.html)(2) << [std::to_integer](../../types/byte.html)<int>(b) << ' ';
        [std::cout](../../io/cout.html) << [std::dec](../../io/manip/hex.html) << "}\n";
    }
     
    int main()
    {
        /* mutable */ float data[1]{3.141592f};
     
        auto const const_bytes = std::as_bytes([std::span](../span.html){data});
     
        print(data[0], const_bytes);
     
        auto const writable_bytes = std::as_writable_bytes([std::span](../span.html){data});
     
        // Change the sign bit that is the MSB (IEEE 754 Floating-Point Standard).
        writable_bytes[3] |= [std::byte](../../types/byte.html){0B1000'0000};
     
        print(data[0], const_bytes);
    }

Possible output: 
    
    
     3.14159 = { D8 0F 49 40 }
    -3.14159 = { D8 0F 49 C0 }
