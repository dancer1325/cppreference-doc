struct formatter<[std::priority_queue](../priority_queue.html)<T, Container, U...>, CharT>; |  |  (since C++23)  
| |   
  
The template specialization of [std::formatter](../../utility/format/formatter.html "cpp/utility/format/formatter") for the container adaptor type [std::priority_queue](../priority_queue.html "cpp/container/priority queue") allows users to convert the underlying container to its textual representation as a collection of elements using [formatting functions](../../utility/format.html "cpp/utility/format"). 

The specialization is enabled if [std::formattable](../../utility/format/formattable.html)<Container, CharT> is true.
