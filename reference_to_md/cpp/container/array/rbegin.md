Returns a reverse iterator to the first element of the reversed *this. It corresponds to the last element of the non-reversed *this. 

If *this is empty, the returned iterator is equal to [rend()](rend.html "cpp/container/array/rend"). 

![range-rbegin-rend.svg](https://upload.cppreference.com/mwiki/images/3/39/range-rbegin-rend.svg)

### Return value

Reverse iterator to the first element. 

### Complexity

Constant. 

### Notes

The [underlying iterator](../../iterator/reverse_iterator/base.html "cpp/iterator/reverse iterator/base") of the returned reverse iterator is the [end iterator](end.html "cpp/container/array/end"). Hence the returned iterator is invalidated if and when the end iterator is invalidated.

### Example

Run this code
    
    
    #include <algorithm>
    #include <array>
    #include <iostream>
    #include <string>
    #include <string_view>
     
    void print(const [std::string_view](../../string/basic_string_view.html) s) { [std::cout](../../io/cout.html) << s << ' '; }
     
    int main()
    {
        const [std::array](../array.html)<[std::string_view](../../string/basic_string_view.html), 8> data
        {
            "▁", "▂", "▃", "▄", "▅", "▆", "▇", "█"
        };
        [std::array](../array.html)<[std::string](../../string/basic_string.html), 8> arr;
     
        [std::copy](../../algorithm/copy.html)(data.cbegin(), data.cend(), arr.begin());
     
        print("Print “arr” in direct order using [cbegin, cend):\t");
        [std::for_each](../../algorithm/for_each.html)(arr.cbegin(), arr.cend(), print);
     
        print("\n\nPrint “arr” in reverse order using [crbegin, crend):\t");
        [std::for_each](../../algorithm/for_each.html)(arr.crbegin(), arr.crend(), print);
    }

Output: 
    
    
    Print “arr” in direct order using [cbegin, cend):	 ▁ ▂ ▃ ▄ ▅ ▆ ▇ █ 
     
    Print “arr” in reverse order using [crbegin, crend):	 █ ▇ ▆ ▅ ▄ ▃ ▂ ▁
