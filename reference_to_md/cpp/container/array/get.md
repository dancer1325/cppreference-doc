Extracts the `I`th element from the array using [tuple-like](../../utility/tuple/tuple-like.html "cpp/utility/tuple/tuple-like") interface. 

`I` must be an integer value in range `[`​0​`, `N`)`. This is enforced at compile time as opposed to [at()](at.html "cpp/container/array/at") or [operator[]](operator_at.html "cpp/container/array/operator at").

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/container/array/get&action=edit&section=4 "Edit section: Example")] Example

Run this code
    
    
    #include <array>
    #include <iostream>
     
    constexpr [std::array](../array.html) v{1, 2, 3};
    static_assert(get<0>(v) == 1 && get<1>(v) == 2 && get<2>(v) == 3);
     
    int main()
    {
        [std::array](../array.html)<int, 3> a;
     
        // set values:
        get<0>(a) = 1, get<1>(a) = 2, get<2>(a) = 3;
     
        // get values:
        [std::cout](../../io/cout.html) << '(' << get<0>(a) << ',' << get<1>(a) << ',' << get<2>(a) << ")\n";
    }

Output: 
    
    
    (1,2,3)
