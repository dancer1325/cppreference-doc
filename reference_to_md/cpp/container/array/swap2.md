Specializes the [std::swap](../../utility/swap.html "cpp/algorithm/swap") algorithm for [std::array](../array.html "cpp/container/array"). Swaps the contents of lhs and rhs. Calls lhs.swap(rhs). This overload participates in overload resolution only if N == 0 or [std::is_swappable_v](../../types/is_swappable.html)<T> is true.  | (since C++17)

### Parameters

---|---|---  
  
### Complexity

Linear in size of the containers. 

###  Exceptions

[`noexcept`](../../language/noexcept_spec.html "cpp/language/noexcept spec") specification: noexcept(noexcept(lhs.swap(rhs))) | (since C++17)

### Example

Run this code
    
    
    #include <algorithm>
    #include <iostream>
    #include <array>
     
    int main()
    {
        [std::array](../array.html)<int, 3> alice{1, 2, 3};
        [std::array](../array.html)<int, 3> bob{7, 8, 9};
     
        auto print = [](const int& n) { [std::cout](../../io/cout.html) << ' ' << n; };
     
        // Print state before swap
        [std::cout](../../io/cout.html) << "Alice:";
        [std::for_each](../../algorithm/for_each.html)(alice.begin(), alice.end(), print);
        [std::cout](../../io/cout.html) << "\nBobby:";
        [std::for_each](../../algorithm/for_each.html)(bob.begin(), bob.end(), print);
        [std::cout](../../io/cout.html) << '\n';
     
        [std::cout](../../io/cout.html) << "-- SWAP\n";
        [std::swap](../../algorithm/swap.html)(alice, bob);
     
        // Print state after swap
        [std::cout](../../io/cout.html) << "Alice:";
        [std::for_each](../../algorithm/for_each.html)(alice.begin(), alice.end(), print);
        [std::cout](../../io/cout.html) << "\nBobby:";
        [std::for_each](../../algorithm/for_each.html)(bob.begin(), bob.end(), print);
        [std::cout](../../io/cout.html) << '\n';
    }

Output: 
    
    
    Alice: 1 2 3
    Bobby: 7 8 9
    -- SWAP
    Alice: 7 8 9
    Bobby: 1 2 3
