const T* data() const noexcept; |  (2) | (since C++11)   
(constexpr since C++17)  
| |   
  
Returns a pointer to the underlying array serving as element storage. The pointer is such that range `[`data()`, `data() +` `[size()](size.html "cpp/container/array/size")`)` is always a [valid range](../../iterator.html#Ranges "cpp/iterator")

If *this is empty, `data()` is not dereferenceable.

### [[edit](https://en.cppreference.com/mwiki/index.php?title=Template:cpp/container/data&action=edit&section=T-1 "Template:cpp/container/data")] Return value

Pointer to the underlying element storage. For non-empty containers, the returned pointer compares equal to the address of the first element. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=Template:cpp/container/data&action=edit&section=T-2 "Template:cpp/container/data")] Complexity

Constant. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=Template:cpp/container/data&action=edit&section=T-3 "Template:cpp/container/data")] Notes

If *this is empty, `data()` may or may not return a null pointer. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=Template:cpp/container/data&action=edit&section=T-4 "Template:cpp/container/data")] Example

Run this code
    
    
    #include <cstddef>
    #include <iostream>
    #include <span>
    #include <array>
     
    void pointer_func(const int* p, [std::size_t](../../types/size_t.html) size)
    {
        [std::cout](../../io/cout.html) << "data = ";
        for ([std::size_t](../../types/size_t.html) i = 0; i < size; ++i)
            [std::cout](../../io/cout.html) << p[i] << ' ';
        [std::cout](../../io/cout.html) << '\n';
    }
     
    void span_func([std::span](../span.html)<const int> data) // since C++20
    {
        [std::cout](../../io/cout.html) << "data = ";
        for (const int e : data)
            [std::cout](../../io/cout.html) << e << ' ';
        [std::cout](../../io/cout.html) << '\n';
    }
     
    int main()
    {
        [std::array](../array.html)<int, 4> container{1, 2, 3, 4};
     
        // Prefer container.data() over &container[0]
        pointer_func(container.data(), container.size());
     
        // std::span is a safer alternative to separated pointer/size.
        span_func({container.data(), container.size()});
    }

Output: 
    
    
    data = 1 2 3 4
    data = 1 2 3 4
