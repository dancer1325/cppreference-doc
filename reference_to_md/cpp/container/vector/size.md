size_type size() const; |  | (noexcept since C++11)   
(constexpr since C++20)  
---|---|---  
| |   
  
Returns the number of elements in the container.

  * [1 Return value](size.html#Return_value)
  * [2 Complexity](size.html#Complexity)

### Return value

[std::distance](../../iterator/distance.html)(begin(), end())

### Complexity

Constant. 

### Example



The following code uses `size` to display the number of elements in a [std::vector](../vector.html)<int>:

Run this code
    
    
    #include <cassert>
    #include <vector>
    Â 
    int main()
    {
        [std::vector](../vector.html)<int> nums;
        [assert](../../error/assert.html)(nums.size() == 0);
        nums = {1, 2, 3, 4};
        [assert](../../error/assert.html)(nums.size() == 4);
    }
