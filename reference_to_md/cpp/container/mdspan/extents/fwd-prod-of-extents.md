constexpr [std::size_t](../../../types/size_t.html) /*fwd-prod-of-extents*/( rank_type i ) const noexcept; |  |  (since C++23)   
(exposition only*)  
---|---|---  
| |   
  
Returns the product of the sizes of extents with index less than i. The behavior is undefined if i <= rank() is false. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/container/mdspan/extents/fwd-prod-of-extents&action=edit&section=1 "Edit section: Parameters")] Parameters

i  |  \-  |  The end index of the range of extents to be multiplied together.   
---|---|---  
  
### [[edit](https://en.cppreference.com/mwiki/index.php?title=cpp/container/mdspan/extents/fwd-prod-of-extents&action=edit&section=2 "Edit section: Return value")] Return value

If i > 0 is true, return the product of [`extent`](extent.html "cpp/container/mdspan/extents/extent")(k) for all `k` in range `[`​0​`, `i`)`, otherwise 1.
