Specializes the [std::swap](../../utility/swap.html "cpp/algorithm/swap") algorithm for [std::unordered_map](../unordered_map.html "cpp/container/unordered map"). Swaps the contents of lhs and rhs. Calls lhs.swap(rhs).

### [[edit](https://en.cppreference.com/mwiki/index.php?title=Template:cpp/container/swap2&action=edit&section=T-5 "Template:cpp/container/swap2")] Example

Run this code
    
    
    #include <algorithm>
    #include <iostream>
    #include <unordered_map>
     
    int main()
    {
        [std::unordered_map](../unordered_map.html)<int, char> alice{{1, 'a'}, {2, 'b'}, {3, 'c'}};
        [std::unordered_map](../unordered_map.html)<int, char> bob{{7, 'Z'}, {8, 'Y'}, {9, 'X'}, {10, 'W'}};
     
        auto print = []([std::pair](../../utility/pair.html)<const int, char>& n)
        {
            [std::cout](../../io/cout.html) << ' ' << n.first << ':' << n.second;
        };
     
        // Print state before swap
        [std::cout](../../io/cout.html) << "Alice:";
        [std::for_each](../../algorithm/for_each.html)(alice.begin(), alice.end(), print);
        [std::cout](../../io/cout.html) << "\nBobby:";
        [std::for_each](../../algorithm/for_each.html)(bob.begin(), bob.end(), print);
        [std::cout](../../io/cout.html) << '\n';
     
        [std::cout](../../io/cout.html) << "-- SWAP\n";
        [std::swap](../../algorithm/swap.html)(alice, bob);
     
        // Print state after swap
        [std::cout](../../io/cout.html) << "Alice:";
        [std::for_each](../../algorithm/for_each.html)(alice.begin(), alice.end(), print);
        [std::cout](../../io/cout.html) << "\nBobby:";
        [std::for_each](../../algorithm/for_each.html)(bob.begin(), bob.end(), print);
        [std::cout](../../io/cout.html) << '\n';
    }

Possible output: 
    
    
    Alice: 1:a 2:b 3:c
    Bobby: 7:Z 8:Y 9:X 10:W
    -- SWAP
    Alice: 7:Z 8:Y 9:X 10:W
    Bobby: 1:a 2:b 3:c
