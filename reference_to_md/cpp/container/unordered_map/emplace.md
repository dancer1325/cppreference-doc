template< class... Args >  
[std::pair](../../utility/pair.html)<iterator, bool> emplace( Args&&... args ); |  | (since C++11)   
(constexpr since C++26)  
---|---|---  
| |   
  
Inserts a new element into the container constructed in-place with the given args, if there is no element with the key in the container. 

The constructor of the new element (i.e. [std::pair](../../utility/pair.html)<const Key, T>) is called with exactly the same arguments as supplied to `emplace`, forwarded via [std::forward](../../utility/forward.html)<Args>(args).... The element may be constructed even if there already is an element with the key in the container, in which case the newly constructed element will be destroyed immediately (see [`try_emplace()`](try_emplace.html "cpp/container/unordered map/try emplace") if this behavior is undesirable). 

If `value_type` is not [EmplaceConstructible](../../named_req/EmplaceConstructible.html "cpp/named req/EmplaceConstructible") into `unordered_map` from args, the behavior is undefined. 

If after the operation the new number of elements is greater than old [`max_load_factor()`](max_load_factor.html "cpp/container/unordered map/max load factor")` *`` `[`bucket_count()`](bucket_count.html "cpp/container/unordered map/bucket count") a rehashing takes place.  
If rehashing occurs (due to the insertion), all iterators are invalidated. Otherwise (no rehashing), iterators are not invalidated.

args  |  \-  |  arguments to forward to the constructor of the element

### [[edit](https://en.cppreference.com/mwiki/index.php?title=Template:cpp/container/emplace_assoc&action=edit&section=T-2 "Template:cpp/container/emplace assoc")] Return value

A pair consisting of an iterator to the inserted element (or to the element that prevented the insertion) and a bool value set to true if and only if the insertion took place. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=Template:cpp/container/emplace_assoc&action=edit&section=T-3 "Template:cpp/container/emplace assoc")] Exceptions

If an exception is thrown for any reason, this function has no effect ([strong exception safety guarantee](../../language/exceptions.html#Exception_safety "cpp/language/exceptions")). 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=Template:cpp/container/emplace_assoc&action=edit&section=T-4 "Template:cpp/container/emplace assoc")] Complexity

Amortized constant on average, worst case linear in the size of the container. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=Template:cpp/container/emplace_assoc&action=edit&section=T-5 "Template:cpp/container/emplace assoc")] Notes

Careful use of `emplace` allows the new element to be constructed while avoiding unnecessary copy or move operations. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=Template:cpp/container/emplace_assoc&action=edit&section=T-6 "Template:cpp/container/emplace assoc")] Example

Run this code
    
    
    #include <iostream>
    #include <string>
    #include <utility>
    #include <unordered_map>
     
    int main()
    {
        [std::unordered_map](../unordered_map.html)<[std::string](../../string/basic_string.html), [std::string](../../string/basic_string.html)> m;
     
        // uses pair's move constructor
        m.emplace([std::make_pair](../../utility/pair/make_pair.html)([std::string](../../string/basic_string.html)("a"), [std::string](../../string/basic_string.html)("a")));
     
        // uses pair's converting move constructor
        m.emplace([std::make_pair](../../utility/pair/make_pair.html)("b", "abcd"));
     
        // uses pair's template constructor
        m.emplace("d", "ddd");
     
        // emplace with duplicate key has no effect
        m.emplace("d", "DDD");
     
        // uses pair's piecewise constructor
        m.emplace([std::piecewise_construct](../../utility/piecewise_construct.html),
                  [std::forward_as_tuple](../../utility/tuple/forward_as_tuple.html)("c"),
                  [std::forward_as_tuple](../../utility/tuple/forward_as_tuple.html)(10, 'c'));
        // an alternative is: m.try_emplace("c", 10, 'c');
     
        for (const auto& p : m)
            [std::cout](../../io/cout.html) << p.first << " => " << p.second << '\n';
    }

Possible output: 
    
    
    a => a
    b => abcd
    c => cccccccccc
    d => ddd
