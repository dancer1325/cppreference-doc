Erases all elements that satisfy the predicate pred from c. 

The predicate pred is satisfied if the expression bool(pred([std::pair](../../utility/pair.html)<const Key&, const T&>(e))) is true, where e is some element in c. 

If `Key` or `T` is not [MoveAssignable](../../named_req/MoveAssignable.html "cpp/named req/MoveAssignable"), the behavior is undefined.

### [[edit](https://en.cppreference.com/mwiki/index.php?title=Template:cpp/container/erase_if&action=edit&section=T-1 "Template:cpp/container/erase if")] Parameters

c  |  \-  |  container adaptor from which to erase   
---|---|---  
pred  |  \-  |  predicate that returns true if the element should be erased   
  
### [[edit](https://en.cppreference.com/mwiki/index.php?title=Template:cpp/container/erase_if&action=edit&section=T-2 "Template:cpp/container/erase if")] Return value

The number of erased elements. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=Template:cpp/container/erase_if&action=edit&section=T-3 "Template:cpp/container/erase if")] Complexity

Exactly c.size() applications of the predicate pred. 

###  Exceptions

If `erase_if` throws, c remains in valid but unspecified (maybe empty) state. 

###  Notes

The algorithm is stable, that is, the order of elements that are not deleted remains unchanged. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=Template:cpp/container/erase_if&action=edit&section=T-6 "Template:cpp/container/erase if")] Example

Run this code
    
    
    #include <iostream>
    #include <flat_map>
     
    void println(auto rem, const auto& container)
    {
        [std::cout](../../io/cout.html) << rem << '{';
        for (char sep[]{0, ' ', 0}; const auto& [key, value] : container)
            [std::cout](../../io/cout.html) << sep << '{' << key << ", " << value << '}', *sep = ',';
        [std::cout](../../io/cout.html) << "}\n";
    }
     
    int main()
    {
        [std::flat_multimap](../flat_multimap.html)<int, char> data
        {
            {1, 'a'}, {2, 'b'}, {3, 'c'}, {4, 'd'},
            {5, 'e'}, {4, 'f'}, {5, 'g'}, {5, 'g'},
        };
        println("Original:\n", data);
     
        const auto count = [std::erase_if](../vector/erase2.html)(data, [](const auto& item)
        {
            const auto& [key, value] = item;
            return (key & 1) == 1;
        });
     
        println("Erase items with odd keys:\n", data);
        [std::cout](../../io/cout.html) << count << " items removed.\n";
    }

Output: 
    
    
    Original:
    {{1, a}, {2, b}, {3, c}, {4, d}, {4, f}, {5, e}, {5, g}, {5, g}}
    Erase items with odd keys:
    {{2, b}, {4, d}, {4, f}}
    5 items removed.
