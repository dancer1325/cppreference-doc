Returns an iterator to the first element of *this. 

If *this is empty, the returned iterator will be equal to end(). 

![range-begin-end.svg](https://upload.cppreference.com/mwiki/images/1/1b/range-begin-end.svg)

### Return value

Iterator to the first element. 

### Complexity

Constant. 

### Example

Run this code
    
    
    #include <iostream>
    #include <flat_map>
     
    int main()
    {
        [std::flat_multimap](../flat_multimap.html)<int, int> map{{4, 13}, {9, 94}, {1, 19}, {4, 42}};
     
        for (auto it = map.cbegin(); it != map.cend(); ++it)
            [std::cout](../../io/cout.html) << '[' << it->first << "] = " << it->second << '\n';
     
        // Unlike std::multimap's bidirectional iterators, the std::flat_multimap
        // iterators are random-access, so they can be used with the operator[]:
        auto it = map.cbegin();
        [assert](../../error/assert.html)(it[1] == 19);
        [assert](../../error/assert.html)(it[4] == 13);
        [assert](../../error/assert.html)(it[4] == 42);
        [assert](../../error/assert.html)(it[9] == 94);
    }

Output: 
    
    
    [1] = 19
    [4] = 13
    [4] = 42
    [9] = 94
