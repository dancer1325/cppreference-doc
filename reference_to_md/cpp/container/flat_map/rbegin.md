Returns a reverse iterator to the first element of the reversed *this. It corresponds to the last element of the non-reversed *this. 

If *this is empty, the returned iterator is equal to rend(). 

![range-rbegin-rend.svg](https://upload.cppreference.com/mwiki/images/3/39/range-rbegin-rend.svg)

### Return value

Reverse iterator to the first element. 

### Complexity

Constant. 

### Notes

The [underlying iterator](../../iterator/reverse_iterator/base.html "cpp/iterator/reverse iterator/base") of the returned reverse iterator is the [end iterator](end.html "cpp/container/flat map/end"). Hence the returned iterator is invalidated if and when the end iterator is invalidated.

### Example

Run this code
    
    
    #include <iomanip>
    #include <iostream>
    #include <string_view>
    #include <flat_map>
     
    int main()
    {
        const [std::flat_map](../flat_map.html)<int, [std::string_view](../../string/basic_string_view.html)> coins
        {
            {10, "dime"},
            {100, "dollar"},
            {50, "half dollar"},
            {5, "nickel"},
            {1, "penny"},
            {25, "quarter"}
        }; // initializer entries in name alphabetical order
     
        [std::cout](../../io/cout.html) << "US coins in circulation, largest to smallest denomination:\n";
        for (auto it = coins.crbegin(); it != coins.crend(); ++it)
            [std::cout](../../io/cout.html) << [std::setw](../../io/manip/setw.html)(11) << it->second << " = ¢" << it->first << '\n';
    }

Output: 
    
    
    US coins in circulation, largest to smallest denomination:
         dollar = ¢100
    half dollar = ¢50
        quarter = ¢25
           dime = ¢10
         nickel = ¢5
          penny = ¢1
