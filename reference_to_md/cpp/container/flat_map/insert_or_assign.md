1,2) If a key equivalent to k already exists in the container, assigns [std::forward](../../utility/forward.html)<M>(obj) to the [`mapped_type`](../flat_map.html#Member_types "cpp/container/flat map") corresponding to the key k. If the key does not exist, inserts the new value as if by 

  * (1,2) try_emplace([std::forward](../../utility/forward.html)<decltype(k)>(k), [std::forward](../../utility/forward.html)<M>(obj)), 
  * (4,5) try_emplace(hint, [std::forward](../../utility/forward.html)<decltype(k)>(k), [std::forward](../../utility/forward.html)<M>(obj)).



The program is ill-formed if either [std::is_assignable_v](../../types/is_assignable.html)<mapped_type&, M> or [std::is_constructible_v](../../types/is_constructible.html)<mapped_type, M> is false.

3,6) If a key equivalent to k already exists in the container, assigns [std::forward](../../utility/forward.html)<M>(obj) to the [`mapped_type`](../flat_map.html#Member_types "cpp/container/flat map") corresponding to the key k. Otherwise, equivalent to 

  * (3) try_emplace([std::forward](../../utility/forward.html)<K>(k), [std::forward](../../utility/forward.html)<M>(obj)), 
  * (6) try_emplace(hint, [std::forward](../../utility/forward.html)<K>(k), [std::forward](../../utility/forward.html)<M>(obj)).



The conversion from k into `key_type` must construct an object u, for which find(k) == find(u) is true. Otherwise, the behavior is undefined.

These overloads participate in overload resolution only if : 

  * The qualified-id `Compare::is_transparent` is valid and denotes a type. 
  * [std::is_constructible_v](../../types/is_constructible.html)<key_type, K> is true. 
  * [std::is_assignable_v](../../types/is_assignable.html)<mapped_type&, M> is true. 
  * [std::is_constructible_v](../../types/is_constructible.html)<mapped_type, M> is true.

### Parameters

k  |  \-  |  the key used both to look up and to insert if not found   
---|---|---  
hint  |  \-  |  iterator to the position before which the new element will be inserted   
obj  |  \-  |  the value to insert or assign   
  
### Return value

1-3) The bool component is true if the insertion took place and false if the assignment took place. The iterator component is pointing at the element that was inserted or updated.

4-6) Iterator pointing at the element that was inserted or updated.

### Complexity

1-3) Same as for [`emplace`](emplace.html "cpp/container/flat map/emplace").

4-6) Same as for [`emplace_hint`](emplace_hint.html "cpp/container/flat map/emplace hint").

### Notes

`insert_or_assign` returns more information than [`operator`](operator_at.html "cpp/container/flat map/operator at")[] and does not require default-constructibility of the mapped type. 

### Example

Run this code
    
    
    #include <flat_map>
    #include <iostream>
    #include <string>
     
    void print_node(const auto& node)
    {
        [std::cout](../../io/cout.html) << '[' << node.first << "] = " << node.second << '\n';
    }
     
    void print_result(auto const& pair)
    {
        [std::cout](../../io/cout.html) << (pair.second ? "inserted: " : "assigned: ");
        print_node(*pair.first);
    }
     
    int main()
    {
        [std::flat_map](../flat_map.html)<[std::string](../../string/basic_string.html), [std::string](../../string/basic_string.html)> map;
     
        print_result(map.insert_or_assign("a", "apple"));
        print_result(map.insert_or_assign("b", "banana"));
        print_result(map.insert_or_assign("c", "cherry"));
        print_result(map.insert_or_assign("c", "clementine"));
     
        for (const auto& node : map)
            print_node(node);
    }

Output: 
    
    
    inserted: [a] = apple
    inserted: [b] = banana
    inserted: [c] = cherry
    assigned: [c] = clementine
    [a] = apple
    [b] = banana
    [c] = clementine
