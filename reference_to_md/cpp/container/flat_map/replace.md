void replace( key_container_type&& key_cont, mapped_container_type&& mapped_cont ); |  |  (since C++23)  
---|---|---  
| |   
  
Replaces the underlying containers [`_c_`](../flat_map.html#Member_objects "cpp/container/flat map"). Equivalent to: 
    
    
    c.keys = std::move(key_cont);
    c.values = std::move(mapped_cont);

The following conditions must be met: 

  * The expression key_cont.size() == mapped_cont.size() is true, 
  * The elements of key_cont are sorted with respect to [`_compare_`](../flat_map.html#Member_objects "cpp/container/flat map"), and 
  * key_cont does not contain equal elements.

Otherwise, the behavior is undefined.

### Parameters

keys_cont  |  \-  |  a sorted keys container of type `KeyContainer`, the contents of which will be moved into *this  
---|---|---  
mapped_cont  |  \-  |  a container of mapped values of type `MappedContainer`, the contents of which will be moved into *this  
  
### Return value

(none) 

### Complexity

Equals to complexity of [`std::move`](../../utility/move.html "cpp/utility/move") applied to adapted containers. 

### Example

Run this code
    
    
    #include <algorithm>
    #include <cassert>
    #include <flat_map>
    #include <print>
    #include <vector>
     
    int main()
    {
        [std::vector](../vector.html)<int> keys{1, 2, 3};
        [assert](../../error/assert.html)(std::[ranges::is_sorted](../../algorithm/ranges/is_sorted.html)(keys));
        [std::vector](../vector.html)<double> values{2.2, 3.3, 1.1};
        [assert](../../error/assert.html)(keys.size() == values.size());
     
        [std::flat_map](../flat_map.html)<int, double> map;
        [assert](../../error/assert.html)(map.empty());
     
        map.replace(keys, values);
        [assert](../../error/assert.html)(map.size() == 3);
        [assert](../../error/assert.html)(map.keys() == 3);
        [assert](../../error/assert.html)(map.values() == 3);
        [assert](../../error/assert.html)(keys.empty());
        [assert](../../error/assert.html)(values.empty());
     
        [std::println](../../io/println.html)("{}", map);
    }

Output: 
    
    
    {1: 2.2, 2: 3.3, 3: 1.1}
