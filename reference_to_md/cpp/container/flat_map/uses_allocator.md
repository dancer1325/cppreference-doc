: [std::bool_constant](../../types/integral_constant.html)<[std::uses_allocator_v](../../memory/uses_allocator.html)<KeyContainer, Allocator> &&

Provides a transparent specialization of the [std::uses_allocator](../../memory/uses_allocator.html "cpp/memory/uses allocator") type trait for [std::flat_map](../flat_map.html): the container adaptor uses allocator if and only if the underlying container does.
