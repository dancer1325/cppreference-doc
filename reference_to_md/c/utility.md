  * language support libraries, and 
  * general-purpose libraries.

## [[edit](https://en.cppreference.com/mwiki/index.php?title=c/utility&action=edit&section=1 "Edit section: Language support")] Language support

Language support libraries provide functions that interact closely with language features and support common language idioms. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=c/utility&action=edit&section=2 "Edit section: Type support")] [Type support](types.html "c/types")

Basic types (e.g. [size_t](types/size_t.html "c/types/size t"), [NULL](types/NULL.html "c/types/NULL")) 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=c/utility&action=edit&section=3 "Edit section: Dynamic memory management")] [Dynamic memory management](memory.html "c/memory")

(e.g. [malloc](memory/malloc.html "c/memory/malloc")) 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=c/utility&action=edit&section=4 "Edit section: Error handling")] [Error handling](error.html "c/error")

Assertions (e.g. [assert](error/assert.html "c/error/assert")) 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=c/utility&action=edit&section=5 "Edit section: Variadic functions")] [Variadic functions](variadic.html "c/variadic")

Support for functions that take an arbitrary number of parameters (via e.g. va_start, va_arg, va_end) 

  


## [[edit](https://en.cppreference.com/mwiki/index.php?title=c/utility&action=edit&section=6 "Edit section: General-purpose utilities")] General-purpose utilities

### [[edit](https://en.cppreference.com/mwiki/index.php?title=c/utility&action=edit&section=7 "Edit section: Program utilities")] [ Program utilities](program.html "c/program")

Termination (e.g. [abort](program/abort.html "c/program/abort"), [atexit](program/atexit.html "c/program/atexit")), environment (e.g. [system](program/system.html "c/program/system")), signals (e.g. [raise](program/raise.html "c/program/raise")) 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=c/utility&action=edit&section=8 "Edit section: Date and time")] [Date and time](chrono.html "c/chrono")

Time tracking (e.g. [time](chrono/time.html "c/chrono/time"), [clock](chrono/clock.html "c/chrono/clock"))
